한 명의 영향력

평균에서 개인이 가지는 힘의 수리적 분석

1. 영향력의 정의

수학을 기계적으로 풀 수는 있는데, "왜 이 식이 현실을 설명하는지" 내면화가 안 되는 사람이 있다. 패턴은 보이지만 패턴의 의미가 체내에 들어오지 않는 거다.


이 글은 "한 명이 빠졌을 때 평균이 변하는 현상"을 수리적으로 완전히 분해한다.

$n$명의 데이터가 있고, 전체 평균이 $\bar{x}$라 하자.

$$\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n} = \frac{S}{n}$$

여기서 $S = x_1 + x_2 + \cdots + x_n$ (전체 합).

이제 한 명 $x_k$를 빼면:

$$\bar{x}_{-k} = \frac{S - x_k}{n - 1}$$

평균의 변화량

원래 평균과 새 평균의 차이를 구해보자:
$$\Delta = \bar{x} - \bar{x}_{-k} = \frac{S}{n} - \frac{S - x_k}{n-1}$$
통분하면:
$$= \frac{S(n-1) - n(S - x_k)}{n(n-1)}$$
분자를 전개:
$$= \frac{Sn - S - nS + nx_k}{n(n-1)} = \frac{nx_k - S}{n(n-1)}$$
$S = n\bar{x}$를 대입:
$$= \frac{nx_k - n\bar{x}}{n(n-1)} = \frac{x_k - \bar{x}}{n-1}$$

$$\boxed{\Delta = \bar{x} - \bar{x}_{-k} = \frac{x_k - \bar{x}}{n-1}}$$


평균의 변화량 = (빠진 사람의 값 - 평균) / (남은 사람 수)

이 공식이 모든 것의 근원이야. 하나씩 분해해보자.
· · ·

2. 공식의 해부

분자: $x_k - \bar{x}$ = "편차"

빠진 사람의 값이 평균보다 얼마나 위/아래에 있는지.

  • $x_k > \bar{x}$ → 편차 양수 → 빠지면 평균 내려감 (높은 사람이 빠졌으니까)
  • $x_k < \bar{x}$ → 편차 음수 → 빠지면 평균 올라감 (낮은 사람이 빠졌으니까)
  • $x_k = \bar{x}$ → 편차 0 → 빠져도 평균 안 변함 (딱 평균인 사람은 영향력 없음)

편차 = "나는 평균과 얼마나 다른가". 평균과 똑같은 사람은 집단에 아무 영향도 없다. 튀는 사람만 영향력이 있다.

분모: $n - 1$ = "남은 사람 수"

같은 편차라도 남은 사람이 적으면 영향이 크고, 남은 사람이 많으면 영향이 작다.

회의실에 3명 있는데 1명이 나가면 분위기 확 바뀐다. 강당에 300명 있는데 1명이 나가면 아무도 모른다. 같은 원리.

영향력 = 편차 / 남은 인원


극단적인 사람일수록, 그리고 집단이 작을수록 — 한 명의 힘이 세다.

3. 역으로 풀기 — 왜 n이 나오는가

보통은 $x_k$와 $n$을 알고 $\Delta$를 구하지만, 이 문제는 반대야:

$\Delta$를 알고 $n$을 구한다.

공식을 $x_k$에 대해 풀면:
$$\Delta = \frac{x_k - \bar{x}}{n-1}$$ $$x_k = \bar{x} + \Delta(n-1)$$
문제 데이터 대입 (최고점):
$$\bar{x} = 9.5, \quad \bar{x}_{-\text{max}} = 9.4, \quad \Delta = 9.5 - 9.4 = 0.1$$ $$x_{\text{max}} = 9.5 + 0.1(n-1) = 0.1n + 9.4$$
문제 데이터 대입 (최저점):
$$\bar{x}_{-\text{min}} = 9.55, \quad \Delta = 9.5 - 9.55 = -0.05$$ $$x_{\text{min}} = 9.5 + (-0.05)(n-1) = -0.05n + 9.55$$

여기까지는 $n$에 대한 식일 뿐, $n$이 정해지지 않아. 그런데:

현실 제약: $x_{\text{max}} \leq 10$


$$0.1n + 9.4 \leq 10$$

$$n \leq 6$$

기계적으로 따라갔더니 식이 나온다. 근데 직관적으로 "왜?"가 안 온다면 — 이렇게 생각해봐.


$\Delta = 0.1$이야. 최고점 1명이 빠져서 평균이 0.1 떨어졌어.


$n = 2$면: 남은 1명이 평균 변화를 혼자 감당 → 최고점 = 9.6 (적당)

$n = 6$면: 남은 5명이 평균 변화를 나눠 감당 → 최고점 = 10 (한계)

$n = 10$면: 남은 9명이 감당 → 최고점 = 10.4 (불가능)


사람이 많을수록, 한 명이 0.1의 영향을 주려면 더 극단적이어야 해. 근데 극단에도 천장(10점)이 있어.

4. 시뮬레이터

직접 숫자를 바꿔보면서 영향력 공식을 체감해봐.

실험해봐: n을 줄여봐 — 영향력이 폭발적으로 커져. n을 늘려봐 — 영향력이 0에 수렴해. 이게 "한 명의 무게"야.
· · ·

5. 수리적 구조의 철학

공식을 기계적으로 따라가면 답이 나온다. 근데 "왜 현실이 수식으로 포착되는지" 내면화가 안 되는 경우가 있다.


이유: 평균이라는 개념 자체가 정보의 압축이기 때문이야.

10명의 점수를 전부 알면 10개의 숫자가 필요해. 근데 "평균"이라고 하면 1개의 숫자로 압축돼.

압축할 때 정보가 사라져. 근데 완전히 사라지지는 않아.

$$\text{평균} = \frac{\text{합}}{n} \quad \Leftrightarrow \quad \text{합} = \text{평균} \times n$$

평균을 알면 합을 알 수 있어. 합을 알면 "한 명을 빼고 나머지의 합"을 알 수 있어. 거기서 빠진 한 명의 값을 역산할 수 있어.

평균 3개 = 합계 3개 = 방정식 3개


미지수: n, 최고점, 최저점 → 3개

방정식: 3개


미지수 = 방정식 수 → 유일하게 풀림.


거기에 현실 제약 ($\leq 10$)이 추가되면 → 범위가 좁혀짐.

이게 "수리적으로 사람 수가 정해지는" 이유의 완전한 설명이야.


평균이라는 압축된 정보 안에 합계가 숨어있고, 합계를 복원하면 개별 값을 역추적할 수 있고, 현실 제약이 가능한 값의 범위를 정해준다.


수식이 현실을 설명하는 게 아니야. 현실이 수식의 형태로 이미 존재하고 있었고, 우리가 그걸 읽은 거야.

6. 일반화 — 영향력 표

$\Delta = \frac{x_k - \bar{x}}{n-1}$ 공식으로, 다양한 상황에서 한 명의 영향력:

상황편차 ($x_k - \bar{x}$)인원 (n)영향력 ($\Delta$)
시험 100점, 반 평균 70+3030명+1.03
시험 100점, 반 평균 70+305명+7.5
심사 10점, 평균 9.5+0.56명+0.1
연봉 10억, 팀 평균 5천만+9.5억10명+1.06억
연봉 10억, 팀 평균 5천만+9.5억100명+960만
같은 편차(+30)라도 5명 반에서는 영향력 7.5, 30명 반에서는 1.03. 7배 차이. 이게 n의 힘이야.

그리고 세 번째 줄 — 이 문제. 심사위원 10점(최고점)이 6명 중에서 평균을 0.1 올리는 거. n=6이면 딱 성립하고, n=7이면 10.1점이 필요해서 불가능.
· · ·

내면화 체크:


"한 명이 빠졌는데 평균이 크게 변했다" → 직감적으로 "사람 적겠다" 떠오르면 내면화 된 거야.

"한 명이 빠졌는데 평균이 거의 안 변했다" → "사람 많겠다" 떠오르면 완성.


수식은 이 직감을 정확한 숫자로 바꿔줄 뿐이야.
