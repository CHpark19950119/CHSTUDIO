1장. 함수의 탄생 — 라이프니츠의 "functio"

1673년, 독일의 수학자 고트프리트 라이프니츠(Gottfried Leibniz)는 곡선의 성질을 연구하고 있었어. 곡선 위의 점마다 접선의 기울기가 달라지는데, 이걸 체계적으로 기술할 언어가 없었거든. 그래서 그는 라틴어 "functio"(수행, 작용)를 빌려와서 "곡선이 수행하는 역할"이라는 뜻으로 처음 사용했어.

라이프니츠가 함수라는 단어를 만든 건 우연이 아니야. 17세기는 과학혁명의 시대였어. 갈릴레오가 "자연은 수학의 언어로 쓰여 있다"고 선언한 뒤, 세상의 모든 변화를 수학으로 기술하려는 열망이 있었지. 그런데 "이 양이 변하면 저 양이 어떻게 변하는가?"를 표현할 도구가 마땅치 않았어.

핵심 통찰: 함수는 "변화하는 두 양 사이의 관계"를 정밀하게 포착하기 위해 탄생했어. 단순히 계산 도구가 아니라, 세상의 인과관계를 수학으로 번역하는 언어인 거야.

라이프니츠 이후, 레온하르트 오일러(1707-1783)가 함수 개념을 크게 확장했어. 오일러는 $f(x)$라는 표기법을 처음 사용한 사람이야. "f라는 규칙에 x를 넣는다" — 이 간결한 표기 덕분에 복잡한 관계도 한 줄로 쓸 수 있게 됐지.

그 뒤 19세기에 디리클레가 현대적 정의를 완성했어. "집합 X의 각 원소에 집합 Y의 원소를 정확히 하나씩 대응시키는 규칙" — 이게 지금 우리가 배우는 함수의 정의야.

자판기 비유를 생각해봐. 500원을 넣고 "콜라" 버튼을 누르면 콜라가 나오지. 매번 같은 입력에 같은 결과. 버튼 하나를 눌렀는데 어떤 때는 콜라, 어떤 때는 사이다가 나오면? 고장난 자판기야. 수학에서는 그런 걸 함수라고 부르지 않아.

2장. 함수의 정의 — 집합에서 집합으로

집합론적 정의

함수를 정확하게 정의하려면 세 가지 개념이 필요해.

$$f : X \to Y$$
$X$ : 정의역(domain) — 입력이 될 수 있는 모든 값의 집합
$Y$ : 공역(codomain) — 출력이 속할 수 있는 집합
$f(X)$ : 치역(range) — 실제로 나오는 출력값들의 집합

중요한 건 공역과 치역이 다를 수 있다는 거야. $f(x) = x^2$에서 공역을 "모든 실수"로 잡으면, 실제로 나오는 값(치역)은 $y \geq 0$뿐이지. 공역은 "나올 수 있는 범위", 치역은 "실제로 나오는 범위"야.

위 그림을 봐. $f(x) = x^2$일 때, 1은 1로, 2는 4로, 3은 9로 대응해. 그런데 $-1$도 1로 대응하지? 다른 입력이 같은 출력을 내는 건 괜찮아. 반대로, 하나의 입력이 두 개의 출력을 내면 함수가 아닌 거야.

대응의 종류

일대일 함수 (단사함수, injection): 다른 입력 → 반드시 다른 출력. $f(a) = f(b)$이면 $a = b$.
예시: $f(x) = 2x + 1$ — 다른 $x$를 넣으면 절대 같은 값이 안 나와.

다대일 함수: 여러 입력이 같은 출력을 낼 수 있음.
예시: $f(x) = x^2$ — $f(3) = f(-3) = 9$

일대일 대응 (전단사함수, bijection): 일대일 + 치역 = 공역. 완벽한 짝짓기.
예시: $f(x) = 2x + 1$ (공역이 모든 실수일 때) — 빠짐없이, 겹침없이 대응.

수직선 판정법

그래프만 보고 함수인지 판단하는 간단한 방법이 있어.

수직선 판정법: 그래프에 아무 수직선($x = k$)을 그었을 때,
교점이 최대 1개 → 함수 O
교점이 2개 이상 → 함수 X

원의 방정식 $x^2 + y^2 = r^2$은 함수가 아니야. 하나의 $x$값에 대해 $y$가 두 개 나오니까. 위쪽 반원 $y = \sqrt{r^2 - x^2}$이나 아래쪽 반원 $y = -\sqrt{r^2 - x^2}$으로 나눠야 각각이 함수가 돼.

정의역을 제한하는 3가지 상황

1. 분모가 0: $f(x) = \dfrac{1}{x-3}$ → $x \neq 3$

2. 짝수 근호 안이 음수: $f(x) = \sqrt{x - 2}$ → $x \geq 2$

3. 로그 진수가 0 이하: $f(x) = \log(x)$ → $x > 0$

3장. 일차함수 깊게 — 직선이 말하는 것들

편의점 알바를 한다고 해봐. 시급이 10,000원이야. 1시간 일하면 10,000원, 2시간이면 20,000원. 교통비로 30,000원이 기본 지급되고. 이 관계를 수식으로 쓰면:
$$f(x) = 10000x + 30000$$ $x$는 일한 시간, $f(x)$는 받는 돈. 이게 일차함수야.

일차함수의 구조

$$f(x) = ax + b$$ $a$ : 기울기 (slope) — 변화율
$b$ : $y$절편 (y-intercept) — $x = 0$일 때의 값

기울기의 물리적 의미

기울기 $a$는 "$x$가 1 늘어날 때 $y$가 얼마나 변하는가"를 나타내. 이걸 변화율(rate of change)이라고도 해.

기울기 = 변화율의 다양한 얼굴

시급: $f(x) = 10000x$ → 기울기 10,000 = "1시간당 10,000원"
속도: $s(t) = 60t$ → 기울기 60 = "1시간당 60km" (시속 60km)
환율: $w(d) = 1350d$ → 기울기 1,350 = "1달러당 1,350원"
택시 요금: $c(d) = 720d + 4800$ → 기울기 720 = "1km당 720원", 기본요금 4,800원

기울기의 부호가 알려주는 것:

$a > 0$ : 오른쪽 위로 올라감 (증가 — 돈이 늘어남, 거리가 늘어남)
$a < 0$ : 오른쪽 아래로 내려감 (감소 — 배터리가 줄어듦, 물이 빠짐)
$a = 0$ : 수평선 (변화 없음 — 상수함수 $f(x) = b$)

기울기 구하는 공식

두 점 $(x_1, y_1)$, $(x_2, y_2)$가 주어지면:

$$a = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\Delta y}{\Delta x}$$
예: 점 $(1, 3)$과 $(4, 9)$
$a = \dfrac{9-3}{4-1} = \dfrac{6}{3} = 2$

$\Delta$(델타)는 "변화량"이라는 뜻이야. $\Delta y / \Delta x$는 "$y$의 변화량을 $x$의 변화량으로 나눈 것" — 이게 나중에 미분에서 $dy/dx$가 돼. 일차함수에서는 변화율이 어디서든 동일하지만, 곡선에서는 점마다 변화율이 달라지거든. 그때 미분이 필요해지는 거야.

두 직선의 관계: 평행과 수직

평행 조건: 기울기가 같으면 평행
$y = 2x + 1$ 과 $y = 2x + 5$ → 기울기 둘 다 2 → 평행

수직 조건: 기울기의 곱이 $-1$
$y = 2x + 1$ 과 $y = -\dfrac{1}{2}x + 3$ → $2 \times (-\dfrac{1}{2}) = -1$ → 수직
왜 수직이면 기울기의 곱이 $-1$일까? 한 직선이 오른쪽 위로 $a$만큼 올라가면, 그 직선에 수직인 직선은 같은 거리만큼 오른쪽 아래로 내려가야 해. "올라가는 비율"의 역수에 부호를 뒤집은 것 — 그게 $-1/a$야.

직선의 방정식 3가지 형태

1. 기울기-절편형: $y = ax + b$ — 기울기와 y절편이 바로 보임

2. 점-기울기형: $y - y_1 = a(x - x_1)$ — 한 점 $(x_1, y_1)$과 기울기 $a$를 알 때

3. 일반형: $Ax + By + C = 0$ — 수직선($x = k$)도 표현 가능

4장. 이차함수 깊게 — 포물선의 모든 것

공을 위로 던져봐. 처음엔 쭉 올라가다가 어느 순간 멈추고, 다시 아래로 떨어지지? 이 궤적이 포물선(parabola)이야. 일차함수가 "꾸준한 변화"를 다룬다면, 이차함수는 "변화의 변화"를 다뤄.

세 가지 형태

1. 일반형(표준형): $f(x) = ax^2 + bx + c$
— $a$, $b$, $c$를 바로 읽을 수 있음. $y$절편 = $c$

2. 꼭짓점형: $f(x) = a(x - h)^2 + k$
— 꼭짓점 $(h, k)$가 바로 보임. 그래프 이동을 파악하기 좋음

3. 인수분해형: $f(x) = a(x - p)(x - q)$
— $x$절편 $p$, $q$가 바로 보임. 꼭짓점의 $x$좌표 = $\dfrac{p+q}{2}$

일반형 → 꼭짓점형: 왜 $-b/2a$ 인가?

이건 단순히 외울 공식이 아니야. 완전제곱식 만들기(completing the square)에서 자연스럽게 나와.

$f(x) = ax^2 + bx + c$ 에서 시작:

$= a\left(x^2 + \dfrac{b}{a}x\right) + c$    ($a$를 묶음)

$= a\left(x^2 + \dfrac{b}{a}x + \dfrac{b^2}{4a^2} - \dfrac{b^2}{4a^2}\right) + c$    (제곱 완성)

$= a\left(x + \dfrac{b}{2a}\right)^2 - \dfrac{b^2}{4a} + c$

$= a\left(x + \dfrac{b}{2a}\right)^2 + \dfrac{4ac - b^2}{4a}$
꼭짓점: $\left(-\dfrac{b}{2a},\ \dfrac{4ac - b^2}{4a}\right)$

$h = -\dfrac{b}{2a}$가 되는 이유: $x^2$의 계수를 1로 만들고 나서, $bx/a$ 항의 절반을 제곱하면 완전제곱식이 만들어지니까. 공식 뒤에는 "완전제곱식"이라는 논리가 있어.

판별식과 그래프의 관계

이차방정식 $ax^2 + bx + c = 0$의 판별식 $D = b^2 - 4ac$는 그래프가 $x$축과 몇 번 만나는지 알려줘.

$D = b^2 - 4ac$

$D > 0$ : 서로 다른 두 실근 → 그래프가 $x$축을 2번 관통
$D = 0$ : 중근 (같은 근 2개) → 그래프가 $x$축에 접함
$D < 0$ : 실근 없음 → 그래프가 $x$축과 안 만남

경제학 응용: 이윤 최대화

네가 카페를 운영한다고 해봐. 커피 가격을 $x$만원으로 설정하면, 판매량은 가격이 올라갈수록 줄어들어. 이윤(profit)을 함수로 만들면 이차함수가 돼.

가격이 너무 낮으면 — 많이 팔리지만 마진이 없어.
가격이 너무 높으면 — 마진은 좋지만 아무도 안 사.
어딘가 "딱 좋은 가격"이 있어. 그게 꼭짓점이야.
예시: 이윤 최대화

판매 수량: $q = 200 - 40p$ (가격 $p$가 올라가면 수량 감소)
단위당 비용: 1만원
이윤: $\pi = (p - 1) \cdot q = (p - 1)(200 - 40p)$

전개: $\pi = -40p^2 + 240p - 200$

꼭짓점: $p = -\dfrac{240}{2 \times (-40)} = 3$

가격 3만원일 때 이윤 최대!
$\pi(3) = -40(9) + 240(3) - 200 = -360 + 720 - 200 = 160$만원
비용 최소화도 마찬가지야. 비용 함수가 $C(x) = ax^2 + bx + c$ ($a > 0$)이면, 아래로 볼록한 포물선이니까 꼭짓점이 최소 비용을 알려줘. 경제학에서 이차함수가 핵심인 이유가 바로 이거야 — 최적화 문제의 기본 도구.

이차함수 그래프: $a$의 역할

$|a|$가 크면 → 그래프가 좁고 가파름 (급격한 변화)
$|a|$가 작으면 → 그래프가 넓고 완만함 (느린 변화)
$a > 0$ → 아래로 볼록 (U자) → 꼭짓점이 최솟값
$a < 0$ → 위로 볼록 (뒤집힌 U) → 꼭짓점이 최댓값

5장. 고차함수 맛보기 — 3차, 4차, 그 너머

일차함수는 직선, 이차함수는 포물선. 그럼 차수가 더 올라가면 어떻게 될까? 그래프가 점점 "꼬이기" 시작해. 차수가 올라갈수록 그래프가 방향을 바꿀 수 있는 횟수가 늘어나거든.

차수와 그래프의 "꼬임"

$n$차 함수의 그래프는 최대 $n - 1$번 방향을 바꿀 수 있어.

1차: 방향 전환 0번 → 직선
2차: 방향 전환 최대 1번 → 포물선 (한 번 꺾임)
3차: 방향 전환 최대 2번 → S자 곡선
4차: 방향 전환 최대 3번 → W자 또는 M자

3차함수: $f(x) = ax^3 + bx^2 + cx + d$

3차함수의 특징
- 양쪽 끝이 반대 방향으로 발산: $a > 0$이면 왼쪽은 $-\infty$, 오른쪽은 $+\infty$
- 변곡점(inflection point)이 정확히 1개 — 오목/볼록이 바뀌는 지점
- $x$축과의 교점이 1개 또는 3개 (실근이 1개 또는 3개)

3차함수의 가장 유명한 모양은 S자 곡선이야. 예를 들어 $f(x) = x^3 - 3x$를 그려보면, $x = -1$에서 극대, $x = 1$에서 극소를 가지는 S자 곡선이 나와.

끝 행동(End Behavior)

최고차항의 계수와 차수가 결정하는 것:

짝수 차수 + $a > 0$: 양쪽 다 $+\infty$ (U자)
짝수 차수 + $a < 0$: 양쪽 다 $-\infty$ (뒤집힌 U)
홀수 차수 + $a > 0$: 왼쪽 $-\infty$, 오른쪽 $+\infty$
홀수 차수 + $a < 0$: 왼쪽 $+\infty$, 오른쪽 $-\infty$
왜 차수가 올라가면 꼬일까? $x$가 아주 클 때는 최고차항이 지배하지만, 중간 영역에서는 낮은 차수 항들도 영향을 미쳐. 이 항들끼리 "밀고 당기면서" 방향 전환이 생기는 거야. 차수가 높을수록 항이 많아지니까 더 복잡한 모양이 가능해지는 거지.

6장. 분수함수와 무리함수 — 금지 영역의 수학

분수함수: 한없이 가까워지지만 닿지 않는

피자 한 판을 $n$명이 나눈다고 해봐. 2명이면 1/2, 10명이면 1/10, 100명이면 1/100. 사람이 많을수록 한 사람 몫은 줄어들지. 근데 아무리 많아져도 0이 되진 않아. 이게 분수함수 $f(n) = 1/n$의 핵심이야. 그리고 0명이서 나누는 건? 불가능하지. $n = 0$은 정의역에 없어.
기본형: $f(x) = \dfrac{k}{x}$

일반형: $f(x) = \dfrac{a}{x - p} + q$

수직 점근선: $x = p$ (분모 = 0이 되는 곳)
수평 점근선: $y = q$ ($x \to \pm\infty$일 때 다가가는 값)

점근선의 직관

점근선(asymptote)은 "영원히 닿지 못하는 벽"이야.

$f(x) = \dfrac{2}{x - 3} + 1$ 이면:

수직 점근선: $x = 3$ — 여기서 $f(x)$가 $\pm\infty$로 폭발
수평 점근선: $y = 1$ — $x$가 아주 커지면 $\dfrac{2}{x-3} \approx 0$이니까 $f(x) \approx 1$
그래프 중심: $(3, 1)$에서 기본 $1/x$ 그래프를 그린 것

무리함수: 제곱근의 세계

$$f(x) = \sqrt{x}$$
정의역: $x \geq 0$ (음수의 제곱근은 실수에서 정의 안 됨)
치역: $y \geq 0$
그래프: 원점에서 시작해서 오른쪽으로 완만하게 올라감

왜 정의역이 제한될까? $\sqrt{-4}$를 생각해봐. 제곱해서 $-4$가 되는 실수는 없어. $2^2 = 4$이고 $(-2)^2 = 4$이지, $-4$는 아니니까. 그래서 제곱근 안에 음수가 들어가면 실수 범위에서 함수가 정의되지 않는 거야.

정의역 제한이 생기는 이유 정리

- 분수함수: 분모 = 0이면 나눗셈이 불가능하니까 ($k/0$은 정의 안 됨)
- 무리함수: 짝수 근호 안이 음수면 실수 범위에서 값이 없으니까
- 로그함수: 진수가 0 이하면 로그가 정의 안 되니까

이 세 가지는 함수 문제에서 정의역을 물어볼 때 항상 체크하는 포인트야.
무리함수의 정의역 예시:

$f(x) = \sqrt{2x - 6}$ → $2x - 6 \geq 0$ → $x \geq 3$

$g(x) = \dfrac{1}{\sqrt{x + 1}}$ → $x + 1 > 0$ (분모라서 등호 제외) → $x > -1$

7장. 합성함수 — 함수를 연결하다

옷을 입는 순서를 생각해봐. 먼저 속옷을 입고($f$), 그 위에 겉옷을 입어($g$). $g(f(x))$야. 순서를 바꿔서 겉옷 먼저 입고 속옷을 입으면? $f(g(x))$ — 그건 작동하지 않지. 합성의 순서는 중요해.
합성함수 정의:

$(g \circ f)(x) = g(f(x))$

"$f$를 먼저 적용하고, 그 결과를 $g$에 넣는다"

읽는 순서와 실행 순서가 반대인 것에 주의!
$g \circ f$를 "g of f"라고 읽지만, f가 먼저 실행됨.

순서가 달라지면 결과도 달라진다

$f(x) = 2x$  (2배 하기)
$g(x) = x + 3$  (3 더하기)

$(g \circ f)(x) = g(f(x)) = g(2x) = 2x + 3$   → "2배 하고 3 더함"
$(f \circ g)(x) = f(g(x)) = f(x+3) = 2(x+3) = 2x + 6$   → "3 더하고 2배 함"

$x = 4$를 넣어보면:
$g(f(4)) = g(8) = 11$
$f(g(4)) = f(7) = 14$

결과가 다르다! $g \circ f \neq f \circ g$ (일반적으로)
"3을 더하고 2를 곱하는 것"과 "2를 곱하고 3을 더하는 것"은 다르다. 실생활에서도 마찬가지야. 할인 10%를 적용한 후 부가세 10%를 붙이는 것과, 부가세를 먼저 붙인 후 할인 10%를 적용하는 것 — 금액이 달라져.

합성함수의 정의역

$g(f(x))$의 정의역을 구하려면 두 가지를 동시에 만족해야 해:

1. $x$가 $f$의 정의역에 속해야 한다.
2. $f(x)$의 값이 $g$의 정의역에 속해야 한다.

예시: $f(x) = x - 4$, $g(x) = \sqrt{x}$

$g(f(x)) = \sqrt{x - 4}$

$f$의 정의역: 모든 실수 ✓
$g$의 정의역: $x \geq 0$ → $f(x) \geq 0$ 필요 → $x - 4 \geq 0$ → $x \geq 4$

실전에서의 합성함수

실생활 합성함수

환율 → 가격: 달러 가격을 원화로 → $g(f(x))$ 여기서 $f$: 달러→유로, $g$: 유로→원
온도 변환: 화씨 → 섭씨 → 켈빈 (두 변환 함수의 합성)
프로그래밍: 함수 파이프라인 — data |> parse |> validate |> save
경제학: 생산 함수 → 비용 함수 → 이윤 함수 (연쇄 관계)
합성 계산 연습:

$f(x) = x^2 - 1$, $g(x) = 3x + 2$

$(g \circ f)(4) = ?$
1단계: $f(4) = 16 - 1 = 15$
2단계: $g(15) = 45 + 2 = 47$

$(f \circ g)(4) = ?$
1단계: $g(4) = 12 + 2 = 14$
2단계: $f(14) = 196 - 1 = 195$

47 vs 195 — 순서 하나로 이렇게 차이나.

8장. 역함수 — "되돌리기" 버튼

신발을 신는 함수 $f$가 있으면, 신발을 벗는 함수 $f^{-1}$이 역함수야. 어떤 과정을 정확히 되돌리는 것. 암호화($f$)와 복호화($f^{-1}$), 출발($f$)과 귀환($f^{-1}$), 포장($f$)과 개봉($f^{-1}$).
역함수 정의:

$f$의 역함수 $f^{-1}$은 $f$의 효과를 되돌리는 함수.

$f(a) = b$   ⟺   $f^{-1}(b) = a$

$(f^{-1} \circ f)(x) = x$   (f를 적용하고 되돌리면 원래 값)
$(f \circ f^{-1})(x) = x$   (되돌리고 f를 적용해도 원래 값)

역함수가 존재하려면

모든 함수에 역함수가 있는 건 아니야. 역함수가 존재하려면 일대일 함수여야 해.

왜 일대일이어야 할까?

$f(x) = x^2$을 생각해봐. $f(3) = 9$이고 $f(-3) = 9$야.
그럼 $f^{-1}(9) = ?$ 3이야, -3이야? 답이 두 개라서 함수가 안 돼.

일대일 함수는 다른 입력이 다른 출력을 내니까, 되돌릴 때 혼란이 없어.

수평선 판정법: 그래프에 수평선을 그어서 교점이 최대 1개면 일대일 → 역함수 존재.

역함수 구하는 법

3단계 절차:

1. $y = f(x)$로 놓는다.
2. $x$와 $y$를 맞바꾼다.
3. $y$에 대해 푼다. → 그게 $f^{-1}(x)$

예시: $f(x) = 2x + 3$

1단계: $y = 2x + 3$
2단계: $x = 2y + 3$
3단계: $2y = x - 3$ → $y = \dfrac{x - 3}{2}$

$$f^{-1}(x) = \frac{x - 3}{2}$$

$y = x$ 대칭

역함수의 가장 멋진 성질: $f$와 $f^{-1}$의 그래프는 직선 $y = x$에 대해 대칭이야.

$x$와 $y$를 맞바꾸는 게 왜 대칭이 되는 걸까? 점 $(a, b)$에서 $x$와 $y$를 바꾸면 $(b, a)$가 돼. $(a, b)$와 $(b, a)$는 직선 $y = x$ 위의 점 $\left(\frac{a+b}{2}, \frac{a+b}{2}\right)$에서 같은 거리에 있어. 즉, $y = x$에 대해 거울상이 되는 거야.

9장. 그래프 변환 — 밀고 당기고 뒤집기

원래 함수 $f(x)$의 그래프를 알면, 간단한 수식 변형만으로 그래프를 이동하고 뒤집고 늘릴 수 있어. 마치 사진 편집처럼 — 이동(translate), 반전(reflect), 확대/축소(scale).

평행이동 (Translation)

상하 이동: $f(x) + k$
$k > 0$ → 위로 $k$만큼 ($y$ 값 전체에 $k$를 더함)
$k < 0$ → 아래로 $|k|$만큼

좌우 이동: $f(x - h)$
$h > 0$ → 오른쪽으로 $h$만큼 (부호 반대!)
$h < 0$ → 왼쪽으로 $|h|$만큼
왜 $f(x - h)$가 오른쪽 이동일까?

직관에 반하는 것 같지만 논리적이야. $f(x)$에서 $x = 0$일 때의 값을 $f(x - 3)$에서 얻으려면 $x - 3 = 0$, 즉 $x = 3$이어야 해. 원래 $x = 0$에 있던 값이 $x = 3$으로 이동한 거야. "$x$에서 3을 빼는 것"은 "원래보다 3만큼 뒤에서 같은 값을 내라"는 뜻이니까 오른쪽으로 밀리는 거지.

대칭 (Reflection)

$x$축 대칭: $-f(x)$ → $y$ 값의 부호를 뒤집음 (위아래 반전)

$y$축 대칭: $f(-x)$ → $x$ 값의 부호를 뒤집음 (좌우 반전)

원점 대칭: $-f(-x)$ → 좌우 + 위아래 동시 반전

확대/축소 (Scaling)

수직 스케일: $a \cdot f(x)$
$|a| > 1$ → 세로로 늘림 (더 가파르게)
$0 < |a| < 1$ → 세로로 압축 (더 완만하게)

수평 스케일: $f(bx)$
$|b| > 1$ → 가로로 압축 (더 빠르게 변화) — 부호 반대!
$0 < |b| < 1$ → 가로로 늘림 (더 느리게 변화)
변환 종합 예시:

$y = -2(x + 1)^2 + 3$ 은 $y = x^2$에서:
1. 왼쪽으로 1 이동 ($x + 1$)
2. 세로로 2배 늘림 (계수 2)
3. $x$축 대칭 (마이너스)
4. 위로 3 이동 ($+3$)

변환 순서: 괄호 안(수평) → 계수(수직 스케일) → 부호(반전) → 상수(수직 이동)

10장. 왜 함수가 중요한가 — 세상을 읽는 렌즈

함수는 "수학의 언어"야. 물리학, 경제학, 컴퓨터 과학, 심지어 심리학까지 — 모든 분야에서 "이것이 변하면 저것이 어떻게 변하는가?"를 기술하는 도구가 함수야. 함수 없이는 과학적 사고 자체가 불가능해.

미적분과의 연결

미분: 함수의 순간 변화율을 구하는 것
- 일차함수의 기울기 → 어디서든 동일 → 미분할 필요 없음
- 이차함수의 기울기 → 점마다 다름 → 미분이 필요!
- $f(x) = x^2$ → $f'(x) = 2x$ → "$x$가 클수록 기울기가 가파르다"

적분: 함수의 누적량을 구하는 것
- 속도 함수를 적분하면 → 이동 거리
- 수입 함수를 적분하면 → 총 수입
- 그래프 아래 면적 = 적분값

지금 배우는 함수의 모든 개념 — 정의역, 기울기, 꼭짓점, 점근선, 합성, 역함수 — 이것들이 미적분의 기초 언어야. 미적분은 "함수가 어떻게 변하는가"를 정밀하게 분석하는 도구인데, 함수 자체를 모르면 미적분이 불가능하지.

경제학과의 연결

수요 함수: $Q_d = f(P)$ — 가격이 올라가면 수요량이 줄어듦 (감소 함수)

공급 함수: $Q_s = g(P)$ — 가격이 올라가면 공급량이 늘어남 (증가 함수)

균형: $f(P) = g(P)$ → 수요 = 공급이 되는 가격 = 균형 가격

비용 함수: $C(q) = aq^2 + bq + c$ → 이차함수 (규모의 비경제)

한계 비용: $MC(q) = C'(q) = 2aq + b$ → 비용 함수의 미분 = 일차함수

물리학과의 연결

위치: $s(t)$ — 시간의 함수
속도: $v(t) = s'(t)$ — 위치의 미분 (순간 변화율)
가속도: $a(t) = v'(t) = s''(t)$ — 속도의 미분

자유낙하: $s(t) = \frac{1}{2}gt^2$ (이차함수)
→ $v(t) = gt$ (일차함수: 속도가 일정하게 증가)
→ $a(t) = g$ (상수함수: 중력가속도 9.8 m/s²)

함수적 사고

함수를 배우는 진짜 이유는 공식을 외우는 게 아니야. "입력과 출력의 관계를 파악하는 사고방식"을 익히는 거야. 가격을 올리면 매출이 어떻게 될까? 공부 시간을 늘리면 성적이 얼마나 오를까? 운동을 하면 체중이 어떻게 변할까? — 이 모든 질문이 본질적으로 함수적 질문이야.
함수의 세계관

1. 일차함수 — 변화가 일정한 세계 (단순한 인과관계)
2. 이차함수 — 최적점이 있는 세계 (최대/최소화 문제)
3. 고차함수 — 복잡하게 꼬이는 세계 (비선형 현상)
4. 분수함수 — 한계와 점근선이 있는 세계 (수확체감, 한계효용)
5. 무리함수 — 금지 영역이 있는 세계 (물리적 제약)
6. 합성함수 — 원인의 연쇄가 있는 세계 (복잡한 인과관계)
7. 역함수 — 되돌릴 수 있는 세계 (가역적 과정)

각 함수 유형은 세상의 다른 패턴을 포착하는 렌즈야.

퀴즈 — 15문제 실력 테스트

답을 누르면 정답과 해설이 나와. 집중해서 풀어봐.

Q1. $f(x) = 3x - 2$ 일 때, $f(5)$ 의 값은?

$f(5) = 3(5) - 2 = 15 - 2 = 13$. $x$에 5를 그대로 대입하면 돼.

Q2. 일차함수에서 기울기 $a$의 의미는?

기울기는 "$x$가 1 늘어날 때 $y$가 얼마나 변하는가"를 나타내. $y$절편 $b$가 $y$축과 만나는 점이야.

Q3. 다음 중 함수인 것은?

$f(x) = x^2$은 하나의 $x$에 대해 정확히 하나의 $y$가 나오니까 함수야. 나머지는 하나의 $x$에 $y$가 2개 나올 수 있어서 함수가 아니야.

Q4. $f(x) = -2x^2 + 8x - 3$의 꼭짓점 $x$좌표는?

$x = -\dfrac{b}{2a} = -\dfrac{8}{2 \times (-2)} = -\dfrac{8}{-4} = 2$. 완전제곱식으로 유도한 공식이야.

Q5. $f(x) = \dfrac{1}{x - 2}$ 의 수직 점근선은?

분모가 0이 되는 $x$값이 수직 점근선이야. $x - 2 = 0$ → $x = 2$.

Q6. $f(x) = \sqrt{3x - 6}$의 정의역은?

$3x - 6 \geq 0$ → $3x \geq 6$ → $x \geq 2$. 등호 포함이니까 $x \geq 2$.

Q7. $f(x) = x + 1$, $g(x) = 2x$ 일 때, $g(f(3))$은?

안쪽부터! $f(3) = 3 + 1 = 4$. 그 다음 $g(4) = 2 \times 4 = 8$.

Q8. $y = 2(x-3)^2 + 5$의 최솟값은?

$a = 2 > 0$이니까 아래로 볼록. $(x-3)^2 \geq 0$이므로 최솟값은 $x = 3$일 때 $y = 5$.

Q9. 두 점 $(1, 4)$와 $(3, 10)$을 지나는 직선의 기울기는?

$a = \dfrac{10-4}{3-1} = \dfrac{6}{2} = 3$.

Q10. $f(x) = \dfrac{3}{x+1} + 2$ 의 수평 점근선은?

$\dfrac{a}{x-p}+q$ 형태에서 수평 점근선은 $y = q$. $x$가 매우 커지면 $\frac{3}{x+1} \approx 0$이니까 $y \approx 2$.

Q11. $f(x) = 2x + 3$ 의 역함수 $f^{-1}(x)$는?

$y = 2x + 3$ → $x = 2y + 3$ → $2y = x - 3$ → $y = \dfrac{x-3}{2}$. $x$와 $y$를 바꾸고 $y$에 대해 풀면 돼.

Q12. $y = x^2$의 그래프를 오른쪽으로 4, 위로 1 이동하면?

오른쪽 $h$ 이동 = $f(x - h)$, 위로 $k$ 이동 = $f(x) + k$. 따라서 $(x - 4)^2 + 1$.

Q13. 두 직선 $y = 3x + 1$과 $y = -\frac{1}{3}x + 5$의 관계는?

$3 \times (-\frac{1}{3}) = -1$. 기울기의 곱이 $-1$이면 수직이야.

Q14. 3차함수 $f(x) = x^3$의 그래프는 최대 몇 번 방향을 바꿀 수 있어?

$n$차 함수는 최대 $n-1$번 방향 전환 가능. 3차 → 최대 2번. (다만 $x^3$ 자체는 방향 전환 없이 S자로 지나감 — "최대"가 핵심)

Q15. 이차함수 $f(x) = x^2 - 6x + 5$의 판별식 $D$는? 그리고 $x$축과의 교점 개수는?
