1

함수의 극한 기초

1-1. 극한의 직관적 의미

"극한"이라는 말은 "한없이 가까이 다가간다"는 뜻이야. 핵심은 도달하느냐가 아니라, 다가가는 과정에서 함수값이 어디로 모이느냐야.

예를 들어, f(x) = 2x에서 x를 3에 가까이 보내면 f(x)는 6에 가까워져. 이걸 수학적으로 이렇게 쓰는 거야:

극한의 표기 limx→3 2x = 6

여기서 x는 3에 "도달"하지 않아도 돼. x = 2.9, 2.99, 2.999, ... 처럼 점점 가까이 가기만 하면 되는 거야. 반대쪽에서도 x = 3.1, 3.01, 3.001, ... 이렇게 다가올 수 있어.

핵심 포인트

극한은 "그 점에서의 함수값"이 아니야. "그 점 근처에서 함수값이 모이는 곳"이야. 함수가 그 점에서 정의되지 않아도 극한은 존재할 수 있어.

a
x → a−
x → a+
f(x) → L
x가 양쪽에서 a로 다가가면, f(x)는 L로 수렴

1-2. limx→a f(x) = L 의 정확한 의미

수학II 수준에서의 정의는 이래:

정의 x가 a와 다른 값을 가지면서 a에 한없이 가까워질 때,
f(x)가 일정한 값 L에 한없이 가까워지면
"x가 a로 갈 때 f(x)의 극한값은 L이다"라고 한다.

핵심을 정리하면:

  1. x ≠ a — x는 a가 아닌 상태로 다가감 (a에 도달하지 않음)
  2. 양쪽 모두 — 왼쪽(a보다 작은 쪽)과 오른쪽(a보다 큰 쪽) 둘 다에서 다가가야 함
  3. 같은 값 L — 양쪽에서 다가갔을 때 f(x)가 모이는 값이 같아야 함

1-3. 좌극한과 우극한

좌극한 (왼쪽 극한) limx→a− f(x) = L1
x가 a보다 작은 쪽에서 a로 다가갈 때의 극한
우극한 (오른쪽 극한) limx→a+ f(x) = L2
x가 a보다 큰 쪽에서 a로 다가갈 때의 극한
극한 존재 조건 limx→a f(x) = L   ⇔   limx→a− f(x) = limx→a+ f(x) = L

즉, 좌극한과 우극한이 같을 때만 극한이 존재해. 이건 극한 존재 판정의 가장 기본적인 도구야.

예제: 좌극한 ≠ 우극한

다음 함수에서 x → 0일 때 극한을 조사해라.

f(x) = { −1  (x < 0) ,   1  (x > 0) }
풀이 보기
STEP 1

좌극한: x < 0에서 접근하면 f(x) = −1이므로

limx→0− f(x) = −1

STEP 2

우극한: x > 0에서 접근하면 f(x) = 1이므로

limx→0+ f(x) = 1

결론

−1 ≠ 1 이므로 극한이 존재하지 않는다.

1-4. 극한값이 존재하지 않는 3가지 경우

극한 부존재 Case 1: 좌극한 ≠ 우극한

왼쪽에서 다가간 값과 오른쪽에서 다가간 값이 다르면 극한은 존재하지 않아.

예: f(x) = |x|/x에서 x → 0
좌극한 = −1, 우극한 = 1  ⇒  극한 없음

극한 부존재 Case 2: 발산 (±∞)

함수값이 한없이 커지거나 작아지면 극한은 존재하지 않아. 수학에서 "극한값이 존재한다"는 건 실수값 L이 있어야 한다는 뜻이야. ∞는 실수가 아니야.

예: f(x) = 1/x²에서 x → 0
양쪽 모두 +∞로 발산  ⇒  극한 없음 (비록 양쪽이 같은 방향이라도!)

극한 부존재 Case 3: 진동

함수값이 특정 값에 수렴하지 않고 계속 흔들리면 극한은 존재하지 않아.

예: f(x) = sin(1/x)에서 x → 0
x가 0에 가까워질수록 1/x가 커져서 sin이 −1과 1 사이를 미친 듯이 진동  ⇒  극한 없음

1-5. 극한값 ≠ 함수값인 경우

이건 정말 중요한 개념이야. 극한값과 함수값은 완전히 별개의 것이야.

예제: 극한값과 함수값이 다른 경우
f(x) = { (x²−1)/(x−1)  (x ≠ 1) ,   5  (x = 1) }
풀이 보기
STEP 1

x ≠ 1일 때 인수분해:

f(x) = (x²−1)/(x−1) = (x+1)(x−1)/(x−1) = x+1

STEP 2

극한값: limx→1 f(x) = limx→1 (x+1) = 2

STEP 3

함수값: f(1) = 5

결론

극한값은 2이고, 함수값은 5. 서로 달라!

극한을 구할 때는 x = 1이 아닌 근방에서의 행동만 보기 때문이야.

자주 하는 실수

"극한값 구하라"는 문제에 f(a)를 그냥 대입하면 안 돼. 특히 구분적 함수(piecewise)에서는 x ≠ a인 부분의 식으로 극한을 구해야 해.

Section 1 핵심 요약
  • 극한 = "다가가는 과정에서 값이 모이는 곳" (도달 아님)
  • 좌극한 = 우극한일 때만 극한 존재
  • 극한 부존재: 좌 ≠ 우, 발산(∞), 진동
  • 극한값 ≠ 함수값 가능
2

극한의 성질 (사칙연산)

limx→a f(x) = α, limx→a g(x) = β (α, β는 실수)일 때, 다음이 성립해:

극한의 사칙연산 성질 ① limx→a c = c   (c는 상수)
② limx→a [c · f(x)] = c · α
③ limx→a [f(x) ± g(x)] = α ± β
④ limx→a [f(x) · g(x)] = α · β
⑤ limx→a [f(x) / g(x)] = α / β   (단, β ≠ 0)
⑥ limx→a [f(x)]n = αn   (n은 양의 정수)
전제 조건을 반드시 확인!

위 성질들은 α와 β가 실수일 때, 즉 각각의 극한이 존재할 때만 쓸 수 있어. 극한이 존재하지 않으면 (발산하거나 진동하면) 이 공식들을 직접 적용할 수 없어.

2-1. 성질 적용 예제

예제 1: 사칙연산 직접 적용
limx→2 (3x² − 2x + 1) 을 구하라.
풀이 보기
STEP 1

다항함수의 극한은 직접 대입하면 돼 (연속함수니까):

limx→2 (3x² − 2x + 1) = 3(4) − 2(2) + 1 = 12 − 4 + 1 = 9

STEP 2

성질로 분해하면:

= 3 · limx→2 x² − 2 · limx→2 x + limx→2 1

= 3(4) − 2(2) + 1 = 9

예제 2: 나눗셈 성질 적용
limx→3 (x² + 1) / (x − 1) 을 구하라.
풀이 보기
STEP 1

분모 확인: limx→3 (x−1) = 2 ≠ 0 ✔

STEP 2

성질 ⑤ 적용 가능:

= lim(x²+1) / lim(x−1) = 10/2 = 5

2-2. 0/0 부정형 — 성질 직접 적용 불가!

분모의 극한이 0이면 성질 ⑤를 직접 쓸 수 없어. 특히 분자의 극한도 0이면 0/0 형태가 되는데, 이걸 "부정형"이라고 해.

주의! 부정형 분자 → 0, 분모 → 0 일 때
lim [f(x)/g(x)] ≠ lim f(x) / lim g(x)
⇒ 변형 후 다시 계산해야 함!

부정형은 "답이 정해지지 않았다"는 뜻이야. 0일 수도, 1일 수도, ∞일 수도 있어. 그래서 인수분해, 유리화 등으로 부정형을 해소한 뒤에 극한을 구해야 해. 이건 다음 섹션에서 아주 자세히 다룰 거야.

3

부정형 해결법 (가장 중요!)

이 섹션이 수학II 극한의 핵심이야

시험 문제의 70% 이상이 부정형 처리를 요구해. 여기를 완벽히 마스터하면 극한 단원은 끝이야.

3-1. 0/0 꼴: 인수분해

가장 기본적이고 가장 많이 나오는 유형이야. 분자와 분모가 모두 0이 되면, 공통인수 (x − a)가 반드시 있어. 그걸 약분하면 돼.

대입 → 0/0 확인
→
분자·분모 인수분해
→
(x−a) 약분
→
다시 대입
예제 1: 기본 인수분해
limx→2 (x² − 4) / (x − 2)
풀이 보기
STEP 1

대입 확인: 분자 = 4 − 4 = 0, 분모 = 0  ⇒  0/0 부정형

STEP 2

분자 인수분해: x² − 4 = (x+2)(x−2)

STEP 3

약분:

(x+2)(x−2) / (x−2) = x + 2   (x ≠ 2이므로 약분 가능)

결론

limx→2 (x+2) = 4

예제 2: 고차 다항식 인수분해
limx→1 (x³ − 1) / (x² − 1)
풀이 보기
STEP 1

대입: 분자 = 0, 분모 = 0  ⇒  0/0 부정형

STEP 2

인수분해:

x³ − 1 = (x−1)(x²+x+1)   (세제곱 차 공식)

x² − 1 = (x−1)(x+1)

STEP 3

약분: (x−1)(x²+x+1) / (x−1)(x+1) = (x²+x+1)/(x+1)

결론

x = 1 대입: (1+1+1)/(1+1) = 3/2

답: 3/2

예제 3: 조립제법 활용
limx→−1 (x³ + 3x² + 3x + 1) / (x² + 2x + 1)
풀이 보기
STEP 1

대입: 분자 = −1+3−3+1 = 0, 분모 = 1−2+1 = 0  ⇒  0/0 부정형

STEP 2

패턴 인식: 분자 = (x+1)³, 분모 = (x+1)²

(못 알아보겠으면 조립제법으로 (x+1)을 뽑아내도 돼)

STEP 3

약분: (x+1)³ / (x+1)² = x+1

결론

limx→−1 (x+1) = 0

인수분해 공식 모음
  • a² − b² = (a−b)(a+b)
  • a³ − b³ = (a−b)(a²+ab+b²)
  • a³ + b³ = (a+b)(a²−ab+b²)
  • x² + (a+b)x + ab = (x+a)(x+b)
  • 인수분해가 안 보이면 조립제법을 써라. x = a가 근이면 (x − a)로 나뉨.

3-2. 0/0 꼴: 유리화 (루트가 있을 때)

분자나 분모에 루트(√)가 있어서 인수분해가 안 될 때는 유리화를 써. 켤레식(conjugate)을 곱해서 루트를 없애는 거야.

유리화 핵심 공식 (√A − √B)(√A + √B) = A − B
(√A − B)(√A + B) = A − B²
0/0 확인
→
켤레식 곱하기 (분자·분모 동시)
→
루트 제거
→
약분 후 대입
예제 1: 분자 유리화
limx→0 (√(x+1) − 1) / x
풀이 보기
STEP 1

대입: 분자 = √1 − 1 = 0, 분모 = 0  ⇒  0/0 부정형

STEP 2

분자에 루트가 있으니 분자를 유리화. 켤레식 (√(x+1) + 1)을 분자·분모에 곱해:

= [(√(x+1) − 1)(√(x+1) + 1)] / [x(√(x+1) + 1)]

STEP 3

분자 정리: (√(x+1))² − 1² = (x+1) − 1 = x

= x / [x(√(x+1) + 1)]

STEP 4

x 약분: = 1 / (√(x+1) + 1)

결론

x = 0 대입: 1/(√1 + 1) = 1/2

답: 1/2

예제 2: 분모 유리화
limx→4 (x − 4) / (√x − 2)
풀이 보기
STEP 1

대입: 분자 = 0, 분모 = 2 − 2 = 0  ⇒  0/0 부정형

STEP 2

분모에 루트가 있으니 분모를 유리화. 켤레식 (√x + 2)를 곱해:

= (x−4)(√x + 2) / [(√x − 2)(√x + 2)]

STEP 3

분모 정리: (√x)² − 4 = x − 4

= (x−4)(√x + 2) / (x − 4)

STEP 4

약분: = √x + 2

결론

x = 4 대입: √4 + 2 = 2 + 2 = 4

답: 4

예제 3: 이중 루트 유리화
limx→0 (√(x+4) − √(4−x)) / x
풀이 보기
STEP 1

대입: 분자 = √4 − √4 = 0, 분모 = 0  ⇒  0/0 부정형

STEP 2

분자의 켤레식 (√(x+4) + √(4−x))를 곱해:

= [(x+4) − (4−x)] / [x(√(x+4) + √(4−x))]

STEP 3

분자 정리: x + 4 − 4 + x = 2x

= 2x / [x(√(x+4) + √(4−x))]

STEP 4

약분: = 2 / (√(x+4) + √(4−x))

결론

x = 0 대입: 2 / (√4 + √4) = 2/4 = 1/2

답: 1/2

3-3. 0/0 꼴: 통분

분수 안에 분수가 있는 복잡한 식에서 0/0이 나오면, 통분으로 정리해야 해.

예제 1: 분수의 극한 — 통분
limx→0 (1/(x+2) − 1/2) / x
풀이 보기
STEP 1

대입: 분자 = 1/2 − 1/2 = 0, 분모 = 0  ⇒  0/0 부정형

STEP 2

분자 통분:

1/(x+2) − 1/2 = [2 − (x+2)] / [2(x+2)] = −x / [2(x+2)]

STEP 3

원래 식 정리:

= [−x / (2(x+2))] / x = −x / [2(x+2) · x]

STEP 4

x 약분: = −1 / [2(x+2)]

결론

x = 0 대입: −1/(2·2) = −1/4

답: −1/4

예제 2: 좀 더 복잡한 통분
limx→1 [1/(x−1) − 2/(x²−1)]
풀이 보기
STEP 1

직접 대입하면 ∞ − ∞ 형태 (이것도 부정형!)

STEP 2

x²−1 = (x−1)(x+1)이니까 공통분모로 통분:

= (x+1)/[(x−1)(x+1)] − 2/[(x−1)(x+1)]

= [(x+1) − 2] / [(x−1)(x+1)]

= (x−1) / [(x−1)(x+1)]

STEP 3

약분: = 1/(x+1)

결론

limx→1 1/(x+1) = 1/2

예제 3: 다항식 + 통분 혼합
limx→3 [x/(x−3) − 9/(x²−3x)]
풀이 보기
STEP 1

x²−3x = x(x−3) 이므로 공통분모 = x(x−3)

STEP 2

통분:

= x²/[x(x−3)] − 9/[x(x−3)]

= (x²−9)/[x(x−3)]

= (x+3)(x−3)/[x(x−3)]

STEP 3

약분: = (x+3)/x

결론

x = 3 대입: 6/3 = 2

3-4. ∞/∞ 꼴: 최고차항으로 나누기

x → ∞ (또는 x → −∞)에서 분자·분모가 모두 ∞로 가면 ∞/∞ 부정형이야. 해결법: 최고차항으로 분자·분모를 나눈다.

∞/∞ 해결 공식 limx→∞ (anxn + ...) / (bmxm + ...) 에서

n < m  ⇒  0
n = m  ⇒  an/bm (최고차 계수의 비)
n > m  ⇒  ±∞ (발산)

이 세 가지 결과를 외우면 기계적으로 풀 수 있어. 하지만 원리를 이해하자.

예제 1: 차수가 같은 경우 (n = m)
limx→∞ (3x² + 2x − 1) / (2x² − x + 5)
풀이 보기
STEP 1

최고차항 = x². 분자·분모를 x²으로 나눔:

= (3 + 2/x − 1/x²) / (2 − 1/x + 5/x²)

STEP 2

x → ∞이면 2/x → 0, 1/x² → 0 등 전부 0:

= (3 + 0 − 0) / (2 − 0 + 0) = 3/2

결론

답: 3/2 (최고차 계수의 비!)

예제 2: 분자 차수가 낮은 경우 (n < m)
limx→∞ (x + 3) / (x² + 1)
풀이 보기
STEP 1

최고차항 = x² (분모). 분자·분모를 x²으로 나눔:

= (1/x + 3/x²) / (1 + 1/x²)

STEP 2

x → ∞: = (0 + 0)/(1 + 0) = 0

결론

답: 0 (분자 차수 < 분모 차수 ⇒ 항상 0)

예제 3: 분자 차수가 높은 경우 (n > m)
limx→∞ (2x³ − x) / (x² + 3)
풀이 보기
STEP 1

분자·분모를 x²으로 나눔:

= (2x − 1/x) / (1 + 3/x²)

STEP 2

x → ∞: 분자 → ∞, 분모 → 1

= ∞/1 = ∞

결론

발산 (+∞) — 극한값이 존재하지 않음

예제 4: 루트가 있는 ∞/∞
limx→∞ √(4x² + 1) / (3x − 2)
풀이 보기
STEP 1

√(4x² + 1) = √(x²(4 + 1/x²)) = |x|√(4 + 1/x²)

x → +∞이므로 |x| = x:

= x√(4 + 1/x²)

STEP 2

원래 식:

= x√(4 + 1/x²) / (3x − 2)

분자·분모를 x로 나눔:

= √(4 + 1/x²) / (3 − 2/x)

결론

x → ∞: √(4+0)/(3−0) = 2/3

답: 2/3

3-5. ∞ − ∞ 꼴

두 개의 무한히 큰 양을 뺄 때 결과가 불확정인 형태야. 유리화 또는 통분으로 해결해.

예제 1: ∞ − ∞ 유리화
limx→∞ (√(x² + 3x) − x)
풀이 보기
STEP 1

대입: √(∞) − ∞ = ∞ − ∞ 부정형

STEP 2

켤레식 (√(x²+3x) + x)를 곱해 유리화:

= [(√(x²+3x) − x)(√(x²+3x) + x)] / (√(x²+3x) + x)

= [(x²+3x) − x²] / (√(x²+3x) + x)

= 3x / (√(x²+3x) + x)

STEP 3

x로 나눔 (x > 0):

= 3 / (√(1+3/x) + 1)

결론

x → ∞: 3/(√1 + 1) = 3/2

답: 3/2

예제 2: ∞ − ∞ 유리화 (계수 다름)
limx→∞ (√(4x² + x) − 2x)
풀이 보기
STEP 1

켤레식 곱하기:

= [(4x²+x) − 4x²] / (√(4x²+x) + 2x) = x / (√(4x²+x) + 2x)

STEP 2

x로 나눔:

= 1 / (√(4+1/x) + 2)

결론

x → ∞: 1/(2+2) = 1/4

예제 3: ∞ − ∞ 통분형
limx→∞ x · [1/x − 1/(x+1)]
풀이 보기
STEP 1

괄호 안 통분: 1/x − 1/(x+1) = 1/[x(x+1)]

STEP 2

= x · 1/[x(x+1)] = 1/(x+1)

결론

x → ∞: 1/∞ = 0

부정형 해결 전략 총정리
부정형상황해결법
0/0다항식인수분해 후 약분
0/0루트 포함유리화 (켤레식)
0/0복합 분수통분 후 약분
∞/∞분수함수최고차항으로 나누기
∞−∞루트 포함유리화
∞−∞분수끼리 뺄셈통분
4

미정계수 결정

극한값이 주어지고 함수의 미지수를 구하는 유형이야. 수능/내신에 빈출되는 핵심 유형이야.

4-1. 핵심 원리

미정계수 결정의 핵심 정리 limx→a f(x)/g(x) = L (L은 실수, L ≠ 0)이고
g(a) = 0 이면  ⇒  반드시 f(a) = 0

왜?

증명 (이해용)

만약 g(a) = 0인데 f(a) ≠ 0이면,
분모 → 0, 분자 → (0이 아닌 수) 이므로 극한은 ±∞로 발산해.
그런데 극한값이 실수 L이라고 했으니 모순이야.
따라서 f(a) = 0이어야만 해 (0/0 부정형이 되어야 유한한 극한이 가능).

주의: L = 0인 경우

극한값이 0이면 f(a) = 0이 반드시 성립하는 건 아니야. 예: limx→1 (x−1)/1 = 0인데 분모 g(1) = 1 ≠ 0. 이 경우 f(a) = 0이긴 하지만, 그건 부정형 때문이 아니라 단순 대입 결과야.

정확히 말하면: 분모가 0이 되는데 극한이 유한하면 ⇒ 분자도 반드시 0.

4-2. 미정계수 문제 풀이 절차

극한값 유한 + 분모 = 0 확인
→
분자 = 0 조건으로 미지수 1개 결정
→
0/0 부정형 풀기
→
극한값 = L에서 나머지 미지수 결정
예제 1: 기본 미정계수
limx→1 (x² + ax + b) / (x − 1) = 3 일 때, a, b의 값을 구하라.
풀이 보기
STEP 1

분모에 x = 1 대입: 1 − 1 = 0

극한값이 3 (유한)  ⇒  분자도 0이어야 함

STEP 2

분자에 x = 1 대입:

1 + a + b = 0  ⇒  a + b = −1   ... ①

STEP 3

분자 = 0인 조건을 알았으니, 분자는 (x − 1)을 인수로 가져.

x² + ax + b = (x − 1)(x + c) 형태. 전개하면:

= x² + (c−1)x − c

비교: a = c − 1, b = −c

STEP 4

약분 후 극한 계산:

limx→1 (x−1)(x+c)/(x−1) = limx→1 (x+c) = 1 + c = 3

⇒ c = 2

결론

a = c − 1 = 1, b = −c = −2

a = 1, b = −2

검산: 1 + (−2) = −1 ✔ (①식 만족)

예제 2: 루트 + 미정계수
limx→0 (√(ax + 4) − b) / x = 1 일 때, a, b의 값을 구하라.
풀이 보기
STEP 1

분모 x = 0 대입: 0. 극한값 1 (유한) ⇒ 분자도 0이어야 함.

STEP 2

분자에 x = 0: √(0+4) − b = 2 − b = 0

⇒ b = 2

STEP 3

b = 2를 대입하고 유리화:

limx→0 (√(ax+4) − 2) / x

켤레식 곱하기:

= lim [(ax+4) − 4] / [x(√(ax+4) + 2)]

= lim ax / [x(√(ax+4) + 2)]

= lim a / (√(ax+4) + 2)

STEP 4

x = 0 대입: a / (√4 + 2) = a/4 = 1

⇒ a = 4

결론

a = 4, b = 2

예제 3: ∞에서의 미정계수
limx→∞ (√(x² + ax) − x) = 5 일 때, a의 값을 구하라.
풀이 보기
STEP 1

∞ − ∞ 꼴이므로 유리화:

= [(x²+ax) − x²] / (√(x²+ax) + x)

= ax / (√(x²+ax) + x)

STEP 2

x로 나눔:

= a / (√(1+a/x) + 1)

STEP 3

x → ∞: a / (√1 + 1) = a/2 = 5

결론

a = 10

예제 4: 복합 미정계수
limx→2 (x² + ax + b) / (x² − 4) = −1 일 때, a + b의 값을 구하라.
풀이 보기
STEP 1

분모: x²−4 = (x−2)(x+2), x = 2 대입시 0. 극한값 유한 ⇒ 분자도 0.

4 + 2a + b = 0  ⇒  2a + b = −4   ... ①

STEP 2

분자는 (x − 2)를 인수로 가짐: x²+ax+b = (x−2)(x+c)

전개: x²+(c−2)x−2c ⇒ a = c−2, b = −2c

STEP 3

약분 후 극한:

limx→2 (x−2)(x+c) / [(x−2)(x+2)] = (2+c)/(2+2) = (2+c)/4 = −1

2 + c = −4  ⇒  c = −6

결론

a = −6−2 = −8, b = −2(−6) = 12

a + b = −8 + 12 = 4

5

함수의 연속

5-1. 연속함수의 정의

극한의 개념을 배웠으니, 이제 "연속"이 뭔지 정확히 알 수 있어. 직관적으로 "그래프를 끊지 않고 한 번에 그릴 수 있다"는 거야.

함수 f(x)가 x = a에서 연속이다 limx→a f(x) = f(a)

이 한 줄에 사실 3가지 조건이 숨어있어:

5-2. 연속의 3조건

연속의 3조건 (모두 만족해야 연속) ① f(a)가 정의되어 있다 (함수값이 존재)
② limx→a f(x)가 존재한다 (극한값이 존재)
③ limx→a f(x) = f(a) (극한값 = 함수값)

하나라도 안 되면 불연속이야. 각각의 위반 사례를 보자:

불연속 유형 1: ① 위반 — f(a) 미정의

예: f(x) = (x²−1)/(x−1)에서 x = 1.
분모가 0이라 f(1)이 정의 안 됨. 극한은 2로 존재하지만, 함수값이 없어서 불연속.

불연속 유형 2: ② 위반 — 극한 부존재

예: 절댓값 부호 함수 f(x) = |x|/x에서 x = 0.
좌극한 = −1, 우극한 = 1이라 극한 자체가 없음.

불연속 유형 3: ③ 위반 — 극한 ≠ 함수값

예: f(x) = { (x²−1)/(x−1) (x≠1), 5 (x=1) }
극한 = 2이고 f(1) = 5. 둘이 다르니까 불연속.

예제: 연속이 되도록 상수 결정
f(x) = { (x²−4)/(x−2)  (x ≠ 2) ,   a  (x = 2) } 가 x = 2에서 연속일 때, a의 값은?
풀이 보기
STEP 1

극한 계산 (x ≠ 2인 부분):

(x²−4)/(x−2) = (x+2)(x−2)/(x−2) = x+2

limx→2 (x+2) = 4

STEP 2

연속 조건 ③: limx→2 f(x) = f(2)

4 = a

결론

a = 4

5-3. 구간에서의 연속

함수가 구간의 모든 점에서 연속이면 "구간에서 연속"이라고 해.

  • 열린 구간 (a, b): 구간 내 모든 점에서 연속
  • 닫힌 구간 [a, b]: (a, b)에서 연속 + x = a에서 우연속 + x = b에서 좌연속

다항함수, 유리함수(분모 ≠ 0), 무리함수(근호 안 ≥ 0)는 정의역에서 연속이야.

5-4. 중간값 정리

중간값 정리 (Intermediate Value Theorem) f(x)가 닫힌 구간 [a, b]에서 연속이고
f(a) ≠ f(b)이면,
f(a)와 f(b) 사이의 임의의 값 k에 대해
f(c) = k인 c가 (a, b)에 적어도 하나 존재

쉽게 말하면: 연속함수의 그래프는 "점프"할 수 없으니, 중간에 있는 모든 값을 반드시 지나간다는 거야.

중간값 정리의 활용 — 근의 존재성

가장 많이 쓰는 곳: "방정식 f(x) = 0의 근이 구간 (a, b)에 있다"를 보일 때.

방법: f(a) < 0, f(b) > 0 (또는 반대)를 보이면 됨.
연속 + 부호가 바뀜 ⇒ 중간에 f(c) = 0인 c 존재!

예제: 근의 존재 증명
f(x) = x³ − 3x + 1이 구간 (0, 1)에서 근을 가짐을 보여라.
풀이 보기
STEP 1

f(x)는 다항함수 ⇒ 모든 실수에서 연속 ✔

STEP 2

f(0) = 0 − 0 + 1 = 1 > 0

f(1) = 1 − 3 + 1 = −1 < 0

결론

f(0) > 0, f(1) < 0이고 f는 [0, 1]에서 연속이므로,
중간값 정리에 의해 f(c) = 0인 c가 (0, 1)에 존재한다.

5-5. 최대·최소 정리

최대·최소 정리 (Extreme Value Theorem) f(x)가 닫힌 구간 [a, b]에서 연속이면,
f(x)는 [a, b]에서 반드시 최댓값과 최솟값을 가진다.
조건 확인!

닫힌 구간이어야 해. 열린 구간 (a, b)에서는 최댓값/최솟값이 없을 수 있어.
예: f(x) = x는 (0, 1)에서 최댓값이 없어 (1에 한없이 가까워지지만 1은 포함 안 됨).

연속이어야 해. 불연속 함수는 닫힌 구간에서도 최대/최소가 없을 수 있어.

6

극한 계산 테크닉 총정리

6-1. 다항함수의 극한

다항함수는 모든 점에서 연속이므로, 극한 = 직접 대입이야. 세상에서 가장 쉬운 케이스.

limx→a (다항식) = (x에 a를 대입한 값)
x → ∞에서 다항함수

limx→∞ (anxn + ...) = limx→∞ anxn
무한대에서는 최고차항만 살아남아. 나머지는 무시해도 돼.

6-2. 분수함수의 극한

분수함수는 3가지 경우로 나뉘어:

상황대입 결과처리법
분모 ≠ 0 확정값 직접 대입 (끝)
분모 = 0, 분자 ≠ 0 k/0 꼴 발산 (±∞) — 좌우극한 부호 확인
분모 = 0, 분자 = 0 0/0 부정형 인수분해/유리화/통분
예제: k/0 형태 (발산 판정)
limx→1 3/(x−1)²
풀이 보기
STEP 1

분자: 3 (≠ 0), 분모: (1−1)² = 0

3/0 꼴 ⇒ 발산

STEP 2

(x−1)²은 항상 ≥ 0이므로, 양쪽에서 +∞로 발산.

결론

+∞으로 발산 (극한 없음)

참고: 만약 분모가 (x−1)이었으면 좌극한 = −∞, 우극한 = +∞로 방향이 달라서 따로 말해야 함.

6-3. 무리함수의 극한

루트가 있는 함수. 0/0이 나오면 유리화가 기본 전략이야.

정의역 주의!

√(f(x))는 f(x) ≥ 0인 곳에서만 정의돼. 좌극한이나 우극한을 구할 때 정의역을 벗어나는 쪽은 고려 안 해도 돼.

예제: 한쪽 극한만 존재
limx→2 √(x−2) / (x−2)
풀이 보기
STEP 1

√(x−2)는 x ≥ 2에서만 정의 ⇒ 우극한만 의미 있음

STEP 2

√(x−2)/(x−2) = √(x−2)/[(√(x−2))²] = 1/√(x−2)

결론

x → 2+일 때 √(x−2) → 0+이므로

limx→2+ = +∞ (발산)

좌극한은 정의역 밖 ⇒ 극한 없음

6-4. 절댓값 함수의 극한

|f(x)|는 f(x)의 부호에 따라 식이 달라져. 그래서 좌극한과 우극한을 따로 구해야 하는 경우가 많아.

절댓값 분해 |f(x)| = { f(x)   (f(x) ≥ 0일 때) ,   −f(x)   (f(x) < 0일 때) }
예제 1: |x| 포함
limx→0 (x + |x|) / x
풀이 보기
STEP 1

좌극한 (x < 0 ⇒ |x| = −x):

(x + (−x))/x = 0/x = 0

limx→0− = 0

STEP 2

우극한 (x > 0 ⇒ |x| = x):

(x + x)/x = 2x/x = 2

limx→0+ = 2

결론

0 ≠ 2 ⇒ 극한 없음

예제 2: |x−a| 포함 인수분해
limx→3 (x² − 9) / |x − 3|
풀이 보기
STEP 1

x²−9 = (x−3)(x+3)

STEP 2

좌극한 (x < 3 ⇒ |x−3| = −(x−3)):

(x−3)(x+3)/[−(x−3)] = −(x+3) → −6

STEP 3

우극한 (x > 3 ⇒ |x−3| = x−3):

(x−3)(x+3)/(x−3) = x+3 → 6

결론

−6 ≠ 6 ⇒ 극한 없음

6-5. 가우스 함수 [x]의 극한

[x] (바닥함수, floor function)는 x를 넘지 않는 가장 큰 정수야. 정수 점에서 "점프"가 생기기 때문에, 정수에서의 극한은 좌우극한을 따로 봐야 해.

가우스 함수 성질 [2.7] = 2,   [3] = 3,   [−1.3] = −2

n이 정수일 때:
limx→n− [x] = n − 1   (왼쪽에서: 정수 하나 아래)
limx→n+ [x] = n   (오른쪽에서: 그 정수 자체)
예제: 가우스 + 극한
limx→2 (x − [x]) 가 존재하는지 조사하라.
풀이 보기
STEP 1

좌극한: x → 2−일 때, [x] = 1 (1 ≤ x < 2이므로)

limx→2− (x − 1) = 2 − 1 = 1

STEP 2

우극한: x → 2+일 때, [x] = 2 (2 ≤ x < 3이므로)

limx→2+ (x − 2) = 2 − 2 = 0

결론

1 ≠ 0 ⇒ 극한 없음

가우스 극한 팁
  • 비정수(예: x → 2.5)에서는 점프가 없어서 직접 대입 OK: [2.5] = 2
  • 정수에서만 좌우극한이 달라짐
  • [x]가 포함된 문제는 거의 100% 좌우극한 따로 구하는 문제야

6-6. 구분적 함수 (piecewise)의 극한

구간별로 정의가 다른 함수. 경계점에서의 극한이 항상 문제야.

접근법 경계점 x = a에서:
좌극한 = 왼쪽 구간의 식으로 계산
우극한 = 오른쪽 구간의 식으로 계산
둘이 같으면 극한 존재
예제: 구분적 함수 + 연속 조건
f(x) = { x² + a  (x < 1) ,   2x + b  (x ≥ 1) } 가 x = 1에서 연속이고
f(1) = 3일 때, a, b를 구하라.
풀이 보기
STEP 1

f(1) = 2(1) + b = 2 + b = 3 ⇒ b = 1

STEP 2

연속 ⇒ 좌극한 = f(1) = 3

좌극한: limx→1− (x²+a) = 1 + a = 3

⇒ a = 2

결론

a = 2, b = 1

7

수능/내신 빈출 유형 10선

실전에서 자주 나오는 유형 10가지를 모았어. 각각 풀 수 있으면 극한 단원은 완벽해.

기본 유형 1: 0/0 인수분해 (직접 출제)
limx→3 (x² − 5x + 6) / (x² − 9)
풀이 보기
STEP 1

대입: 분자 = 9 − 15 + 6 = 0, 분모 = 9 − 9 = 0 ⇒ 0/0

STEP 2

인수분해:

분자: x²−5x+6 = (x−2)(x−3)

분모: x²−9 = (x−3)(x+3)

STEP 3

약분: (x−2)/(x+3)

결론

(3−2)/(3+3) = 1/6

기본 유형 2: 루트 유리화 (0/0)
limx→1 (x − 1) / (√(2x+1) − √3)
풀이 보기
STEP 1

대입: 분자 = 0, 분모 = √3 − √3 = 0 ⇒ 0/0

STEP 2

분모 유리화: (√(2x+1) + √3) 곱하기

= (x−1)(√(2x+1)+√3) / [(2x+1) − 3]

= (x−1)(√(2x+1)+√3) / (2x − 2)

= (x−1)(√(2x+1)+√3) / [2(x−1)]

STEP 3

약분: = (√(2x+1)+√3)/2

결론

(√3 + √3)/2 = 2√3/2 = √3

중급 유형 3: 미정계수 결정 (0/0 + 극한값 주어짐)
limx→1 (ax² + bx − 3) / (x − 1) = 5 일 때, a + b 를 구하라.
풀이 보기
STEP 1

분모 → 0, 극한 유한 ⇒ 분자 → 0

a + b − 3 = 0  ⇒  a + b = 3   ... ①

STEP 2

분자 = (x−1) × (일차식) 형태. ax²+bx−3 = a(x−1)(x+c)라 하면

전개: a(x²+(c−1)x−c) = ax²+a(c−1)x−ac

비교: b = a(c−1), −3 = −ac ⇒ c = 3/a

STEP 3

약분 후 극한:

limx→1 a(x+c) = a(1+3/a) = a + 3 = 5

⇒ a = 2, ①에서 b = 1

결론

a + b = 3

(사실 ①에서 이미 답이 나왔지만, a, b 각각의 값도 구할 수 있다는 걸 보여준 거야)

중급 유형 4: ∞/∞ + 루트
limx→∞ (√(9x² + x) − 3x)
풀이 보기
STEP 1

∞ − ∞ 꼴. 유리화:

= [(9x²+x) − 9x²] / (√(9x²+x) + 3x) = x / (√(9x²+x) + 3x)

STEP 2

x로 나눔 (x > 0):

= 1 / (√(9+1/x) + 3)

결론

x → ∞: 1/(3+3) = 1/6

중급 유형 5: 절댓값 + 좌우극한
limx→2 |x² − 4| / (x − 2) 이 존재하는지 조사하라.
풀이 보기
STEP 1

x²−4 = (x−2)(x+2)

x > 2이면 x²−4 > 0 ⇒ |x²−4| = (x−2)(x+2)

x < 2이면 x²−4 < 0 ⇒ |x²−4| = −(x−2)(x+2)

STEP 2

우극한: (x−2)(x+2)/(x−2) = x+2 → 4

좌극한: −(x−2)(x+2)/(x−2) = −(x+2) → −4

결론

4 ≠ −4 ⇒ 극한 없음

중급 유형 6: 구분적 함수의 연속 조건
f(x) = { ax + 3  (x < 2) ,   x² + b  (x ≥ 2) } 가 모든 실수에서 연속일 때, a + b를 구하라.
풀이 보기
STEP 1

경계점 x = 2에서만 확인하면 됨 (나머지는 다항함수라 자동 연속).

STEP 2

좌극한: limx→2− (ax+3) = 2a + 3

우극한 = f(2): 4 + b

STEP 3

연속 조건: 좌극한 = 우극한 = f(2)

2a + 3 = 4 + b   ... ①

이 문제는 미지수 2개, 식 1개라 a + b를 직접 구해야 해.

①에서: 2a − b = 1

주의

조건이 하나뿐이면 a + b를 직접 구할 수 없어... 라고 생각할 수 있지만, 문제에서 "모든 실수에서 연속"이라 했으니 추가 조건을 찾아봐야 해.

이 문제에서는 ①만으로는 a + b 결정 불가능. 실제 시험 문제라면 추가 조건이 있을 거야.

여기선 2a − b = 1이라는 관계만 확인하는 게 핵심 포인트.

결론

연속 조건으로부터 2a − b = 1 (경계점 일치 조건)

추가 조건(예: f(0) = 5 등)이 주어지면 a, b 각각 결정 가능.

고급 유형 7: 극한의 대소 관계 (샌드위치 정리)
모든 실수 x에서 x² − 1 ≤ f(x) ≤ x² + x − 1 일 때,
limx→1 f(x) 를 구하라.
풀이 보기
STEP 1

limx→1 (x²−1) = 0

limx→1 (x²+x−1) = 1+1−1 = 1

잠깐!

양쪽 극한이 0과 1로 다르다. 샌드위치 정리는 양쪽이 같아야 적용 가능해.

이 상태로는 f(x)의 극한을 정확히 구할 수 없어... 라고 생각하면 함정이야.

STEP 2

사실 이 문제는 샌드위치 정리를 직접 쓰는 게 아니라, 부등식에서 극한의 존재를 읽는 문제야.

x = 1에서: 0 ≤ f(1) ≤ 1. f(1)은 0에서 1 사이 어딘가.

극한값도 마찬가지: 0 ≤ lim f(x) ≤ 1

하지만 정확한 값은 f의 구체적 형태를 알아야 해. 출제 의도가 샌드위치 정리 적용이라면, 원래 양쪽 함수의 극한이 같도록 설계돼 있어야 정상이야.

정석 예시로 바꾸면

만약 x²−1 ≤ f(x) ≤ x²+2x−3이고 x → 1이면:

양쪽 다 0 ⇒ lim f(x) = 0 (샌드위치!)

고급 유형 8: 극한값을 이용한 함수 결정
이차함수 f(x)가 다음 두 조건을 만족할 때, f(x)를 구하라.
(가) limx→1 f(x)/(x−1) = 4
(나) limx→−1 f(x)/(x+1) = −2
풀이 보기
STEP 1

(가)에서: 분모 → 0, 극한 유한 ⇒ f(1) = 0

(나)에서: 분모 → 0, 극한 유한 ⇒ f(−1) = 0

STEP 2

f(x)는 이차함수이고 f(1) = 0, f(−1) = 0이므로:

f(x) = a(x−1)(x+1) = a(x²−1)

STEP 3

(가)에서 a 결정:

limx→1 a(x−1)(x+1)/(x−1) = limx→1 a(x+1) = 2a = 4

⇒ a = 2

STEP 4

검산 (나):

limx→−1 2(x−1)(x+1)/(x+1) = limx→−1 2(x−1) = 2(−2) = −4

≠ −2? 조건 (나)와 안 맞네.

이건 f(x) = a(x−1)(x+1)이 유일한 형태가 아니기 때문이야. 이차함수 f(x) = ax²+bx+c로 놓아야 해.

재풀이

f(x) = ax² + bx + c

f(1) = 0: a + b + c = 0 ... ①

f(−1) = 0: a − b + c = 0 ... ②

① − ②: 2b = 0 ⇒ b = 0

①: a + c = 0 ⇒ c = −a

⇒ f(x) = a(x²−1) (맞았음)

(나): limx→−1 a(x−1)(x+1)/(x+1) = a(−1−1) = −2a = −2 ⇒ a = 1

결론

f(x) = x² − 1

검산 (가): lim 1·(x+1) = 2 ... 이러면 (가)가 4가 아니라 2?
이 문제는 (가)=4, (나)=−2를 동시에 만족하는 이차함수가 없어. 시험 문제라면 조건이 양립 가능하게 설계돼 있어. 여기서는 풀이 과정을 보는 거야: "분모 → 0, 극한 유한 ⇒ 분자 = 0"이 핵심.

고급 유형 9: 합성함수의 극한
limx→0 f(x) = 2 이고 limx→2 g(x) = 5 일 때,
limx→0 g(f(x)) 의 값을 구하라. (단 f, g는 적절한 조건을 만족)
풀이 보기
STEP 1

x → 0일 때 f(x) → 2.

그리고 f(x) → 2일 때 g(f(x)) → g(2 근방) → 5.

결론

limx→0 g(f(x)) = 5

주의

이건 f(x) ≠ 2 (x ≠ 0 근방에서)이거나, g가 x = 2에서 연속일 때 성립해. 시험에서는 보통 이 조건이 성립하도록 출제돼.

만약 f(x) = 2 (상수)이고 g(2) = 3, limx→2g(x) = 5이면,
g(f(x)) = g(2) = 3이지 5가 아니야. 극한과 함수값이 다를 수 있으니 주의.

킬러 유형 10: 극한 + 연속 + 미정계수 복합
f(x) = { (x² + ax + b)/(x − 1)  (x ≠ 1) ,   c  (x = 1) } 가 x = 1에서 연속이고
limx→2 f(x) = 7 일 때, a + b + c를 구하라.
풀이 보기
STEP 1

x = 1에서 연속 ⇒ limx→1 f(x) = f(1) = c

분모 x−1 → 0이므로, 극한이 유한하려면 분자도 0:

1 + a + b = 0  ⇒  a + b = −1   ... ①

STEP 2

분자 인수분해: x²+ax+b = (x−1)(x+d)

전개: x²+(d−1)x−d

비교: a = d−1, b = −d

STEP 3

x = 1에서의 극한:

limx→1 (x−1)(x+d)/(x−1) = 1 + d = c   ... ②

STEP 4

limx→2 f(x) = 7 조건 (x ≠ 1이니까 분수식 사용):

f(2) = (4+2a+b)/(2−1) = 4+2a+b = 7

2a + b = 3   ... ③

STEP 5

① + ③ 연립: a+b = −1, 2a+b = 3

빼기: a = 4, b = −5

d = −b = 5, ②에서 c = 1+5 = 6

결론

a + b + c = 4 + (−5) + 6 = 5

검산

f(x) = (x²+4x−5)/(x−1) = (x−1)(x+5)/(x−1) = x+5  (x≠1)

limx→1 = 6 = c ✔

limx→2 = 7 ✔

✓

최종 체크리스트

이것만 확실히 하면 극한은 끝이야

극한의 의미를 설명할 수 있다 (다가가는 과정, 도달 아님)
좌극한 = 우극한 ⇔ 극한 존재
극한 부존재 3가지: 좌 ≠ 우, 발산, 진동
극한의 사칙연산 성질과 적용 전제조건
0/0 부정형 → 인수분해, 유리화, 통분
∞/∞ 부정형 → 최고차항 나누기 (n<m: 0, n=m: 계수비, n>m: 발산)
∞−∞ 부정형 → 유리화 또는 통분
미정계수: 분모 → 0 + 극한 유한 ⇒ 분자 → 0
연속의 3조건: 함수값 존재, 극한 존재, 둘이 같음
중간값 정리로 근의 존재성 증명
절댓값/가우스/구분적 함수 → 좌우극한 따로!
루트가 있으면 유리화부터 떠올린다
