1. 집합의 역사 — 칸토어, 러셀, 수학의 기초 위기

19세기 말, 수학은 화려한 성취 뒤에 아무도 예상 못한 위기를 숨기고 있었다. 미적분학은 뉴턴과 라이프니츠 이후 200년간 물리학과 공학을 떠받치는 도구가 되었지만, 정작 "수(number)란 무엇인가?", "무한이란 무엇인가?" 같은 근본적 질문에는 누구도 깔끔하게 답하지 못했어. 100층짜리 건물을 올려놓고, 1층 기둥이 안개 위에 서 있었던 셈이야.

칸토어의 낙원

독일 수학자 게오르크 칸토어(Georg Cantor, 1845-1918)는 이 위기에 정면으로 맞섰다. 그의 무기는 놀라울 만큼 단순한 개념이었어:

칸토어의 정의 (1895)
"집합(Menge)이란, 우리의 직관이나 사고에 의해 확정적으로 구별되는 대상(Objekte)들을 하나의 전체(Ganzes)로 모은 것이다."

칸토어는 이 단순한 개념 위에 집합론(Set Theory)이라는 거대한 체계를 세웠다. 그가 보여준 결과는 충격적이었어:

  • 자연수와 짝수는 "같은 크기"의 무한이다 (1:1 대응이 가능하니까)
  • 하지만 자연수와 실수는 "다른 크기"의 무한이다 (대각선 논법)
  • 무한에도 등급이 있다 — $\aleph_0 < \aleph_1 < \aleph_2 < \cdots$

다비트 힐베르트는 이렇게 선언했다: "아무도 칸토어가 우리를 위해 창조한 이 낙원에서 우리를 쫓아내지 못할 것이다." 하지만 낙원에는 뱀이 숨어 있었다.

러셀의 역설 — 낙원의 뱀

1901년, 영국 철학자 버트런드 러셀(Bertrand Russell)이 칸토어의 집합론에서 치명적인 모순을 발견했다. 칸토어의 정의가 너무 자유로웠던 거야. "확정적으로 구별되는 대상들의 모임"이라면, 다음과 같은 집합도 허용되지 않겠어?
러셀의 역설 $R = \{x \mid x \notin x\}$
"자기 자신을 원소로 포함하지 않는 모든 집합의 집합"

이제 묻자: $R$은 자기 자신의 원소인가?

  • 만약 $R \in R$이면 → 정의에 의해 $R \notin R$ (모순!)
  • 만약 $R \notin R$이면 → 정의에 의해 $R \in R$ (모순!)

어느 쪽이든 모순. 이건 사소한 퍼즐이 아니야. 수학의 기초 전체가 흔들렸다. "아무 조건이든 집합을 만들 수 있다"는 칸토어의 자유로운 원칙이 자기파괴적이라는 뜻이니까.

일상적 비유: 어떤 마을에 이발사가 있다. 이 이발사는 "스스로 면도하지 않는 사람만 면도해준다." 그럼 이발사는 자기 자신을 면도하는가? 면도하면 → 스스로 면도하는 사람이므로 면도하면 안 됨. 면도하지 않으면 → 스스로 면도하지 않는 사람이므로 면도해야 됨. 이게 러셀의 역설의 일상 버전이야.

수학의 기초 위기와 해결

러셀의 역설 이후 수학자들은 세 진영으로 갈라졌다:

  • 논리주의(러셀, 화이트헤드): 집합 만드는 규칙을 엄격히 제한하자 → 『수학 원리(Principia Mathematica)』 3권, 1+1=2를 증명하는 데 362쪽
  • 직관주의(브라우어): 구성할 수 없는 건 존재하지 않는다고 치자
  • 형식주의(힐베르트): 공리를 정하고, 모순이 없다는 걸 증명하자

결국 ZFC 공리계(Zermelo-Fraenkel + Choice)가 표준으로 자리잡았다. 핵심은 "아무 조건이든 집합이 된다"를 폐기하고, 이미 존재하는 집합에서만 조건으로 부분집합을 만들 수 있다(분류 공리)로 제한한 거야. 이렇게 하면 러셀의 역설은 애초에 성립하지 않아.

고등학교에서 다루는 집합
우리가 쓰는 집합은 이 위기 이후 안전하게 정리된 것들이야. "10 이하의 소수의 집합" 같은 명확한 조건으로 만든 집합은 아무 문제가 없어. 고등학교 수준에서 러셀의 역설을 걱정할 필요는 없지만, 집합론이 왜 이렇게 엄밀한지 이해하면 수학을 보는 눈이 달라진다.

2. 집합의 표현 — 원소나열, 조건제시, 공집합의 철학

집합의 정의, 다시

집합(set)이란 어떤 대상이 그 모임에 속하는지 아닌지를 명확하게 판별할 수 있는 대상들의 모임이야. 이 "명확성" 기준이 전부야. 집합을 이루는 개별 대상을 원소(element)라고 하고, "$a$가 집합 $A$의 원소다"를 $a \in A$로 쓴다.

"키가 큰 학생의 모임" → 집합이 아님 ("큰"의 기준이 모호)
"키가 170cm 이상인 학생의 모임" → 집합 (기준이 명확)
"아름다운 숫자의 모임" → 집합 아님
"10의 양의 약수의 모임" → 집합 ({1, 2, 5, 10})

원소나열법과 조건제시법

원소나열법 — 원소를 직접 나열 $A = \{1, 2, 3, 4, 5\}$

원소나열법은 직관적이지만 한계가 있어. $B = \{2, 4, 6, 8, \ldots\}$처럼 무한집합은 끝까지 나열할 수 없으니까 "$\ldots$"에 의존해야 하는데, 이건 패턴을 읽는 사람의 해석에 맡기는 거라 엄밀하지 않아.

조건제시법 — 조건으로 정의 $B = \{x \mid x\text{는 양의 짝수}\} = \{x \in \mathbb{N} \mid 2 \mid x\}$

조건제시법이 더 강력하다. 무한집합도 정확하게 표현할 수 있고, 조건이 명확하기만 하면 돼. 형식: $\{x \mid P(x)\}$ — "$P(x)$를 만족하는 모든 $x$의 모임". 여기서 $P(x)$가 명확한 조건이어야 한다는 것.

집합의 핵심 성질

  • 원소의 유일성: 같은 원소는 한 번만 센다. $\{1, 2, 2, 3\} = \{1, 2, 3\}$
  • 순서 무관: 나열 순서는 상관없다. $\{a, b, c\} = \{c, a, b\}$
  • 소속의 명확성: 모든 대상에 대해 속하는지 아닌지가 결정됨

공집합 $\emptyset$ — 아무것도 없는 것의 존재

원소가 하나도 없는 집합을 공집합(empty set)이라 하고 $\emptyset$ 또는 $\{\}$로 쓴다. "아무것도 없는 모음"이 왜 중요할까?

"$x^2 = -1$을 만족하는 실수의 집합"을 생각해봐. 그런 실수는 없어. 하지만 이 조건 자체는 완벽히 명확하지 — 아무 실수를 넣어도 "만족 안 함"이 결정되니까. 그래서 이것도 엄연히 집합이야. 단지 원소가 없을 뿐.

공집합은 논리학에서 공진리(vacuous truth)와 깊이 연결돼 있다. "공집합의 모든 원소는 날 수 있다"는 참이야. 왜? 공집합에는 원소가 없으므로, "날 수 없는 원소"를 찾아서 반박할 수가 없거든. 반례가 존재하지 않으면 참 — 이게 공진리야.

공집합의 핵심 성질
  • $\emptyset$은 모든 집합의 부분집합: $\emptyset \subseteq A$ (항상 참, 공진리)
  • $\emptyset$은 유일하다: 원소 없는 집합은 단 하나
  • $\emptyset \neq \{\emptyset\}$: 전자는 원소 0개, 후자는 원소 1개(그 원소가 공집합)
  • $|\emptyset| = 0$ (원소 개수)
$\{\emptyset\}$과 $\emptyset$이 다르다는 건 처음에 혼란스럽지만, 이렇게 비유하면 돼: 빈 상자($\emptyset$)와 빈 상자를 넣은 상자($\{\emptyset\}$)는 다르다. 겉보기엔 둘 다 "아무것도 쓸모없어 보이지만", 수학적으로는 완전히 다른 대상이야.

수 집합 표기

$\mathbb{N}$: 자연수    $\mathbb{Z}$: 정수    $\mathbb{Q}$: 유리수    $\mathbb{R}$: 실수    $\mathbb{C}$: 복소수
$\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}$

3. 부분집합 — 왜 $2^n$개인가

부분집합의 정의

집합 $A$의 모든 원소가 집합 $B$에도 속하면, $A$는 $B$의 부분집합(subset)이라 하고 $A \subseteq B$로 쓴다.

$A \subseteq B \iff (\forall x)(x \in A \Rightarrow x \in B)$

이 정의를 잘 봐. "모든 $x$에 대해, $x \in A$이면 $x \in B$이다." 이건 조건문($\Rightarrow$)이야. 그래서 $A = \emptyset$이면, 전건($x \in A$)이 항상 거짓이므로 조건문 전체는 항상 참. 이게 공집합이 모든 집합의 부분집합인 이유야. 공진리가 또 등장하지.

부분집합 vs 진부분집합

  • $A \subseteq B$: $A$는 $B$의 부분집합 ($A = B$ 허용)
  • $A \subsetneq B$ (또는 $A \subset B$): $A$는 $B$의 진부분집합 ($A \neq B$이고 $A \subseteq B$)

부분집합의 개수: $2^n$ 증명

원소가 $n$개인 집합의 부분집합 개수는 정확히 $2^n$개야. 왜?

증명 1 — 선택 논증 (직관적)

$A = \{a_1, a_2, \ldots, a_n\}$이라 하자. 부분집합 하나를 만들려면, 각 원소에 대해 "넣는다/안 넣는다" 두 가지 선택을 해야 해.

$a_1$: 2가지 선택 $\times$ $a_2$: 2가지 $\times \cdots \times$ $a_n$: 2가지 $= 2^n$

따라서 부분집합의 총 개수는 $2^n$개. $\blacksquare$
증명 2 — 이항정리 이용

원소 $k$개짜리 부분집합의 개수는 $\binom{n}{k}$이므로, 전체 부분집합 수는:

$$\sum_{k=0}^{n} \binom{n}{k} = \binom{n}{0} + \binom{n}{1} + \cdots + \binom{n}{n}$$ 이항정리에서 $(1+x)^n = \sum_{k=0}^{n} \binom{n}{k} x^k$에 $x=1$을 대입하면: $$\sum_{k=0}^{n} \binom{n}{k} = 2^n \quad \blacksquare$$
증명 3 — 수학적 귀납법

$P(n)$: "원소 $n$개인 집합의 부분집합 수는 $2^n$"

기저: $n=0$ → $\emptyset$의 부분집합은 $\emptyset$ 하나 → $2^0=1$ ✓

귀납: $P(k)$ 참이라 가정. $|A|=k+1$인 집합 $A$에서 원소 $a$를 하나 고르자. $A$의 부분집합은 두 종류:
(i) $a$를 포함하지 않는 것 → $A \setminus \{a\}$의 부분집합 → 귀납가정에 의해 $2^k$개
(ii) $a$를 포함하는 것 → (i)의 각각에 $a$를 추가 → $2^k$개
합계: $2^k + 2^k = 2^{k+1}$ $\blacksquare$
예제: $\{1, 2, 3\}$의 부분집합 전체
$2^3 = 8$개:
$\emptyset, \{1\}, \{2\}, \{3\}, \{1,2\}, \{1,3\}, \{2,3\}, \{1,2,3\}$

진부분집합: $2^3 - 1 = 7$개 (자기 자신 제외)
공집합 아닌 부분집합: $2^3 - 1 = 7$개 (공집합 제외)
공집합 아닌 진부분집합: $2^3 - 2 = 6$개

멱집합

집합 $A$의 모든 부분집합을 원소로 하는 집합을 멱집합(power set)이라 하고 $\mathcal{P}(A)$로 쓴다. $|\mathcal{P}(A)| = 2^{|A|}$. 칸토어는 놀라운 사실을 증명했는데, 어떤 집합이든 그 멱집합과 1:1 대응이 불가능하다 ($|A| < |\mathcal{P}(A)|$). 이것이 무한의 등급을 만드는 핵심 원리야.

포함관계의 논리적 의미

집합의 상등(같음) 증명
$A = B$를 증명하려면: $A \subseteq B$ 그리고 $B \subseteq A$를 각각 보이면 돼.

이것을 쌍포함(double inclusion) 증명법이라 한다. 집합 관련 증명에서 가장 기본적이고 중요한 기법이야.

4. 집합 연산 심화 — 포함배제, 드모르간, 대칭차

기본 연산

합집합 (Union) $A \cup B = \{x \mid x \in A \text{ 또는 } x \in B\}$
교집합 (Intersection) $A \cap B = \{x \mid x \in A \text{ 그리고 } x \in B\}$
여집합 (Complement) — 전체집합 $U$ 기준 $A^c = \{x \in U \mid x \notin A\}$
차집합 (Difference) $A - B = A \setminus B = \{x \mid x \in A \text{ 그리고 } x \notin B\} = A \cap B^c$
대칭차 (Symmetric Difference) $A \triangle B = (A \setminus B) \cup (B \setminus A) = (A \cup B) \setminus (A \cap B)$

대칭차 $A \triangle B$는 "$A$ 또는 $B$에 속하지만 둘 다에는 속하지 않는 원소"의 집합이야. XOR 연산과 같다고 생각하면 돼.

벤 다이어그램

집합 연산의 성질

법칙합집합교집합
교환$A \cup B = B \cup A$$A \cap B = B \cap A$
결합$(A \cup B) \cup C = A \cup (B \cup C)$$(A \cap B) \cap C = A \cap (B \cap C)$
분배$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
항등$A \cup \emptyset = A$$A \cap U = A$
보원$A \cup A^c = U$$A \cap A^c = \emptyset$
멱등$A \cup A = A$$A \cap A = A$
흡수$A \cup (A \cap B) = A$$A \cap (A \cup B) = A$

드모르간 법칙 — 증명

드모르간 법칙 (De Morgan's Laws) $(A \cup B)^c = A^c \cap B^c$
$(A \cap B)^c = A^c \cup B^c$
증명: $(A \cup B)^c = A^c \cap B^c$

쌍포함법으로 증명한다.

(i) $(A \cup B)^c \subseteq A^c \cap B^c$:
$x \in (A \cup B)^c$라 하자.
→ $x \notin A \cup B$ (여집합 정의)
→ $x \notin A$ 그리고 $x \notin B$ ("또는"의 부정은 "그리고"의 부정)
→ $x \in A^c$ 그리고 $x \in B^c$
→ $x \in A^c \cap B^c$ ✓

(ii) $A^c \cap B^c \subseteq (A \cup B)^c$:
$x \in A^c \cap B^c$라 하자.
→ $x \in A^c$ 그리고 $x \in B^c$
→ $x \notin A$ 그리고 $x \notin B$
→ $x \notin A \cup B$
→ $x \in (A \cup B)^c$ ✓

(i), (ii)에 의해 $(A \cup B)^c = A^c \cap B^c$. $\blacksquare$

포함배제 원리 (Inclusion-Exclusion)

합집합의 원소 개수를 세는 핵심 공식이야. 단순히 더하면 교집합 부분이 중복 계산되니까, 빼줘야 해.

2집합 $|A \cup B| = |A| + |B| - |A \cap B|$
3집합 $|A \cup B \cup C| = |A| + |B| + |C| - |A \cap B| - |B \cap C| - |A \cap C| + |A \cap B \cap C|$

3집합 공식의 직관: 각각 더하면(+3) 겹치는 부분을 초과 계산하니까 두 개씩 겹치는 걸 빼(-3). 그런데 세 개가 겹치는 중심부는 3번 더하고 3번 빼서 0이 되어버리니까 다시 1번 더해(+1).

예제: 포함배제 응용
학생 100명 중 수학 좋아하는 사람 60명, 영어 좋아하는 사람 45명, 둘 다 좋아하는 사람 25명. 둘 다 싫어하는 사람은?

$|A \cup B| = 60 + 45 - 25 = 80$
둘 다 싫어하는 사람: $100 - 80 = 20$명
예제: 3집합 포함배제
100명 중 축구 50, 농구 40, 야구 30, 축구∩농구 15, 농구∩야구 10, 축구∩야구 12, 모두 5명.
적어도 하나 좋아하는 사람: $50+40+30-15-10-12+5 = 88$명
아무것도 안 좋아하는 사람: $100-88=12$명

5. 명제와 논리 — 진리표, 논리 연결사의 세계

명제란 무엇인가

명제(proposition)란 참(True) 또는 거짓(False) 중 정확히 하나의 진리값을 가지는 문장이야. "이 문장은 거짓이다"처럼 참도 거짓도 아닌 문장은 명제가 아니야 (이것도 역설이지).

명제인 것과 아닌 것
✓ "2는 소수다" (참인 명제)
✓ "$3 + 5 = 9$" (거짓인 명제 — 거짓이어도 명제야)
✗ "$x + 2 = 5$" (명제가 아님 — $x$ 값에 따라 참/거짓이 달라짐 → 조건문)
✗ "문을 닫아라" (명령문, 참/거짓 판별 불가)
✗ "수학은 재미있다" (주관적, 사람마다 다름)

조건문(열린 명제)과 진리집합

$p(x)$: "$x$는 3의 배수이다" — 이건 $x$가 정해져야 참/거짓이 결정돼. 이런 걸 조건문(또는 열린 명제, 명제함수)이라 해. 전체집합 $U$에서 $p(x)$를 참으로 만드는 $x$의 집합을 진리집합이라 하고 $P$로 쓴다:

$P = \{x \in U \mid p(x)\text{가 참}\}$
예제
$U = \{1, 2, 3, \ldots, 12\}$
$p(x)$: "$x$는 3의 배수" → $P = \{3, 6, 9, 12\}$
$q(x)$: "$x$는 짝수" → $Q = \{2, 4, 6, 8, 10, 12\}$

6. 역, 이, 대우 — 진리표로 완전 증명

조건명제 $p \rightarrow q$로부터 세 가지 관련 명제를 만들 수 있어:

원래 명제와 3가지 변환 원래: $p \rightarrow q$
역(converse): $q \rightarrow p$
이(inverse): $\lnot p \rightarrow \lnot q$
대우(contrapositive): $\lnot q \rightarrow \lnot p$

진리표로 관계 완전 확인

$p$$q$$p \rightarrow q$
(원래)		$q \rightarrow p$
(역)		$\lnot p \rightarrow \lnot q$
(이)		$\lnot q \rightarrow \lnot p$
(대우)
TTTTTT
TFFTTF
FTTFFT
FFTTTT
진리표에서 읽어내는 핵심 사실
  • 원래와 대우: 진리값이 완전히 동일 → 논리적 동치 ($p \rightarrow q \equiv \lnot q \rightarrow \lnot p$)
  • 역과 이: 진리값이 서로 동일 → 역과 이도 동치 ($q \rightarrow p \equiv \lnot p \rightarrow \lnot q$)
  • 원래와 역: 진리값이 다름 → 원래가 참이어도 역은 거짓일 수 있음
  • 원래와 이: 진리값이 다름 → 원래가 참이어도 이는 거짓일 수 있음
원래 명제와 대우가 동치인 이유 (논리적 증명)

$p \rightarrow q$의 정의: $\lnot p \lor q$
$\lnot q \rightarrow \lnot p$의 정의: $\lnot(\lnot q) \lor \lnot p = q \lor \lnot p = \lnot p \lor q$

둘 다 $\lnot p \lor q$이므로 동치. $\blacksquare$

대우증명법 (Proof by Contrapositive)

"$p \rightarrow q$를 증명하라"는 문제를 "$\lnot q \rightarrow \lnot p$를 증명하라"로 바꿔 풀 수 있어. 동치니까.

예제: 대우증명법
명제: "$n^2$이 짝수이면 $n$은 짝수이다."

대우: "$n$이 홀수이면 $n^2$은 홀수이다."

증명: $n$이 홀수 → $n = 2k+1$ (정수 $k$)
$n^2 = (2k+1)^2 = 4k^2 + 4k + 1 = 2(2k^2+2k) + 1$
이건 홀수. 따라서 대우가 참이므로 원래 명제도 참. $\blacksquare$
예제: 직접 증명이 어려울 때
명제: "$3n + 2$가 홀수이면 $n$은 홀수이다."

대우: "$n$이 짝수이면 $3n+2$는 짝수이다."

증명: $n = 2k$ → $3(2k) + 2 = 6k + 2 = 2(3k+1)$ → 짝수. $\blacksquare$

귀류법과의 관계

귀류법(proof by contradiction): "$p$이고 $\lnot q$"를 가정한 뒤 모순을 이끌어내서 $p \rightarrow q$를 증명하는 방법. 대우증명법과 비슷하지만 더 일반적이야. 대우증명법은 "$\lnot q$에서 $\lnot p$를 유도"하고, 귀류법은 "$p$와 $\lnot q$에서 아무 모순이든 유도"해.

예제: 귀류법 — $\sqrt{2}$가 무리수임을 증명
$\sqrt{2}$가 유리수라 가정하면 $\sqrt{2} = \frac{a}{b}$ (기약분수).
양변 제곱: $2 = \frac{a^2}{b^2}$ → $a^2 = 2b^2$ → $a^2$은 짝수 → $a$는 짝수
$a = 2k$ 대입: $4k^2 = 2b^2$ → $b^2 = 2k^2$ → $b$도 짝수
$a, b$ 둘 다 짝수 → 기약분수 가정에 모순. $\blacksquare$

7. 충분조건과 필요조건 심화

정의

명제 $p \rightarrow q$가 참일 때:

충분조건과 필요조건 $p$는 $q$이기 위한 충분조건 (sufficient condition)
$q$는 $p$이기 위한 필요조건 (necessary condition)

진리집합으로 이해하기

$p \rightarrow q$가 참 $\iff$ $P \subseteq Q$ (진리집합의 포함관계). 이걸 그림으로 보면:

핵심: 진리집합 크기와 조건의 관계
  • $P \subseteq Q$ → $p$는 충분, $q$는 필요 → 작은 집합이 충분조건
  • $P \supseteq Q$ → $p$는 필요, $q$는 충분
  • $P = Q$ → $p$와 $q$는 서로 필요충분조건 ($p \leftrightarrow q$)
  • $P \not\subseteq Q$이고 $Q \not\subseteq P$ → 충분도 필요도 아님

일상 예시로 정리

예시 1: 서울 시민 → 대한민국 국민
$p$: 서울 시민이다, $q$: 대한민국 국민이다
$P \subset Q$ (서울 시민은 국민의 부분집합)
→ "서울 시민이면 국민이다" (참) → $p$는 $q$의 충분조건
→ "국민이면 서울 시민이다" (거짓 — 부산 시민도 국민) → $q$는 $p$의 필요조건
예시 2: $x = 2$ → $x^2 = 4$
$p$: $x=2$, $q$: $x^2=4$
$P = \{2\}$, $Q = \{-2, 2\}$ → $P \subset Q$
→ $p$는 $q$의 충분조건 (2이면 확실히 제곱이 4)
→ $q$는 $p$의 필요조건 (제곱이 4여야 2일 수 있음)
→ $p$는 $q$의 필요조건은 아님 ($x=-2$면 $q$는 참이지만 $p$는 거짓)
예시 3: 필요충분 (iff)
$p$: "$n$은 2의 배수", $q$: "$n$은 짝수"
$P = Q$ (정확히 같은 집합)
→ $p \leftrightarrow q$ → $p$는 $q$의 필요충분조건

시험에서 자주 틀리는 패턴

"$p$는 $q$의 충분조건"이라는 말이 나오면 $p \rightarrow q$가 참이라는 뜻이야.
"$p$는 $q$의 필요조건"이라는 말이 나오면 $q \rightarrow p$가 참이라는 뜻이야.

헷갈리면 진리집합 크기를 떠올려: 충분조건의 진리집합이 더 작다(또는 같다).

네 가지 경우 완전 정리

$P$와 $Q$의 관계$p$는 $q$의$q$는 $p$의
$P \subsetneq Q$충분 (필요 아님)필요 (충분 아님)
$P \supsetneq Q$필요 (충분 아님)충분 (필요 아님)
$P = Q$필요충분필요충분
$P \not\subseteq Q$, $Q \not\subseteq P$충분도 필요도 아님충분도 필요도 아님

8. 절대부등식 — 산술기하, 코시-슈바르츠

절대부등식이란

절대부등식(absolute inequality)이란 변수가 어떤 범위의 모든 값을 가져도 항상 성립하는 부등식이야. "$x^2 \geq 0$"이 가장 단순한 예. 이건 $x$가 뭐든 참이지.

기본 절대부등식
모든 실수 $a$에 대해: $a^2 \geq 0$ (등호: $a = 0$)
모든 실수 $a, b$에 대해: $(a-b)^2 \geq 0 \Rightarrow a^2 + b^2 \geq 2ab$ (등호: $a=b$)

산술-기하 평균 부등식 (AM-GM)

AM-GM 부등식 양수 $a, b$에 대해: $\dfrac{a+b}{2} \geq \sqrt{ab}$
(등호 조건: $a = b$)

말로 하면: 산술평균은 기하평균 이상이다.

증명 1 — 제곱의 비음수성

$(\sqrt{a} - \sqrt{b})^2 \geq 0$ (실수의 제곱은 항상 $\geq 0$)
$a - 2\sqrt{ab} + b \geq 0$
$a + b \geq 2\sqrt{ab}$
$\dfrac{a+b}{2} \geq \sqrt{ab}$ $\blacksquare$

등호: $\sqrt{a} = \sqrt{b}$ 즉 $a = b$
증명 2 — 기하학적 (원과 현)

반지름 $r = \frac{a+b}{2}$인 반원을 생각하자. 지름 위의 한 점에서 수선을 세우면 그 높이가 $\sqrt{ab}$가 된다 (원의 성질). 반원 안의 수선은 반지름을 넘을 수 없으므로 $\sqrt{ab} \leq \frac{a+b}{2}$. 등호는 $a=b$일 때 (수선이 정확히 중심에서).
증명 3 — 변수 치환

$a = m+d, \; b = m-d$ ($m = \frac{a+b}{2}$, $d = \frac{a-b}{2}$)로 놓으면:
산술평균 $= m$
기하평균 $= \sqrt{(m+d)(m-d)} = \sqrt{m^2 - d^2} \leq m$
$d^2 \geq 0$이므로 $m^2 - d^2 \leq m^2$. $\blacksquare$

$n$변수 일반화

일반 AM-GM 양수 $a_1, a_2, \ldots, a_n$에 대해:
$\dfrac{a_1 + a_2 + \cdots + a_n}{n} \geq \sqrt[n]{a_1 a_2 \cdots a_n}$

AM-GM 응용

예제 1: 최솟값 구하기
양수 $x$에 대해 $f(x) = x + \frac{4}{x}$의 최솟값은?

AM-GM: $x + \frac{4}{x} \geq 2\sqrt{x \cdot \frac{4}{x}} = 2\sqrt{4} = 4$
등호: $x = \frac{4}{x}$ → $x^2 = 4$ → $x = 2$ (양수)
따라서 최솟값은 4 (when $x=2$)
예제 2: 3변수
양수 $a, b, c$이고 $abc = 8$일 때, $a + b + c$의 최솟값은?

AM-GM: $\frac{a+b+c}{3} \geq \sqrt[3]{abc} = \sqrt[3]{8} = 2$
$a + b + c \geq 6$ (등호: $a = b = c = 2$)

코시-슈바르츠 부등식 (Cauchy-Schwarz)

코시-슈바르츠 부등식 $(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \geq (a_1 b_1 + a_2 b_2 + \cdots + a_n b_n)^2$

$n=2$인 경우:

$(a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2$
증명 1 — 직접 전개 ($n=2$)

$(a^2+b^2)(c^2+d^2) - (ac+bd)^2$
$= a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2 - a^2c^2 - 2abcd - b^2d^2$
$= a^2d^2 - 2abcd + b^2c^2$
$= (ad - bc)^2 \geq 0$ $\blacksquare$

등호: $ad = bc$ 즉 $\frac{a}{c} = \frac{b}{d}$
증명 2 — 이차함수 판별식 (일반 $n$)

$f(t) = \sum_{i=1}^{n}(a_i t - b_i)^2 \geq 0$ (모든 실수 $t$에 대해)
전개하면: $\left(\sum a_i^2\right)t^2 - 2\left(\sum a_i b_i\right)t + \sum b_i^2 \geq 0$

이 이차함수가 항상 $\geq 0$이려면 판별식 $D \leq 0$:
$4\left(\sum a_i b_i\right)^2 - 4\left(\sum a_i^2\right)\left(\sum b_i^2\right) \leq 0$
$\left(\sum a_i b_i\right)^2 \leq \left(\sum a_i^2\right)\left(\sum b_i^2\right)$ $\blacksquare$
증명 3 — 벡터 내적 해석

벡터 $\mathbf{a} = (a_1, \ldots, a_n)$, $\mathbf{b} = (b_1, \ldots, b_n)$에 대해
$\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}||\mathbf{b}|\cos\theta$
$|\cos\theta| \leq 1$이므로 $|\mathbf{a} \cdot \mathbf{b}| \leq |\mathbf{a}||\mathbf{b}|$
양변 제곱하면 코시-슈바르츠. $\blacksquare$
예제: 코시-슈바르츠 응용
양수 $a, b$이고 $a + b = 1$일 때, $\frac{1}{a} + \frac{1}{b}$의 최솟값은?

코시-슈바르츠: $(a+b)\left(\frac{1}{a}+\frac{1}{b}\right) \geq \left(\sqrt{a \cdot \frac{1}{a}} + \sqrt{b \cdot \frac{1}{b}}\right)^2 = (1+1)^2 = 4$
$1 \cdot \left(\frac{1}{a}+\frac{1}{b}\right) \geq 4$
최솟값 4 (등호: $a = b = \frac{1}{2}$)

부등식 증명의 3대 전략

부등식 증명 전략 정리
  1. 차를 만들어 $\geq 0$ 보이기: $A - B = (\text{완전제곱식}) \geq 0$
  2. AM-GM 직접 적용: 곱이 상수이면 합의 최솟값, 합이 상수이면 곱의 최댓값
  3. 코시-슈바르츠: 분수의 합의 최솟값에 강력

확인 퀴즈 (15문항)

Q1. 러셀의 역설에서, 집합 $R = \{x \mid x \notin x\}$에 대해 $R \in R$이라 가정하면 어떤 결론이 나오는가?
탭하여 정답 확인
$R \in R$이면, $R$의 정의에 의해 $R \notin R$이어야 한다. 모순. 반대로 $R \notin R$이면 정의에 의해 $R \in R$. 어느 쪽이든 모순이므로, 이런 집합은 허용되어서는 안 된다.
Q2. $\{\emptyset\}$의 원소 개수와 $\emptyset$의 원소 개수를 각각 구하라.
탭하여 정답 확인
$|\emptyset| = 0$ (원소 없음), $|\{\emptyset\}| = 1$ (원소가 $\emptyset$ 하나). 빈 상자와 빈 상자를 넣은 상자의 차이.
Q3. 원소가 5개인 집합의 부분집합 개수, 진부분집합 개수, 공집합이 아닌 진부분집합 개수를 각각 구하라.
탭하여 정답 확인
부분집합: $2^5 = 32$개
진부분집합: $32 - 1 = 31$개 (자기 자신 제외)
공집합 아닌 진부분집합: $32 - 2 = 30$개 (자기 자신과 공집합 제외)
Q4. $U = \{1,2,...,10\}$, $A = \{1,3,5,7,9\}$, $B = \{2,3,5,7\}$일 때, $A \triangle B$를 구하라.
탭하여 정답 확인
$A \triangle B = (A \setminus B) \cup (B \setminus A)$
$A \setminus B = \{1, 9\}$, $B \setminus A = \{2\}$
$A \triangle B = \{1, 2, 9\}$
Q5. 포함배제: 학생 80명, 수학 동아리 35명, 과학 동아리 28명, 둘 다 12명. 어디에도 안 든 학생은?
탭하여 정답 확인
$|A \cup B| = 35 + 28 - 12 = 51$
어디에도 안 든 학생: $80 - 51 = 29$명
Q6. 드모르간 법칙을 이용: $U = \{1,...,10\}$, $A = \{1,2,3\}$, $B = \{3,4,5\}$일 때 $(A \cap B)^c$를 구하라.
탭하여 정답 확인
$(A \cap B)^c = A^c \cup B^c$
$A \cap B = \{3\}$ → $(A \cap B)^c = \{1,2,4,5,6,7,8,9,10\}$
검증: $A^c = \{4,5,6,7,8,9,10\}$, $B^c = \{1,2,6,7,8,9,10\}$
$A^c \cup B^c = \{1,2,4,5,6,7,8,9,10\}$ ✓
Q7. $p \rightarrow q$에서 $p$가 거짓이면 $p \rightarrow q$의 진리값은? 이유도 설명하라.
탭하여 정답 확인
항상 참(T). 조건명제에서 전건이 거짓이면, 결론이 뭐든 전체는 참이다 (공진리, vacuous truth). "비가 오면 우산 줄게"에서 비가 안 오면, 우산을 줬든 안 줬든 약속을 어긴 게 아니니까.
Q8. "모든 소수는 홀수이다"의 역, 이, 대우를 쓰고, 각각의 참/거짓을 판별하라.
탭하여 정답 확인
원래: "소수이면 홀수이다" → 거짓 (반례: 2는 소수이지만 짝수)
역: "홀수이면 소수이다" → 거짓 (반례: 9)
이: "소수가 아니면 홀수가 아니다(짝수)" → 거짓 (반례: 9는 소수 아닌 홀수)
대우: "홀수가 아니면(짝수이면) 소수가 아니다" → 거짓 (반례: 2는 짝수이면서 소수)

원래와 대우의 진리값이 같고(둘 다 거짓), 역과 이의 진리값이 같다(둘 다 거짓).
Q9. $p$: "$x > 3$", $q$: "$x > 1$". $p$는 $q$의 무슨 조건인가?
탭하여 정답 확인
$P = (3, \infty)$, $Q = (1, \infty)$ → $P \subset Q$
$p \rightarrow q$ 참 (3보다 크면 당연히 1보다 큼) → $p$는 $q$의 충분조건
$q \rightarrow p$ 거짓 (반례: $x=2$) → $p$는 $q$의 필요조건은 아님
답: $p$는 $q$이기 위한 충분조건이지만 필요조건은 아니다
Q10. $p$: "$x^2 = 4$", $q$: "$x = 2$". $p$는 $q$의 무슨 조건인가?
탭하여 정답 확인
$P = \{-2, 2\}$, $Q = \{2\}$ → $Q \subset P$
$p \rightarrow q$ 거짓 (반례: $x=-2$) → $p$는 $q$의 충분조건이 아님
$q \rightarrow p$ 참 ($x=2$이면 $x^2=4$) → $p$는 $q$의 필요조건
답: $p$는 $q$이기 위한 필요조건이지만 충분조건은 아니다
Q11. AM-GM을 이용: 양수 $x$에 대해 $x + \frac{9}{x}$의 최솟값을 구하라.
탭하여 정답 확인
$x + \frac{9}{x} \geq 2\sqrt{x \cdot \frac{9}{x}} = 2\sqrt{9} = 6$
등호: $x = \frac{9}{x}$ → $x = 3$
최솟값: 6
Q12. 코시-슈바르츠: $(1^2+2^2+3^2)(x^2+y^2+z^2) \geq (x+2y+3z)^2$에서 $x+2y+3z=14$이고 등호가 성립할 때 $x, y, z$를 구하라.
탭하여 정답 확인
등호 조건: $\frac{x}{1} = \frac{y}{2} = \frac{z}{3} = k$
$x = k, y = 2k, z = 3k$ → $k + 4k + 9k = 14$ → $14k = 14$ → $k = 1$
$x = 1, y = 2, z = 3$
Q13. 3집합 포함배제: 100명, A=45, B=35, C=30, A∩B=10, B∩C=8, A∩C=12, A∩B∩C=3. 적어도 하나에 속하는 사람은?
탭하여 정답 확인
$|A \cup B \cup C| = 45+35+30-10-8-12+3 = 83$명
Q14. 대우증명법: "$n^2$이 3의 배수이면 $n$은 3의 배수이다"를 대우로 증명하라.
탭하여 정답 확인
대우: "$n$이 3의 배수가 아니면 $n^2$은 3의 배수가 아니다."

$n$이 3의 배수가 아니면 $n = 3k+1$ 또는 $n = 3k+2$.
경우 1: $n = 3k+1$ → $n^2 = 9k^2+6k+1 = 3(3k^2+2k)+1$ → 3의 배수 아님 ✓
경우 2: $n = 3k+2$ → $n^2 = 9k^2+12k+4 = 3(3k^2+4k+1)+1$ → 3의 배수 아님 ✓
대우가 참이므로 원래 명제도 참. $\blacksquare$
Q15. "$a^2+b^2 \geq 2ab$"를 세 가지 방법으로 증명하라. (차 이용, AM-GM, 코시-슈바르츠)
탭하여 정답 확인
방법 1 (차): $a^2+b^2-2ab = (a-b)^2 \geq 0$ ✓

방법 2 (AM-GM): $\frac{a^2+b^2}{2} \geq \sqrt{a^2 \cdot b^2} = |ab| \geq ab$ → $a^2+b^2 \geq 2ab$ ✓

방법 3 (코시-슈바르츠): $(1^2+1^2)(a^2+b^2) \geq (a+b)^2$ → $2(a^2+b^2) \geq a^2+2ab+b^2$ → $a^2+b^2 \geq 2ab$ ✓
