1장. 극한이 뭔데?

🔥 겨울에 히터를 켰다고 해봐. 방 온도가 10도에서 시작해. 히터를 켜면 온도가 올라가기 시작해. 15도, 18도, 19.5도, 19.8도, 19.9도, 19.95도... 점점 20도에 가까워지지? 근데 정확히 20도가 되는 순간은 언제야? 사실 수학적으로 보면, 20도에 한없이 가까워지는 것이지 딱 20도가 되는 게 아닐 수도 있어. 이게 극한이야.
lim f(x) = L x→a "x가 a에 한없이 가까워질 때, f(x)는 L에 한없이 가까워진다"

극한에서 핵심은 이거야: "도달"이 아니라 "접근". x가 a에 실제로 도달하는 게 아니라, a 주변에서 한없이 가까워질 때 함수값이 어디로 향하는지를 보는 거야.

왜 "도달"이 아니라 "접근"일까?

f(x) = (x²-1)/(x-1)을 생각해봐. x = 1을 넣으면 0/0으로 정의가 안 돼. 근데 x가 1에 가까워지면?

x = 0.9 → f(x) = 1.9 x = 0.99 → f(x) = 1.99 x = 0.999 → f(x) = 1.999 x = 1.001 → f(x) = 2.001 x = 1.01 → f(x) = 2.01 → 2에 가까워지고 있어!

x = 1에서는 정의가 안 되지만, x가 1에 가까워질 때 함수값은 분명히 2를 향하고 있어. 이걸 lim(x→1) f(x) = 2라고 써.

2장. 간단한 극한 — 그냥 대입해

극한의 좋은 소식: 대부분의 경우, 그냥 x에 값을 대입하면 돼.

lim (3x + 2) = 3(4) + 2 = 14 x→4 lim (x² - 5x + 6) = 9 - 15 + 6 = 0 x→3 lim √(x + 5) = √(4 + 5) = 3 x→4

대입이 되는 이유

다항함수, 유리함수(분모 ≠ 0), 무리함수 등 "연속인 함수"에서는 극한값 = 함수값이야. 그냥 대입하면 끝. 문제가 되는 건 대입했을 때 0/0이나 ∞/∞ 같은 이상한 게 나올 때야.

극한의 성질 (계산 도구)

lim f(x) = L, lim g(x) = M 일 때: 1. lim [f(x) + g(x)] = L + M 2. lim [f(x) × g(x)] = L × M 3. lim [f(x) / g(x)] = L / M (M ≠ 0) 4. lim [c × f(x)] = c × L 5. lim [f(x)]ⁿ = Lⁿ

이 성질들 덕분에 복잡한 식도 조각조각 나눠서 극한을 구할 수 있어.

3장. 0/0 부정형 — 인수분해로 풀어내기

🧩 대입했더니 0/0이 나왔어. 이건 "답이 없다"가 아니라 "아직 답을 모른다"는 뜻이야. 0/0은 부정형(indeterminate form)이라고 해. 0이 될 수도 있고, 5가 될 수도 있고, ∞가 될 수도 있어. 추가 작업이 필요하다는 신호지.
0/0은 "0"이 아니야!
0/0 부정형은 분자와 분모가 둘 다 0으로 가고 있다는 뜻이야. 실제 극한값은 분자가 0으로 가는 속도와 분모가 0으로 가는 속도의 경쟁에 따라 달라져.

해결법 1: 인수분해

lim x² - 4 x→2 ----- x - 2 대입하면: (4-4)/(2-2) = 0/0 → 부정형! 인수분해: = lim (x+2)(x-2) / (x-2) x→2 = lim (x+2) ← (x-2) 약분! x→2 = 2 + 2 = 4

0/0이 나오면, 분자와 분모에 공통인수가 있다는 뜻이야. 그걸 찾아서 약분하면 깔끔하게 풀려.

좀 더 복잡한 예시

lim x² - 5x + 6 x→2 ----------- x² - 4 대입: (4-10+6)/(4-4) = 0/0 분자: x² - 5x + 6 = (x-2)(x-3) 분모: x² - 4 = (x-2)(x+2) = lim (x-2)(x-3) / (x-2)(x+2) x→2 = lim (x-3)/(x+2) x→2 = (2-3)/(2+2) = -1/4

해결법 2: 유리화

무리식(루트)이 나오면 인수분해가 안 될 수 있어. 그때는 유리화를 써.

lim √(x+3) - 2 x→1 ---------- x - 1 대입: (√4 - 2)/(1-1) = 0/0 분자·분모에 (√(x+3) + 2)를 곱해: = lim (x+3-4) / [(x-1)(√(x+3)+2)] x→1 = lim (x-1) / [(x-1)(√(x+3)+2)] x→1 = lim 1 / (√(x+3)+2) x→1 = 1 / (2+2) = 1/4

유리화의 원리

(a-b)(a+b) = a²-b²를 이용하는 거야. 루트를 제곱하면 루트가 사라지니까, 곱해서 제곱 형태로 만드는 거지. 분자에 루트가 있으면 분자를 유리화, 분모에 있으면 분모를 유리화해.

4장. 무한대 — 끝없이 커지면?

🚀 x가 무한히 커지면 함수값은 어떻게 될까? 1/x는 0에 가까워지고, x²는 끝없이 커져. 무한으로 갈 때의 행동을 보는 것도 극한이야.
lim 1/x = 0 (x가 커지면 1/x는 작아져) x→∞ lim x² = ∞ (x가 커지면 x²은 폭발) x→∞ lim (3x+1)/x = 3 (나눠보면 3+1/x → 3) x→∞

∞/∞ 부정형 — 최고차항이 승자

분자와 분모 다 무한으로 가면 ∞/∞ 부정형이야. 이때는 최고차항으로 나눠서 해결해.

lim 3x² + 5x - 1 x→∞ ------------ 2x² - x + 4 최고차항 x²으로 분자·분모를 나누면: = lim (3 + 5/x - 1/x²) / (2 - 1/x + 4/x²) x→∞ x→∞이면 5/x→0, 1/x²→0, ... = 3/2

최고차항 비교 법칙

분자 차수 = n, 분모 차수 = m일 때:

n < m → 극한 = 0 (분모가 더 빨리 커짐) n = m → 극한 = 최고차항 계수비 (위 예시처럼) n > m → 극한 = ±∞ (분자가 더 빨리 커짐)

외울 필요 없어. "누가 더 빨리 무한으로 가나" 경쟁이라고 생각하면 돼.

무한대의 함정

∞는 숫자가 아니야. 그래서:

∞ - ∞ = ??? (0이 아닐 수 있어)
∞ / ∞ = ??? (1이 아닐 수 있어)
0 × ∞ = ??? (0이 아닐 수 있어)

이런 것들 전부 부정형이야. 추가 작업 없이 답을 알 수 없어.

5장. 좌극한과 우극한

🚶 건물 3층에 가려는데, 계단으로 2층에서 올라갈 수도 있고, 4층에서 내려올 수도 있어. 둘 다 3층에 도착하면 좋겠지만, 만약 2층에서 올라가면 3층인데 4층에서 내려오면 5층이라면? 그건 "3층"이라는 극한이 존재하지 않는다는 뜻이야.
좌극한: lim f(x) (왼쪽에서 접근) x→a⁻ 우극한: lim f(x) (오른쪽에서 접근) x→a⁺ 극한이 존재하려면: 좌극한 = 우극한 이어야 해!

예시: 절댓값 함수

f(x) = |x| / x x > 0이면: |x|/x = x/x = 1 x < 0이면: |x|/x = -x/x = -1 lim f(x) = -1 (좌극한) x→0⁻ lim f(x) = 1 (우극한) x→0⁺ 좌극한 ≠ 우극한 → lim f(x) 존재하지 않음! x→0

왼쪽에서 오면 -1이고 오른쪽에서 오면 1이니까, 0에서의 극한은 존재하지 않아.

구간별 함수에서 극한

{ x + 1, x < 2 f(x) = { { x² - 1, x ≥ 2 좌극한: lim (x+1) = 3 x→2⁻ 우극한: lim (x²-1) = 3 x→2⁺ 좌극한 = 우극한 = 3 → lim f(x) = 3 x→2

다행히 양쪽이 같으니까 극한이 존재해. 이런 확인은 구간별 함수에서 필수야.

6장. 연속 — 끊김 없는 함수

✏️ 연필을 종이에서 떼지 않고 쭉 그릴 수 있는 그래프. 그게 연속함수야. 중간에 구멍이 있거나, 점프하거나, 끊기면 연속이 아니야.
f(x)가 x = a에서 연속이려면 3가지 조건: 1. f(a)가 정의되어 있어야 하고 2. lim f(x)가 존재해야 하고 x→a 3. lim f(x) = f(a) 이어야 해 x→a 세 가지 중 하나라도 안 되면 불연속!

불연속의 종류

제거 가능 불연속: 구멍만 있는 경우. 극한은 존재하지만 f(a)가 없거나 극한과 다름.
예: f(x) = (x²-1)/(x-1), x=1에서 구멍

점프 불연속: 좌극한 ≠ 우극한.
예: f(x) = |x|/x, x=0에서 -1에서 1로 점프

무한 불연속: 극한이 ±∞.
예: f(x) = 1/x, x=0에서

연속인지 확인해보기: { (x²-4)/(x-2), x ≠ 2 f(x) = { { 5, x = 2 1. f(2) = 5 ← 정의됨 ✓ 2. lim (x²-4)/(x-2) = lim (x+2) = 4 ← 존재 ✓ x→2 x→2 3. 4 ≠ 5 ← f(2) ≠ 극한 ✗ → x = 2에서 불연속! (제거 가능)

7장. 미분 맛보기 — 극한이 만드는 마법

📈 극한을 왜 배울까? 바로 미분을 하기 위해서야. 미분은 "순간 변화율"을 구하는 거야. 자동차 속도계가 보여주는 "지금 이 순간의 속도"가 바로 미분이지.

서울에서 부산까지 400km를 4시간에 갔어. 평균 속도는?

평균 속도 = 거리/시간 = 400/4 = 100 km/h

근데 이건 "평균"이지, 매 순간 100km/h로 달린 건 아니잖아. 신호에서 멈추기도 하고, 고속도로에서는 130으로 달리기도 했을 거야. "딱 지금 이 순간"의 속도를 알고 싶으면?

순간 속도 = lim [f(t+h) - f(t)] / h h→0 시간 간격 h를 0에 가깝게 줄이면 → 평균 속도가 순간 속도가 돼!

이게 미분의 정의야

f'(x) = lim [f(x+h) - f(x)] / h h→0 예: f(x) = x² f'(x) = lim [(x+h)² - x²] / h h→0 = lim [x² + 2xh + h² - x²] / h h→0 = lim [2xh + h²] / h h→0 = lim (2x + h) h→0 = 2x

f(x) = x²의 미분은 f'(x) = 2x야. x = 3에서의 순간 변화율은 2(3) = 6이라는 뜻이지. 극한이 없으면 미분도 없어. 그래서 극한을 먼저 배우는 거야.

8장. 왜 부정형이 중요할까?

경제학에서의 극한

경제학은 극한 투성이야. 한계비용, 한계수입, 한계효용 — 전부 "아주 조금 더 생산/소비했을 때의 변화"를 말하는데, 이게 다 미분이고, 미분의 기초가 극한이야.

한계비용 = lim [C(q+1) - C(q)] / 1 q→∞는 아니고, 개념적으로 "물건 1개 더 만들 때 추가되는 비용" = 비용함수의 미분 C'(q)

속도와 가속도

위치 함수 s(t)의 미분 = 속도 v(t)
속도 함수 v(t)의 미분 = 가속도 a(t)

전부 극한으로 정의돼. "변화의 순간적인 비율"을 측정하는 게 극한의 핵심 역할이야.

복리와 자연상수 e

lim (1 + 1/n)ⁿ = e ≈ 2.71828... n→∞ 이자를 n번으로 쪼개서 복리로 붙이면 n이 무한히 커질 때 → e에 수렴 이게 자연상수 e의 정의야!

금융, 인구 증가, 방사성 붕괴 등 연속적인 성장/감소를 다루는 모든 곳에서 e가 나와. 극한이 만들어낸 가장 유명한 상수지.

극한은 "가까이 가는 것"을 엄밀하게 다루는 도구야. 그리고 이 도구 위에 미분, 적분, 급수 등 현대 수학의 거의 모든 것이 세워져 있어. 오늘 배운 건 그 거대한 건물의 기초를 놓은 거야.

퀴즈 — 극한 실력 테스트

10문제. 후반부로 갈수록 어려워져.

Q1. lim(x→3) (2x + 1) = ?

연속함수니까 대입하면 돼. 2(3) + 1 = 7.

Q2. lim(x→1) (x² - 1)/(x - 1) = ?

0/0 부정형. 인수분해: (x+1)(x-1)/(x-1) = x+1. x→1 대입하면 2.

Q3. lim(x→∞) (5x² + 3x)/(2x² - 1) = ?

분자·분모 최고차 항이 둘 다 x². 계수비 = 5/2.

Q4. 0/0은 어떤 값인가?

0/0은 부정형이야. 실제 극한값은 상황마다 달라. 인수분해나 유리화 등으로 추가 작업을 해야 알 수 있어.

Q5. f(x)가 x=a에서 연속이려면 필요한 조건이 아닌 것은?

연속의 조건은 정의, 극한 존재, 극한=함수값 세 가지야. f'(a) 존재(미분가능)는 연속보다 더 강한 조건이야. 연속이라고 반드시 미분 가능한 건 아니거든 (예: |x|의 x=0).

Q6. lim(x→∞) (3x + 1)/(x³ - 2) = ?

분자 차수(1) < 분모 차수(3)이니까 극한은 0. 분모가 훨씬 빨리 커지거든.

Q7. lim(x→2) (x² - 5x + 6)/(x - 2) = ?

0/0 부정형. 분자 = (x-2)(x-3). 약분하면 lim(x→2)(x-3) = 2-3 = -1.

Q8. 좌극한 = 3, 우극한 = 5일 때, 극한은?

좌극한 ≠ 우극한이면 극한은 존재하지 않아. 평균을 내는 게 아니야!

Q9. f(x) = x²의 미분을 극한 정의로 구하면?

lim[(x+h)²-x²]/h = lim[2xh+h²]/h = lim(2x+h) = 2x. 7장에서 풀었던 그 문제야.

Q10. lim(x→0) [√(x+4) - 2] / x = ?

0/0 부정형. 유리화: [√(x+4)-2][√(x+4)+2] / [x(√(x+4)+2)] = (x+4-4)/[x(√(x+4)+2)] = x/[x(√(x+4)+2)] = 1/(√(x+4)+2). x→0 대입하면 1/(2+2) = 1/4.
