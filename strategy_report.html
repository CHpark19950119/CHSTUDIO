2. SWOT 분석

Strengths (강점)

1. 실전 PSAT 90.66점 — 합격선까지 2.67점 — "0베이스"가 아니라 이미 사정권. 2024년 80점대에서 2025년 90.66까지 1년 만에 10점 상승. 이 상승 곡선은 학습 효과가 나타나고 있다는 증거.

2. "집중하면 폭발하는" 패턴 — 수능 전과목 4등급이었지만, 수원대 GPA 3.96 + 외대 편입 독학 합격 + JLPT N1. 기본 스펙이 아니라 집중 투입 시의 폭발력이 증명됨. 7급 PSAT에서도 같은 패턴(1년에 +10점).

3. AI 인프라 — Claude Code + Firestore + 텔레그램 자동화 + 학습 기록 시스템이 이미 구축됨. 합격수기 분석, 모의문제 생성, 오답 패턴 분석, 학습 피드백을 학원 없이 받을 수 있는 환경.

4. 전업 집중 가능 — 직장 병행이 아닌 전업. 부모 월20만 + 정다영 지원으로 생활비 해결. 하루 순공 6시간+ 확보가 물리적으로 가능.

5. 자기 객관화 능력 — 수필에서 증명됨. "공부를 최선을 다해 한 적이 없다"는 인식. 자기 패턴을 분석하는 데이터(로드맵 v13)가 이미 존재.

v2에서 추가 삭제: "사회과목 감각"을 강점에서 제외. 중학교 때 사회를 잘한 것과 국제법/국제정치학은 별개. 근거 없는 자신감보다는 실제 시험 데이터(PSAT 90.66)가 더 강력한 근거.

Weaknesses (약점)

1. 5급-7급 갭 문제 — 5급 PSAT는 2년 연속 70점대. 7급에서 90.66까지 올랐지만 5급 수준으로는 전혀 올라가지 않음. 7급 외무영사직에 올인하면 이 문제는 해소되지만, "내가 5급도 될 수 있다"는 미련이 집중을 방해할 수 있음.

2. 실행력 불안정 — 본인 진단. 순공 목표 미달이 상시적. 편입 때의 집중력(PC방 1시간 제한, 단어 무한 암기)이 지금은 재현 안 됨. "고통스럽고 나이를 의식한다"(10호기). 이건 "의지"가 아니라 환경+동기 문제.

3. 혼자 준비하는 구조적 한계 — "합격자 전원이 고시반/관리반/스터디에 소속됐다. 천홍은 혼자다." (로드맵 분석). 학원 없음, 스터디 없음. 토익 만료 착각도 혼자 준비한 결과.

4. 전공(2차) 미준비 — 7급 추가채용에서 1차만 합격하고 2차는 준비 못 함. 헌법/국제법/국제정치학 체계적 학습 경험 전무.

5. 어학 미확보 — 영어(TOEIC 790+), 제2외국어 성적 모두 확보 필요. JLPT N1이 있으니 일본어를 제2외국어로 쓸 수 있는지 확인 필요.

Opportunities (기회)

1. 합격선까지 2.67점 — 2025년 90.66으로 이미 사정권. 93.33까지 2.67점 = 75문 중 2문 더 맞추면 됨. 0베이스에서 시작하는 것과 차원이 다른 출발점.

2. 경쟁률 하락 추세 — 120:1(2022) → 66:1(2025). 상위권은 남아 있지만, 중하위권 이탈로 실질 경쟁 강도는 감소.

3. 제2외국어 필기 폐지 + JLPT N1 보유 — 2024년부터 공인성적 제출 방식. JLPT N1이 활용 가능하다면 제2외국어 준비가 불필요해짐 (확인 필요).

4. 올해(2026) 시험 가능 — 1차가 7/18, 2차가 10/15. 3.5개월(1차) + 6.5개월(2차) 남았음. 90.66 기반이라 3.5개월이면 93+ 도달 가능성이 현실적.

Threats (위협)

1. 합격선 급등 추세 — 87.66(2023) → 89.33(2024) → 93.33(2025). 3년간 5.67점 상승. 2026년 95점대 진입 가능성. 90.66이었던 네 점수가 내년엔 커트라인 밑일 수도 있음.

2. 재수/삼수생 — 2~4년차 수험생이 다수. 기출 5~10회독 한 사람들과 경쟁. 너도 2년차지만, 실질 집중도가 그들과 같았는지가 문제.

3. "숨고 싶다" 심리 — "5급을 준비할 자격을 가진 사람이 아니라고 느끼기 때문이다. 그래서 숨고 싶다." 이 심리가 7급에도 전이될 수 있음. 학습 회피의 근본 원인이 능력 아닌 자기 인식에 있다.

4. 2차(전공) 돌파 불확실 — 1차를 넘겨도 2차에서 헌법/국제법/국제정치학을 13주 안에 합격선까지 올려야 함. 2024년 2차 커트라인 90.66은 PSAT보다 높음.

3. 핵심 전략

전략 A: PSAT 자료해석 집중

근거: 합격선 93.33이면 3과목 합산 280/300. 현재 90.66이니 2.67점(=약 2문제) 부족. 이 2문제를 어디서 건지느냐가 핵심. 언어논리와 상황판단은 "감(읽기 능력)"에 의존해 단기 상승 폭이 제한적. 반면 자료해석은 계산 훈련과 유형 학습으로 가장 확실하게 점수를 올릴 수 있는 과목. 합격수기에서도 자료해석이 합격/불합격을 가른 사례가 반복적으로 나옴.

과목별 현재 점수 파악 필수: 2025년 외무영사직 90.66점이 언어논리/자료해석/상황판단 각각 몇 점인지 확인해야 함. 자료해석이 약한 건지, 상황판단이 약한 건지에 따라 전략이 완전히 달라짐. 기억나면 과목별 점수를 기입할 것.

단권화란?

여러 출처(기본서, 강의 필기, 기출 해설, 판례)의 정보를 하나의 기본서에 집중시키는 작업. 최종적으로 시험 직전에 "이 한 권만 보면 된다"는 상태를 만드는 것.

단권화 5단계 방법론

1기본서 선택 (1회독 전)

  • 과목당 기본서 딱 1권 확정. 바꾸지 않는다.
  • 국제법: 이상구 기본서 (합격수기 최다 추천)
  • 국제정치학: 김태형 기본서 or 전인수 기본서
  • 헌법: 황남기 기본서 (7급용, 분량 적절)
  • 선택 기준: "어렵지만 정확한 책" > "쉽지만 빠진 게 있는 책"

21~2회독: 뼈대 잡기

  • 1회독: 처음부터 끝까지 이해 안 돼도 넘긴다. 목적은 "전체 지도 그리기".
  • 소요: 과목당 2~3주 (하루 2시간 기준)
  • 하는 것: 목차를 외우고, 각 장이 무엇에 대한 건지 한 줄로 요약할 수 있을 정도.
  • 안 하는 것: 줄 긋기, 메모, 정리. 1회독에서 정리하면 시간 낭비.
  • 2회독: 핵심 개념에 형광펜 표시. 이해 안 되는 부분에 ? 표시. 여전히 정리하지 않음.

33~4회독: 기출 연동 + 표시 시작

  • 기출문제를 풀면서 "이 내용이 기본서 어디에 있는지" 찾아서 별(★) 표시.
  • 기출에 나온 적 있는 내용 = 별 1개. 2번 이상 = 별 2개.
  • 강의 필기나 판례 요약을 기본서 해당 페이지 여백에 포스트잇으로 붙임.
  • 포스트잇 규칙: 한 장에 3줄 이내. 길면 요약 부족.
  • 3회독 끝나면 ? 표시의 70%는 자연 해소됨. 남은 30%만 별도 리스트.

45~7회독: 속도전 + 백지현출

  • 이 단계부터 1회독 소요 시간이 급격히 줄어야 함 (2주 → 3~4일).
  • 백지현출 방법: 빈 종이에 "제3장: OO조약법" 쓰고, 기억나는 핵심 내용을 전부 쓴다. 쓴 다음 기본서와 대조. 빠진 것에 빨간 밑줄.
  • 백지현출 시점: 한 장(chapter)을 읽은 직후. 읽기→현출→대조를 한 세트로.
  • 틀린 것/빠진 것만 모은 "오답 포스트잇"을 기본서에 추가.

58~10회독: 시험 직전 정리

  • 기본서 전체를 1~2일 만에 훑을 수 있는 상태가 목표.
  • 별(★) 표시 + 포스트잇 + 오답 포스트잇만 집중. 나머지는 눈으로 스캔.
  • 최종 정리: A4 한 장에 과목 전체의 "체계도" 그리기. 이것이 시험장에서 머릿속에 펼칠 지도.

단권화 도구

  • 형광펜 2색: 노랑(핵심 개념) + 분홍(기출 출제 포인트)
  • 포스트잇 2색: 파랑(강의/판례 보충) + 빨강(오답/약점)
  • ? 표시: 연필 (나중에 지울 수 있도록)
  • 별(★): 검정 펜 (기출 빈도)

핵심 원칙: "이해하고 넘어가자"가 아니라 "80%에서 넘기고 반복으로 채운다"

네 패턴 경고: "완벽히 이해하고 넘어가려다 진도가 안 나가는" 것이 가장 큰 위험. 이해 못 해도 ? 표시하고 넘겨. 3회독 때 70%는 자연 해소됨. 나머지 30%는 AI에게 질문.

"31세 + 실무 경험 없음" 대비

면접에서 "왜 이 나이에 시작했는가"는 100% 나오는 질문. 방어가 아니라 공격으로 전환해야 함:

  • "늦었기에 더 명확한 동기가 있다" — 자기 객관화 스토리
  • "공무원은 정년이 60세, 29년 남았다" — 시간은 충분
  • "실무 경험 없음" → "다양한 관점에서 정책을 바라볼 수 있음"으로 프레이밍
  • 자기기술서에서 "지금까지의 시행착오가 공직관을 형성했다"는 서사 구성

실행: 필기 합격 후 최소 2개월 면접 집중. 스터디 참여 필수. AI 모의면접으로 매일 1세트.

전략 D: AI를 지도자로 — 한계와 가능성

배경: 수필에서 "지도자가 없고 외부의 자원이 없던 시절"이라고 씀. 지금은 AI가 있음. 하지만 AI가 학원 강사를 완전히 대체할 수 있는가?

AI가 잘하는 것 (활용해야 할 것)

1. 개념 설명/질문 응답 — "이 조문이 무슨 뜻이야?" "A이론과 B이론 차이가 뭐야?" 수준의 질문에는 교수급 설명 가능. 24시간 무제한.

2. 기출 분석/패턴 발견 — 기출 10년치를 넣으면 출제 빈도, 유형 분류, 오답 패턴을 정리해줌. 사람이 직접 하면 며칠 걸릴 작업.

3. 모의문제 생성 — 기출 스타일 변형 문제, 약점 부분 집중 문제 생성 가능. 단, 실제 시험의 난이도/함정을 100% 재현하진 못함.

4. Feynman 기법 파트너 — 배운 것을 AI에게 설명하면 "이해가 부정확한 부분"을 지적해줌. 혼자 공부하는 사람에게 가장 가치 있는 기능.

5. 학습 계획/진도 관리 — 매주 학습량 추적, 약점 과목 경고, 타임라인 조정. 이건 이미 구축된 인프라로 가능.

6. 오답 구조 분석 — "왜 틀렸는지"를 "계산 실수 / 개념 혼동 / 함정 미인식 / 시간 부족" 등으로 분류. 감정적 "아 또 틀렸네" 대신 구조적 피드백.

AI가 못하는 것 (한계, 솔직하게)

1. 실전 감각 전달 불가 — "이 유형은 시험장에서 이렇게 접근해" 같은 노하우는 실제 합격자/강사만 줄 수 있음. AI는 문제를 풀 수 있지만 "시험장의 압박 속에서 어떻게 판단하는지"는 모름.

2. 최신 출제 경향 지연 — 2025~2026 최신 기출의 미묘한 경향 변화(난이도, 출제위원 교체 등)를 실시간으로 반영하지 못함. 강사/커뮤니티가 이런 정보를 더 빨리 잡음.

3. 채점/답안 퀄리티 검증 — 객관식이라 채점 자체는 명확하지만, "이 선지가 왜 틀린 건지" 미묘한 경우 AI의 해석이 출제자 의도와 다를 수 있음.

4. 동기 부여와 멘탈 관리 — AI는 "잘했어" "힘내" 따위의 말을 할 수 있지만, 3개월차 슬럼프에서 실질적으로 끌어올려주는 건 사람(스터디, 멘토, 가족). AI에게 정서적 지지를 기대하면 안 됨.

5. 강제성 부여 불가 — 학원은 "출석"이라는 물리적 강제가 있음. AI에게 "오늘 안 했으면 알려줘"라고 할 수는 있지만, 무시할 수 있음. 자기 규율이 전제.

결론: AI는 "보조 강사"이지 "담임 선생"이 아님

AI 활용 비율 추천: 전체 학습의 20~30%

나머지 70~80%: 기본서 직접 읽기, 기출 직접 풀기, 백지현출, 시간 측정 실전연습

구체적 AI 활용 루틴

  • 매일 아침: 전날 오답 AI에게 설명 → 피드백 (15분)
  • 개념 막힐 때: 즉시 질문 (기다리지 않고)
  • 주 1회: 그 주 학습 내용 Feynman 설명 → 이해도 체크 (30분)
  • 주 1회: AI 생성 모의문제 20문 풀기 (실전과 다름을 인지하면서)
  • 월 1회: 진도/약점 종합 리뷰 → 다음 달 계획 조정
내 의견 — 전략 D (AI 활용)

전략 E: 수학/경제학 — 보조 루틴 (폐기 아님)

v1 수정: v1에서 "수학/경제학에 시간 쓰지 말라"고 했는데, 이는 너무 극단적. 완전 폐기가 아니라 보조 루틴으로 유지.

왜 보조 루틴인가?

1. 불안 해소: "이것도 해야 하는데 안 하고 있다"는 불안이 오히려 메인 공부를 방해. 하루 30분~1시간이라도 하면 그 불안이 줄어듦.

2. 기초 체력: 수학적 사고력은 PSAT 자료해석의 기반. 숨마쿰라우데 수학II 수준의 기초는 자료해석 계산 속도에도 도움.

3. 장기 투자: 합격 후에도 경제학 기초는 외교부 업무(통상, FTA)에 직결. 완전 폐기하면 합격 후 다시 처음부터.

보조 루틴 규칙

시간: 하루 최대 1시간 (절대 초과 금지)
시점: 저녁 (메인 공부 끝난 후, 또는 쉬는 날)
내용: 숨마쿰라우데 수학II 진도 나가기
빈도: 주 4~5일

절대 금지:
- 메인 공부 시간을 빼서 수학/경제학에 쓰는 것
- "오늘 수학이 재미있어서 3시간 했다" — 이건 도피
- 수학/경제학을 "본업"으로 착각하는 것

메인: PSAT + 전공 (하루 5~6시간)

보조: 수학/경제학 (하루 0.5~1시간, 저녁)

보조가 메인을 침범하는 순간 중단.

내 의견 — 전략 E (수학/경제학)

4. 하지 말아야 할 것

1. 여러 교재 돌리기 — 한 권 10회독 > 다섯 권 2회독. "이 책이 별로야" 느낌은 3회독부터 사라짐.

2. 완벽한 이해 후 넘어가기 — 80%면 넘기고 반복. 완벽주의가 적. 이해 안 되는 건 ? 표시.

3. 경쟁자와 비교하기 — "4년차 재수생"은 네 머릿속에만 크게 보임. 실제로는 매년 신규 합격자가 30~40%.

4. 감상에 빠지기 — 수필은 아침 30분만. 나머지는 실행. "동기 부여 영상" 보는 것도 감상.

5. 학원/인강 쇼핑 — 학원 안 갈 거면 강의 비교에 시간 쓰지 마. 기본서 1권 + AI 질문으로 충분한지 먼저 테스트.

6. 계획 세우기에 시간 쓰기 — 이 보고서가 계획. 추가 계획 세우기는 공부 회피. "오늘 뭐 하지?" 고민에 30분 쓰면 그 30분이 공부.

7. 보조 루틴(수학)이 메인을 침범 — 수학이 "재미있다"고 3시간 하는 건 공부가 아니라 도피. 1시간 알람.

주말 루틴: 주 6일 공부 + 일요일 반일(오전만 or 오후만). 완전 쉬는 날은 슬럼프 유발.

모의고사 날 (주 1회): 오전 전체를 PSAT 모의고사(150분) + 채점/오답 분석에 투입.

내 의견 — 일일 루틴

7. 최종 판단

2026년 합격 가능성 — 솔직한 평가 (v2 데이터 기반)

1차 통과 확률: 40~55%

v2 수정. "0베이스에서 20~30%"가 아니라 "90.66에서 93+ 도달"의 확률. 2.67점 갭은 75문 중 2문제. 3.5개월간 오답 패턴 교정 + 약점 과목 집중이면 충분히 가능한 범위. 단, 합격선이 95점대로 올라가면 확률 급감. 합격선 유지가 전제.

1차 통과 시 2차 합격 확률: 30~40%

v2 하향 수정. 2024년 2차 커트라인이 90.66(PSAT보다 높음). 전공 0베이스에서 13주 안에 90+ 달성은 매우 도전적. 전공은 "암기량" 싸움이라 시간 투입이 선형적이긴 하지만, 국제법/헌법의 분량이 상당. 1차 전부터 전공 병행이 필수.

2026 or 2027 합격 확률 (2년 이내): 65~75%

v2 상향 수정. 90.66이라는 기반이 있고, 1년간 10점 상승 실적이 있음. 올해 1차 경험 + 전공 기반 구축 → 내년 본격 합격. 편입 때의 패턴("집중하면 폭발")이 재현되면 올해도 가능.

합격을 위한 비타협 조건

1. 매일 순공 5시간+ 유지 (예외: 주 1일 반일)

2. 한 권 10회독 원칙 (교재 바꾸지 않기)

3. 보조 루틴(수학)이 메인을 침범하지 않기

4. 주 1회 모의고사로 실전 감각 쌓기

5. 어학 5~6월 내 확보 (TOEIC + 제2외국어)

지금 당장 해야 할 것 (이번 주)

1. 2025년 외무영사직 PSAT 과목별 점수 확인 — 90.66이 언어/자료/상황 각 몇 점인지가 전략의 핵심. 성적 통지서 확인.

2. 2025년 오답 복기 — 기억나는 범위에서 "왜 틀렸는지" 유형 분류 (계산 실수 / 시간 부족 / 개념 모름 / 함정)

3. 제2외국어 확인 — JLPT N1이 7급 외무영사직 제2외국어로 인정되는지 확인. 인정되면 어학 부담 대폭 경감.

4. TOEIC 점수 확인 — 현재 유효한 점수가 있는지, 없으면 응시 일정 잡기

5. 전공 기본서 선택 확정: 국제법(이상구) + 헌법(황남기) + 국제정치학(미정) — 1차 전부터 전공 병행 시작
