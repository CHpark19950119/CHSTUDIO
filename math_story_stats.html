1장. 확률이 뭔데?

🎲 주사위를 던진다. 1이 나올 확률은? 6면 중 1면이니까 1/6. 짝수가 나올 확률은? 2, 4, 6 세 가지니까 3/6 = 1/2. 이렇게 "일어날 가능성을 숫자로 표현한 것"이 확률이야.
확률 = 원하는 경우의 수 / 전체 경우의 수 P(A) = n(A) / n(S) P: 확률 (Probability) A: 원하는 사건 S: 전체 표본 공간 n: 경우의 수

확률의 기본 성질

1. 0 ≤ P(A) ≤ 1 (확률은 0 이상 1 이하) 2. P(S) = 1 (뭔가는 반드시 일어남) 3. P(A') = 1 - P(A) (A가 안 일어날 확률 = 1 - A가 일어날 확률)

확률 0은 "절대 안 일어남", 확률 1은 "반드시 일어남"이야. 주사위에서 7이 나올 확률은 0이고, 1~6 중 하나가 나올 확률은 1이지.

카드 예시

52장 카드에서 한 장을 뽑을 때:

하트가 나올 확률 = 13/52 = 1/4 에이스가 나올 확률 = 4/52 = 1/13 하트 에이스 = 1/52 하트 또는 에이스 = ? P(하트 ∪ 에이스) = P(하트) + P(에이스) - P(하트 ∩ 에이스) = 13/52 + 4/52 - 1/52 = 16/52 = 4/13

합의 법칙: "또는"은 더하고, 겹치는 건 빼. 하트 에이스를 두 번 세면 안 되니까.

독립 사건과 곱의 법칙

동전을 두 번 던질 때, 첫 번째가 앞면이고 두 번째도 앞면일 확률은?

P(앞면 그리고 앞면) = P(앞면) × P(앞면) = 1/2 × 1/2 = 1/4

두 사건이 서로 영향을 안 주면 독립이야. 독립 사건의 "그리고"는 곱하면 돼. 첫 번째 동전이 앞면이든 뒷면이든, 두 번째 동전에는 영향이 없으니까.

2장. 조건부 확률 — 정보가 바꾸는 확률

🌂 내일 비 올 확률이 30%래. 그런데 지금 하늘에 먹구름이 잔뜩 끼어 있어. 그러면 비 올 확률이 여전히 30%일까? 아니지. "먹구름이 끼어 있다"는 정보가 확률을 바꿔. 이게 조건부 확률이야.
P(A|B) = P(A ∩ B) / P(B) "B가 일어났을 때, A도 일어날 확률" P(비|먹구름) = P(비 그리고 먹구름) / P(먹구름)

구체적 예시

어떤 반 학생 40명 중:

안경 O안경 X합계
남학생81220
여학생61420
합계142640
한 명을 뽑았는데 남학생이야. 이 학생이 안경을 쓸 확률은? P(안경|남학생) = 8/20 = 2/5 전체에서 안경 쓸 확률: 14/40 = 7/20 남학생 중 안경 쓸 확률: 8/20 = 2/5 → "남학생"이라는 정보가 확률을 바꿨어!

조건부 확률의 함정

P(A|B) ≠ P(B|A)야. 이걸 혼동하면 큰일나.

P(감기|기침) ≠ P(기침|감기) 기침할 때 감기일 확률 ← 의사가 알고 싶은 것 감기일 때 기침할 확률 ← 이건 다른 질문

범죄 수사, 의료 진단, 스팸 필터 등 현실에서 이 혼동이 자주 발생해. 조건을 정확히 읽는 게 중요해.

3장. 기댓값 — "평균적으로 얼마를 기대할 수 있나"

🎰 복권 한 장에 1,000원. 1등(100만원) 확률 1/10,000, 2등(1만원) 확률 1/100, 꽝 확률 나머지. 이 복권, 사는 게 이득일까? 기댓값을 계산하면 답이 나와.
기댓값 E(X) = Σ (값 × 확률) 복권 기댓값: = 1,000,000 × (1/10,000) + 10,000 × (1/100) + 0 × (나머지) = 100 + 100 + 0 = 200원 복권 가격 1,000원인데 기댓값 200원 → 평균적으로 800원 손해!

기댓값이 투자액보다 작으면 손해야. 물론 운 좋으면 100만원을 탈 수도 있지만, 수천 번 반복하면 한 장당 평균 800원씩 잃는다는 뜻이야.

주사위의 기댓값

E(X) = 1×(1/6) + 2×(1/6) + 3×(1/6) + 4×(1/6) + 5×(1/6) + 6×(1/6) = (1+2+3+4+5+6) / 6 = 21/6 = 3.5

주사위를 무한히 많이 던지면, 평균이 3.5에 수렴해. 물론 한 번도 3.5가 나온 적은 없지만, "기대할 수 있는 평균"이 3.5라는 거야.

기댓값과 의사결정

보험, 투자, 도박 — 전부 기댓값으로 판단할 수 있어.

게임: 동전을 던져서 앞면 → 3,000원 받음 뒷면 → 2,000원 잃음 참가비: 0원 E = 3000×(1/2) + (-2000)×(1/2) = 1500 - 1000 = 500원 → 기댓값 양수. 장기적으로 이득!

4장. 평균, 분산, 표준편차

📊 시험 점수 평균이 70점이래. 근데 이게 다 60~80점 사이에 모여 있는 건지, 30점짜리와 100점짜리가 섞인 건지는 평균만으로는 몰라. "점수가 얼마나 퍼져 있는가"를 알려주는 게 분산과 표준편차야.
평균 (mean): μ = (x₁ + x₂ + ... + xₙ) / n 분산 (variance): σ² = Σ(xᵢ - μ)² / n 표준편차 (standard deviation): σ = √(분산)

왜 제곱을 할까?

각 값에서 평균을 뺀 "편차"를 그냥 더하면, 양수와 음수가 상쇄돼서 0이 돼. 그래서 제곱해서 전부 양수로 만든 다음 평균을 내는 거야.

직접 계산해보자.

점수: 60, 70, 80, 90, 100 평균: (60+70+80+90+100)/5 = 400/5 = 80 편차: -20, -10, 0, 10, 20 편차²: 400, 100, 0, 100, 400 분산: (400+100+0+100+400)/5 = 1000/5 = 200 표준편차: √200 ≈ 14.14

표준편차가 말해주는 것

표준편차가 작으면 데이터가 평균 주위에 몰려 있고,
표준편차가 크면 데이터가 넓게 퍼져 있어.

A반: 평균 80, 표준편차 5 → 75~85점 근처에 집중 B반: 평균 80, 표준편차 20 → 60~100점까지 흩어져 있음 평균이 같아도 분포가 완전히 다를 수 있어!

분산의 계산 꿀팁

σ² = E(X²) - [E(X)]² "제곱의 평균 - 평균의 제곱" 점수: 60, 70, 80, 90, 100 E(X²) = (3600+4900+6400+8100+10000)/5 = 33000/5 = 6600 [E(X)]² = 80² = 6400 σ² = 6600 - 6400 = 200 ← 위와 같은 결과!

이 공식이 편차를 일일이 구하는 것보다 훨씬 빨라. 시험에서도 이 공식을 많이 써.

5장. 가중평균 — 모든 것이 동등하진 않아

⚖️ 중간고사 40%, 기말고사 60%로 성적을 매긴대. 중간 80점, 기말 90점이면 평균이 85점일까? 아니야. 기말이 더 비중이 크니까. 이럴 때 쓰는 게 가중평균이야.
가중평균 = Σ(값 × 가중치) / Σ(가중치) 중간 80점(40%) + 기말 90점(60%): = 80 × 0.4 + 90 × 0.6 = 32 + 54 = 86점 단순 평균 85점과 다르지?

가중평균의 실전 활용

A 주식: 수익률 10%, 투자금 300만원 B 주식: 수익률 -5%, 투자금 200만원 포트폴리오 수익률 = ? = (10% × 300 + (-5%) × 200) / (300 + 200) = (30 - 10) / 500 = 20/500 = 4% 단순 평균 (10-5)/2 = 2.5%와는 다름!

투자금(가중치)이 다르니까, A에 더 많이 투자한 만큼 A의 수익률이 더 크게 반영되는 거야.

GPA 계산도 가중평균이야. 3학점 과목의 A와 1학점 과목의 A는 같은 A여도 무게가 달라. 경제학 지표도 마찬가지야. GDP 성장률, 물가지수 전부 가중평균 기반이야.

6장. 정규분포 — 세상에서 가장 유명한 곡선

🔔 한국 성인 남성의 키를 쭉 조사하면, 대부분 170cm 근처에 몰려 있고, 150cm나 190cm는 드물어. 이 분포를 그래프로 그리면 종 모양이 돼. 이게 바로 정규분포(Normal Distribution)야.
정규분포 N(μ, σ²) μ: 평균 (종 모양의 중심) σ: 표준편차 (폭의 넓이) 특징: 1. 좌우 대칭 (평균 중심) 2. 평균 = 중앙값 = 최빈값 3. 평균에서 멀어질수록 빈도 급감

68-95-99.7 규칙

정규분포에서 가장 중요한 규칙이야. 외워두면 감이 와.

μ ± 1σ 안에 → 약 68%의 데이터 μ ± 2σ 안에 → 약 95%의 데이터 μ ± 3σ 안에 → 약 99.7%의 데이터 예: 키 평균 173cm, 표준편차 6cm 1σ: 167~179cm에 68% 2σ: 161~185cm에 95% 3σ: 155~191cm에 99.7% → 191cm 넘는 사람은 0.15% 미만!

표준정규분포 Z

모든 정규분포를 평균 0, 표준편차 1로 변환할 수 있어. 이걸 표준화라고 해.

Z = (X - μ) / σ 키 180cm인 사람의 Z값은? (μ=173, σ=6) Z = (180 - 173) / 6 = 7/6 ≈ 1.17 → 평균보다 1.17 표준편차만큼 큼 → 상위 약 12%에 해당

Z값으로 바꾸면 "평균에서 표준편차 몇 개 떨어져 있나"를 바로 알 수 있어. 서로 다른 시험 점수도 Z값으로 비교할 수 있지.

왜 정규분포가 그렇게 많이 나올까?

중심극한정리(Central Limit Theorem) 때문이야. 어떤 분포든, 표본을 충분히 많이 뽑아서 평균을 내면 정규분포에 가까워져. 주사위 하나는 균등분포지만, 100개 주사위의 평균은 정규분포를 따라. 그래서 자연현상, 측정 오차, 시험 점수 등 많은 것들이 정규분포를 따르는 거야.

7장. 경제학에서 통계가 왜 중요할까?

📈 "올해 경제성장률 2.5%", "실업률 3.7%", "소비자물가지수 전년 대비 3.2% 상승" — 뉴스에서 맨날 나오는 이 숫자들, 전부 통계야. 경제학은 통계 없이는 한 발짝도 못 나가.

GDP와 평균의 함정

1인당 GDP가 3만 달러래. 근데 재벌이 몇 명 있으면 평균이 확 올라가지. 평균만으로는 실상을 알 수 없어. 그래서 중앙값, 분산, 지니계수 같은 걸 같이 봐야 해.

5명의 소득 (만원): 100, 200, 300, 400, 9000 평균: 2,000만원 (한 명이 왜곡) 중앙값: 300만원 (실제 체감에 가까움)

기댓값과 리스크

투자에서 기댓값이 같아도 분산이 다르면 위험도가 달라.

투자 A: 기댓값 5%, 표준편차 2% 투자 B: 기댓값 5%, 표준편차 15% A: 거의 3~7% 사이 (안정적) B: -10%~20% 오갈 수 있음 (위험)

경제학에서는 기댓값(수익)과 분산(리스크)을 동시에 고려해. 이게 현대 포트폴리오 이론의 기초야.

회귀분석 맛보기

"교육 수준이 높으면 소득도 높을까?" 이런 질문에 답하려면 회귀분석을 써. 데이터에서 패턴(직선이나 곡선)을 찾아내는 거야.

소득 = a × 교육년수 + b + 오차 데이터를 모아서 a와 b를 추정 → "교육 1년 추가 시 소득 평균 X만원 증가" 이게 정책 결정의 근거가 돼.

이건 나중에 더 깊이 배우겠지만, 핵심은 통계가 "느낌"이 아니라 "숫자 근거"를 제공한다는 거야. 경제학이 과학이 되려면 통계가 필수야.

확률과 통계는 불확실한 세상을 이해하는 도구야. 100% 확실한 건 거의 없지만, 확률로 판단하면 더 나은 결정을 내릴 수 있어. 경제학을 공부하면 할수록 이 도구의 가치를 실감하게 될 거야.

퀴즈 — 확률과 통계 실력 테스트

10문제를 풀어봐.

Q1. 주사위를 한 번 던질 때, 3 이하가 나올 확률은?

3 이하: 1, 2, 3 → 3가지. 전체 6가지. 3/6 = 1/2.

Q2. 동전 2개를 던질 때, 적어도 하나가 앞면일 확률은?

전체: (앞앞, 앞뒤, 뒤앞, 뒤뒤) 4가지. 둘 다 뒷면인 경우만 빼면 돼. 1 - 1/4 = 3/4.

Q3. 주사위 기댓값은?

(1+2+3+4+5+6)/6 = 21/6 = 3.5. 3장에서 계산했지.

Q4. 데이터 {2, 4, 6, 8, 10}의 분산은?

평균 = 6. 편차: -4, -2, 0, 2, 4. 편차제곱: 16, 4, 0, 4, 16. 분산 = 40/5 = 8.

Q5. 중간 70점(30%), 기말 90점(70%)일 때 가중평균은?

70 x 0.3 + 90 x 0.7 = 21 + 63 = 84점.

Q6. 정규분포에서 평균 ± 2 표준편차 안에 들어가는 데이터의 비율은 약?

68-95-99.7 규칙! ±1σ=68%, ±2σ=95%, ±3σ=99.7%.

Q7. P(A) = 0.3, P(B) = 0.5, A와 B가 독립일 때 P(A ∩ B) = ?

독립이면 P(A ∩ B) = P(A) x P(B) = 0.3 x 0.5 = 0.15.

Q8. 표준편차가 큰 데이터의 특징은?

표준편차는 "데이터가 평균에서 얼마나 퍼져 있는가"를 나타내. 클수록 넓게 퍼져 있어.

Q9. 평균 170cm, 표준편차 5cm인 정규분포에서 Z = (X-170)/5일 때, 키 180cm의 Z값은?

Z = (180 - 170) / 5 = 10/5 = 2. 평균에서 표준편차 2개만큼 위에 있다는 뜻이야.

Q10. 게임: 이기면 +5000원(확률 40%), 지면 -3000원(확률 60%). 기댓값은?

E = 5000 x 0.4 + (-3000) x 0.6 = 2000 - 1800 = +200원. 장기적으로 살짝 이득이야.
