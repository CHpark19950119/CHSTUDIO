1. 데카르트 혁명 — 좌표계의 발명이 왜 혁명인가

1637년, 프랑스 철학자 르네 데카르트(Rene Descartes)는 『방법서설(Discours de la methode)』 부록으로 『기하학(La Geometrie)』을 발표했다. 침대에 누워 천장의 파리를 바라보다 영감을 얻었다는 유명한 일화가 전해지는데 — 사실 여부는 불확실하지만, 그가 한 일의 본질은 정확히 "위치를 숫자로 번역하는 것"이었다.

두 세계의 결합

데카르트 이전 2천 년간, 수학은 두 갈래로 나뉘어 있었다:

  • 기하학: 유클리드가 정리한 도형과 공간의 세계. 컴퍼스와 자로 증명하는 학문.
  • 대수학: 알콰리즈미 이래 방정식과 숫자를 다루는 학문.

"직각삼각형에서 빗변의 길이를 구하라" — 이건 기하 문제였어. "$x^2 + y^2 = 25$를 만족하는 $x, y$를 구하라" — 이건 대수 문제였고. 두 문제가 사실 같은 걸 말하고 있다는 것을 체계적으로 연결한 사람이 데카르트야.

데카르트의 핵심 아이디어
수직으로 교차하는 두 수직선을 설정하면, 평면 위의 모든 점을 정확히 하나의 순서쌍 $(x, y)$으로 표현할 수 있다. 거꾸로, 모든 순서쌍은 정확히 하나의 점에 대응한다.

이 순간 도형은 방정식이 되고, 방정식은 도형이 된다. 기하학과 대수학이 하나로 합쳐졌다.

이게 왜 혁명이냐면:

  • 번역: 기하학적 직관("원은 중심에서 같은 거리의 점들")이 대수적 식($x^2+y^2=r^2$)으로 번역됨
  • 계산: 도형 문제를 방정식 풀이로 해결 가능 — 기계적, 체계적
  • 발견: 방정식을 먼저 쓰고, 그것이 어떤 도형인지 나중에 확인하는 것도 가능. 새로운 곡선(포물선, 타원, 쌍곡선)을 대수적으로 분류할 수 있게 됨
  • 미적분의 기초: 뉴턴과 라이프니츠의 미적분학은 데카르트 좌표 없이는 불가능했다

왜 하필 수직 교차인가? 비스듬한 좌표계(사교좌표, oblique coordinates)도 수학적으로 가능하다. 하지만 직교좌표에서는 피타고라스 정리가 바로 적용되어 거리 공식이 깔끔해진다. 편리함이 표준이 되는 전형적인 예야. 일반상대론에서는 좌표축이 직교하지 않는 경우(리만 기하)도 다루는데, 그건 훨씬 나중 이야기.

2. 거리 공식 — 피타고라스에서 n차원까지

두 점 사이의 거리

좌표평면 위 두 점 $A(x_1, y_1)$, $B(x_2, y_2)$ 사이의 거리는 어떻게 구할까? 이게 좌표기하학의 첫 번째 도구야.

유도

$A$에서 수평선, $B$에서 수직선을 그으면 직각삼각형이 생긴다.
가로 길이: $|x_2 - x_1|$, 세로 길이: $|y_2 - y_1|$

피타고라스 정리에 의해: $$d^2 = (x_2 - x_1)^2 + (y_2 - y_1)^2$$ $$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$
두 점 사이의 거리 공식 $d(A, B) = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
예제
$A(1, 2)$, $B(4, 6)$ 사이의 거리:
$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9+16} = \sqrt{25} = 5$

n차원 확장

놀라운 점은, 같은 논리가 더 높은 차원으로 자연스럽게 확장된다는 거야.

3차원 $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$
n차원 (유클리드 거리) $d(\mathbf{p}, \mathbf{q}) = \sqrt{\displaystyle\sum_{i=1}^{n}(p_i - q_i)^2}$

이 공식은 머신러닝에서 데이터 포인트 사이의 유사도를 측정할 때도 쓴다 (K-최근접 이웃 알고리즘 등). 고등학교 공식이 현대 AI의 기초가 되는 셈이야.

다른 거리 개념: 택시 거리 (맨해튼 거리)

유클리드 거리만이 "거리"는 아니야. 맨해튼처럼 격자 도로만 다닐 수 있다면?

택시 거리 (Manhattan distance) $d_{\text{taxi}}(A, B) = |x_2-x_1| + |y_2-y_1|$

$A(1,1)$, $B(5,5)$라면: 유클리드 거리 $= \sqrt{32} \approx 5.66$, 택시 거리 $= 4+4=8$. 택시 거리가 항상 유클리드 거리 이상이야 (삼각부등식의 등호가 직각일 때만 성립).

3. 내분과 외분 — 가중평균과 무게중심

내분점

선분 $AB$ 위의 점 $P$가 $AP:PB = m:n$을 만족할 때, $P$를 $A$와 $B$를 $m:n$으로 내분하는 점이라 한다.

내분점 공식 $A(x_1, y_1)$, $B(x_2, y_2)$를 $m:n$으로 내분하는 점:
$P\left(\dfrac{mx_2 + nx_1}{m+n},\; \dfrac{my_2 + ny_1}{m+n}\right)$
유도

$AP:PB = m:n$이므로 $P$는 $A$에서 전체 거리의 $\frac{m}{m+n}$만큼 이동한 위치.

x좌표: $x_P = x_1 + \frac{m}{m+n}(x_2 - x_1) = \frac{(m+n)x_1 + m(x_2-x_1)}{m+n} = \frac{mx_2+nx_1}{m+n}$
y좌표도 마찬가지. $\blacksquare$

특히 $m = n = 1$이면 중점:

중점 공식 $M\left(\dfrac{x_1+x_2}{2},\; \dfrac{y_1+y_2}{2}\right)$

내분점은 가중평균이다

내분점 공식을 잘 보면, 이건 사실 가중평균(weighted average)이야. $A$와 $B$에 각각 $n$과 $m$의 가중치를 준 평균이지. $B$에 가까울수록 $m$이 크다는 점에 주의해 — $m:n$에서 $m$은 $A$ 쪽 길이가 아니라 $B$ 쪽 가중치야.

시소를 생각해봐: $A$에 $n$kg 추, $B$에 $m$kg 추를 달면, 균형점이 내분점이야. 무거운 쪽에 가까워지지.

외분점

선분 $AB$의 연장선 위에 있으면서 $AP:PB = m:n$을 만족하는 점 $P$가 외분점이야 ($m \neq n$).

외분점 공식 $P\left(\dfrac{mx_2 - nx_1}{m-n},\; \dfrac{my_2 - ny_1}{m-n}\right)$
외분점에서 $m = n$이면 분모가 0이 되어 점이 존재하지 않는다. 기하학적으로, $AP:PB = 1:1$인 외분점은 "무한히 먼 점"이 되어버려. 이건 사영기하학(projective geometry)에서 다루는 "무한원점"과 관련이 있다.

삼각형의 무게중심

삼각형 $ABC$의 세 꼭짓점 좌표가 주어지면, 무게중심(centroid) $G$는:

무게중심 $G\left(\dfrac{x_1+x_2+x_3}{3},\; \dfrac{y_1+y_2+y_3}{3}\right)$
유도

무게중심은 각 중선(꼭짓점에서 대변의 중점으로 그은 선분)의 교점이고, 각 중선을 $2:1$로 내분한다.

$A(x_1,y_1)$에서 $BC$의 중점 $M\left(\frac{x_2+x_3}{2}, \frac{y_2+y_3}{2}\right)$까지의 중선을 $2:1$로 내분:
$G_x = \frac{2 \cdot \frac{x_2+x_3}{2} + 1 \cdot x_1}{2+1} = \frac{x_1+x_2+x_3}{3}$ $\blacksquare$

무게중심 = 세 좌표의 산술평균. 가중치가 모두 같은 가중평균, 즉 "공평한" 평균이야.

4. 직선의 방정식 심화 — 4가지 형태 완전 정리

기울기의 의미

기울기(slope) $m$은 "$x$가 1 증가할 때 $y$가 얼마나 변하는가"를 나타내:

$m = \dfrac{y_2 - y_1}{x_2 - x_1} = \dfrac{\Delta y}{\Delta x} = \dfrac{\text{y의 변화량}}{\text{x의 변화량}}$
기울기 = 변화율 = (나중에 배울) 미분
직선의 기울기는 미적분에서 배울 도함수(derivative)의 가장 단순한 형태야. 곡선에서는 기울기가 점마다 달라지는데, 그 "순간 기울기"가 미분이다. 직선은 기울기가 일정한 유일한 곡선이야. 그래서 직선의 방정식을 완벽히 이해하는 것이 미적분의 기초가 된다.

4가지 직선 방정식 형태

1. 기울기-절편형 (Slope-Intercept)

$y = mx + b$

$m$: 기울기, $b$: y절편. 가장 친숙한 형태. 기울기와 y절편을 바로 읽을 수 있다.

2. 점-기울기형 (Point-Slope)

$y - y_1 = m(x - x_1)$

한 점 $(x_1, y_1)$과 기울기 $m$을 알 때 가장 편리. 문제 풀 때 가장 많이 쓰게 될 형태야.

3. 두 점형 (Two-Point)

$\dfrac{y - y_1}{y_2 - y_1} = \dfrac{x - x_1}{x_2 - x_1}$

두 점 $(x_1,y_1)$, $(x_2,y_2)$를 지나는 직선. 점-기울기형에서 $m = \frac{y_2-y_1}{x_2-x_1}$을 대입한 것.

4. 일반형 (Standard/General)

$ax + by + c = 0 \quad (a, b\text{가 동시에 0이 아닌)}$

모든 직선(수직선 포함)을 표현할 수 있는 가장 일반적인 형태. $b = 0$이면 수직선 $x = -\frac{c}{a}$.

형태장점단점
$y = mx + b$기울기/절편 즉시 확인수직선 표현 불가
$y - y_1 = m(x - x_1)$한 점 + 기울기로 즉시 작성수직선 표현 불가
두 점형두 점 직접 대입수직선이면 분모 0
$ax + by + c = 0$모든 직선 표현 가능기울기가 즉시 안 보임

절편형 (추가)

x절편 $a$, y절편 $b$ (둘 다 0이 아닐 때) $\dfrac{x}{a} + \dfrac{y}{b} = 1$

점과 직선 사이의 거리

점 $(x_0, y_0)$에서 직선 $ax+by+c=0$까지의 거리 $d = \dfrac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}$
증명 (벡터 투영법)

직선 $ax+by+c=0$ 위의 임의의 점을 $Q(x_1, y_1)$라 하자 ($ax_1+by_1+c=0$).
점 $P(x_0, y_0)$에서 직선까지의 거리는, 벡터 $\vec{QP}$를 직선의 법선벡터 $\vec{n} = (a, b)$ 방향으로 투영한 길이야.

$d = \frac{|\vec{QP} \cdot \vec{n}|}{|\vec{n}|} = \frac{|a(x_0-x_1) + b(y_0-y_1)|}{\sqrt{a^2+b^2}}$

$ax_1+by_1 = -c$이므로:
$= \frac{|ax_0+by_0 - (ax_1+by_1)|}{\sqrt{a^2+b^2}} = \frac{|ax_0+by_0+c|}{\sqrt{a^2+b^2}}$ $\blacksquare$
증명 2 (직접 계산 — 수선의 발)

$P(x_0, y_0)$에서 직선 $ax+by+c=0$에 내린 수선의 발 $H$를 구하자.
수선의 방향벡터는 법선벡터 $(a, b)$와 같으므로:
$H = P - t\vec{n}$ where $t = \frac{ax_0+by_0+c}{a^2+b^2}$

$d = |PH| = |t| \cdot |\vec{n}| = \frac{|ax_0+by_0+c|}{a^2+b^2} \cdot \sqrt{a^2+b^2} = \frac{|ax_0+by_0+c|}{\sqrt{a^2+b^2}}$ $\blacksquare$
예제: 점과 직선 사이의 거리
점 $(3, 4)$에서 직선 $3x + 4y - 5 = 0$까지의 거리:
$d = \frac{|3(3)+4(4)-5|}{\sqrt{9+16}} = \frac{|9+16-5|}{\sqrt{25}} = \frac{20}{5} = 4$
응용: 두 평행한 직선 사이의 거리
$ax+by+c_1=0$과 $ax+by+c_2=0$ 사이의 거리:
첫 직선 위의 아무 점에서 두 번째 직선까지의 거리를 구하면 된다.
$d = \frac{|c_1-c_2|}{\sqrt{a^2+b^2}}$

5. 두 직선의 관계 — 평행, 수직, 교각

평행

두 직선이 평행할 조건 $\ell_1: y = m_1 x + b_1, \quad \ell_2: y = m_2 x + b_2$
$\ell_1 \parallel \ell_2 \iff m_1 = m_2 \text{ (그리고 } b_1 \neq b_2\text{)}$

기울기가 같으면 평행. 기울기도 같고 절편도 같으면 같은 직선(일치).

수직

두 직선이 수직일 조건 $\ell_1 \perp \ell_2 \iff m_1 \cdot m_2 = -1$
증명 (내적을 이용)

직선 $y = m_1 x$의 방향벡터: $\vec{d_1} = (1, m_1)$
직선 $y = m_2 x$의 방향벡터: $\vec{d_2} = (1, m_2)$

두 직선이 수직 $\iff$ $\vec{d_1} \cdot \vec{d_2} = 0$
$1 \cdot 1 + m_1 \cdot m_2 = 0$
$m_1 m_2 = -1$ $\blacksquare$
$m_1 m_2 = -1$이라는 건, 한쪽 기울기가 양수면 다른 쪽은 음수이고, 크기는 역수 관계라는 뜻이야. 예를 들어 기울기가 $2$인 직선에 수직인 직선의 기울기는 $-\frac{1}{2}$. 하나가 가파르면 다른 하나는 완만하고, 방향이 반대.

두 직선이 이루는 각

두 직선의 교각 $\theta$ $\tan\theta = \left|\dfrac{m_1 - m_2}{1 + m_1 m_2}\right| \quad (m_1 m_2 \neq -1)$
유도

직선 $\ell_1$이 x축과 이루는 각 $\alpha_1$, 직선 $\ell_2$가 이루는 각 $\alpha_2$라 하면:
$m_1 = \tan\alpha_1$, $m_2 = \tan\alpha_2$
두 직선의 교각 $\theta = \alpha_1 - \alpha_2$ (또는 그 보각)

$\tan\theta = \tan(\alpha_1-\alpha_2) = \frac{\tan\alpha_1 - \tan\alpha_2}{1+\tan\alpha_1\tan\alpha_2} = \frac{m_1-m_2}{1+m_1 m_2}$

예각을 취하기 위해 절댓값. $\blacksquare$
예제
$\ell_1: y = 2x+1$, $\ell_2: y = \frac{1}{3}x - 2$의 교각:
$\tan\theta = \left|\frac{2 - \frac{1}{3}}{1 + 2 \cdot \frac{1}{3}}\right| = \left|\frac{\frac{5}{3}}{\frac{5}{3}}\right| = 1$
$\theta = 45°$

일반형에서의 평행/수직 조건

$a_1 x + b_1 y + c_1 = 0$ 과 $a_2 x + b_2 y + c_2 = 0$ 평행: $a_1 b_2 = a_2 b_1$ (그리고 일치가 아닌)
수직: $a_1 a_2 + b_1 b_2 = 0$

일반형에서 $(a, b)$가 법선벡터라는 점에서 유도됨: 평행이면 법선벡터도 평행, 수직이면 법선벡터도 수직.

6. 원의 방정식 심화 — 표준형, 일반형, 완전제곱식

원의 정의

원이란 평면 위에서 한 점(중심)으로부터 일정한 거리(반지름)에 있는 점들의 집합이야. 데카르트의 혁명 덕분에, 이 기하학적 정의를 방정식으로 쓸 수 있다.

표준형

원의 표준형 (중심 $(a,b)$, 반지름 $r$) $(x-a)^2 + (y-b)^2 = r^2$

이건 거리 공식 자체야: "점 $(x,y)$에서 중심 $(a,b)$까지의 거리 = $r$".

일반형과 완전제곱식

표준형을 전개하면 일반형이 돼:

원의 일반형 $x^2 + y^2 + Dx + Ey + F = 0$

여기서 $D = -2a$, $E = -2b$, $F = a^2 + b^2 - r^2$. 역으로, 일반형에서 표준형으로 돌아가려면 완전제곱식(completing the square)을 써야 해.

일반형 → 표준형 변환

$x^2 + y^2 + Dx + Ey + F = 0$

$x$ 묶기: $x^2 + Dx = \left(x + \frac{D}{2}\right)^2 - \frac{D^2}{4}$
$y$ 묶기: $y^2 + Ey = \left(y + \frac{E}{2}\right)^2 - \frac{E^2}{4}$

대입: $\left(x + \frac{D}{2}\right)^2 + \left(y + \frac{E}{2}\right)^2 = \frac{D^2+E^2}{4} - F$

중심: $\left(-\frac{D}{2}, -\frac{E}{2}\right)$, 반지름: $r = \sqrt{\frac{D^2+E^2}{4}-F}$
$\frac{D^2+E^2}{4} - F > 0$이어야 원이 된다.
$= 0$이면 점(반지름 0), $< 0$이면 나타내는 도형이 없다.
예제: 일반형 → 표준형
$x^2 + y^2 - 4x + 6y - 3 = 0$

$(x^2 - 4x + 4) + (y^2 + 6y + 9) = 3 + 4 + 9$
$(x-2)^2 + (y+3)^2 = 16$

중심 $(2, -3)$, 반지름 $4$

원의 기본 성질

  • 원 위의 한 점에서의 접선은 그 점과 중심을 잇는 반지름에 수직
  • 현의 수직이등분선은 중심을 지남
  • 중심에서 현까지의 거리를 $d$, 반지름을 $r$, 현의 반 길이를 $l$이라 하면: $r^2 = d^2 + l^2$

세 점으로 원 결정

일직선 위에 있지 않은 세 점이 주어지면, 그 세 점을 지나는 원은 유일하게 존재한다. 일반형 $x^2+y^2+Dx+Ey+F=0$에 세 점을 대입하면 $D, E, F$에 대한 연립방정식 3개가 나오고, 이를 풀면 된다.

예제: 세 점 $(0,0)$, $(4,0)$, $(0,6)$을 지나는 원
$x^2+y^2+Dx+Ey+F=0$에 대입:
$(0,0)$: $F = 0$
$(4,0)$: $16+4D+F=0$ → $D = -4$
$(0,6)$: $36+6E+F=0$ → $E = -6$

$x^2+y^2-4x-6y=0$ → $(x-2)^2+(y-3)^2 = 13$
중심 $(2,3)$, 반지름 $\sqrt{13}$

7. 원과 직선 — 판별식 vs 거리, 접선

교점의 개수

원 $(x-a)^2+(y-b)^2=r^2$과 직선 $y=mx+n$의 관계를 판정하는 두 가지 방법이 있어.

방법 1: 판별식

직선의 식을 원에 대입하면 $x$에 대한 이차방정식이 나온다. 그 판별식 $D$로 교점 수를 판정:

$D > 0$: 두 점에서 만남 (할선)
$D = 0$: 한 점에서 접함 (접선)
$D < 0$: 만나지 않음 (이선)

방법 2: 중심-직선 거리

중심 $(a,b)$에서 직선까지의 거리 $d$를 구해서 반지름 $r$과 비교:

$d < r$: 두 점에서 만남
$d = r$: 접함
$d > r$: 만나지 않음
어떤 방법이 더 좋은가?
거리 방법이 대부분의 경우 더 효율적이야. 이차방정식을 풀 필요 없이, 거리 공식 한 번으로 판정할 수 있으니까. 판별식 방법은 교점의 좌표 자체가 필요할 때 써.

접선의 방정식

원 위의 점에서의 접선

원 $x^2+y^2=r^2$ 위의 점 $(x_1, y_1)$에서의 접선 $x_1 x + y_1 y = r^2$
유도

접선은 접점과 원점을 잇는 반지름에 수직.
반지름 방향벡터: $(x_1, y_1)$
접선 위의 점 $(x, y)$에서 접점까지의 벡터: $(x-x_1, y-y_1)$

수직 조건: $x_1(x-x_1) + y_1(y-y_1) = 0$
$x_1 x + y_1 y = x_1^2 + y_1^2 = r^2$ (점이 원 위에 있으므로) $\blacksquare$
일반 원 $(x-a)^2+(y-b)^2=r^2$ 위의 점 $(x_1, y_1)$에서의 접선 $(x_1-a)(x-a) + (y_1-b)(y-b) = r^2$

원 밖의 점에서의 접선

원 밖의 점 $P$에서는 두 개의 접선을 그을 수 있다. 접선의 기울기를 $m$으로 놓고, "$P$를 지나는 직선 $y - y_0 = m(x-x_0)$이 원에 접한다" 조건($d = r$)을 쓰면 $m$에 대한 이차방정식이 나와. 두 근이 두 접선의 기울기.

예제: 원 밖의 점에서 접선
원 $x^2+y^2=5$에 점 $(3, 1)$에서 그은 접선의 방정식:

(i) $(3,1)$이 원 위의 점인지 확인: $9+1=10 \neq 5$ → 원 밖의 점

(ii) 접선: $y-1 = m(x-3)$ → $mx-y+(1-3m) = 0$
원의 중심 $(0,0)$에서의 거리 = $\sqrt{5}$:
$\frac{|1-3m|}{\sqrt{m^2+1}} = \sqrt{5}$
$(1-3m)^2 = 5(m^2+1)$
$9m^2 - 6m + 1 = 5m^2 + 5$
$4m^2 - 6m - 4 = 0$ → $2m^2 - 3m - 2 = 0$
$(2m+1)(m-2) = 0$ → $m = -\frac{1}{2}$ 또는 $m = 2$

접선: $y = 2x-5$ 또는 $y = -\frac{1}{2}x + \frac{5}{2}$

현의 길이

원의 중심에서 현까지의 거리 $d$일 때, 현의 길이 $\ell = 2\sqrt{r^2 - d^2}$

직각삼각형: 반지름 $r$ = 빗변, 중심-현 거리 $d$ = 한 변, 현의 반 길이 = 다른 변.

8. 자취의 방정식 — 조건이 만드는 도형

자취란 무엇인가

자취(locus)란 주어진 조건을 만족하는 점 전체의 집합이야. "원"이라는 도형 자체가 "한 점에서 일정한 거리에 있는 점의 자취"인 셈이지. 자취 문제는 기하학적 조건을 대수적 방정식으로 번역하는 연습이야 — 데카르트 혁명의 본질.

자취 방정식 구하는 방법

자취 방정식 풀이 전략
  1. 자취 위의 임의의 점을 $P(x, y)$로 놓는다
  2. 주어진 조건을 $x, y$로 표현한다
  3. 정리하여 $x, y$ 사이의 관계식(방정식)을 얻는다
  4. 필요하면 범위 조건을 확인한다
예제 1: 두 점에서 같은 거리
$A(1, 0)$과 $B(5, 0)$에서 같은 거리에 있는 점의 자취는?

$PA = PB$:
$\sqrt{(x-1)^2+y^2} = \sqrt{(x-5)^2+y^2}$
양변 제곱: $(x-1)^2 = (x-5)^2$
$x^2-2x+1 = x^2-10x+25$
$8x = 24$ → $x = 3$

자취: 직선 $x = 3$ (선분 $AB$의 수직이등분선)
예제 2: 거리의 비가 일정
$A(0, 0)$, $B(6, 0)$에서의 거리의 비가 $PA:PB = 1:2$인 점의 자취는?

$PA = \frac{1}{2}PB$ → $2PA = PB$ → $4PA^2 = PB^2$:
$4(x^2+y^2) = (x-6)^2+y^2$
$4x^2+4y^2 = x^2-12x+36+y^2$
$3x^2+3y^2+12x-36=0$
$x^2+y^2+4x-12=0$
$(x+2)^2+y^2 = 16$

자취: 중심 $(-2, 0)$, 반지름 4인 원 (아폴로니우스의 원)
아폴로니우스의 원(Apollonius Circle): 두 고정점에서의 거리의 비가 일정한 점의 자취는 (비가 1이 아닐 때) 항상 원이 된다. 비가 정확히 1이면 수직이등분선(직선). 이건 기원전 3세기 아폴로니우스가 발견한 결과야.
예제 3: 직선까지의 거리와 점까지의 거리가 같은 자취
점 $F(0, 1)$과 직선 $y = -1$로부터 같은 거리에 있는 점의 자취는?

점까지의 거리 = 직선까지의 거리:
$\sqrt{x^2+(y-1)^2} = |y+1|$
양변 제곱: $x^2+(y-1)^2 = (y+1)^2$
$x^2+y^2-2y+1 = y^2+2y+1$
$x^2 = 4y$

자취: 포물선 $y = \frac{x^2}{4}$ (초점 $F$, 준선 $y=-1$)

포물선의 정의 자체가 "초점과 준선에서 같은 거리인 점의 자취"야.
예제 4: 움직이는 점의 자취
원 $x^2+y^2=16$ 위의 점 $Q$와 점 $A(4, 0)$을 잇는 선분 $QA$의 중점 $M$의 자취는?

$Q = (4\cos\theta, 4\sin\theta)$, $M = \left(\frac{4\cos\theta+4}{2}, \frac{4\sin\theta}{2}\right) = (2\cos\theta+2, 2\sin\theta)$

$x = 2\cos\theta+2$, $y = 2\sin\theta$
$(x-2)^2+y^2 = 4\cos^2\theta+4\sin^2\theta = 4$

자취: 중심 $(2, 0)$, 반지름 2인 원

자취와 이차곡선

자취 문제를 풀다 보면 원, 포물선, 타원, 쌍곡선 같은 이차곡선(conic sections)이 자주 등장해. 이 곡선들은 모두 $ax^2+bxy+cy^2+dx+ey+f=0$ 꼴의 이차방정식으로 표현돼. 원은 그 특수한 경우($a=c$, $b=0$)이고, 나머지는 수학II에서 다룬다.

이차곡선의 분류 (미리보기)
  • 원: 한 점에서 일정 거리
  • 타원: 두 점에서의 거리의 합이 일정
  • 쌍곡선: 두 점에서의 거리의 차가 일정
  • 포물선: 한 점과 한 직선에서의 거리가 같음
모두 원뿔을 평면으로 자른 단면이어서 "원뿔곡선(conic section)"이라 부른다.

확인 퀴즈 (15문항)

Q1. 데카르트의 좌표평면이 혁명적인 이유를 한 문장으로 설명하라.
탭하여 정답 확인
기하학(도형)과 대수학(방정식)을 통합하여, 도형 문제를 방정식으로, 방정식 문제를 그래프로 해결할 수 있게 만들었다.
Q2. $A(2, -1)$과 $B(8, 7)$ 사이의 거리를 구하라.
탭하여 정답 확인
$d = \sqrt{(8-2)^2+(7-(-1))^2} = \sqrt{36+64} = \sqrt{100} = 10$
Q3. $A(2, -1)$과 $B(8, 7)$의 택시 거리(맨해튼 거리)는?
탭하여 정답 확인
$d_{\text{taxi}} = |8-2| + |7-(-1)| = 6+8 = 14$
Q4. $A(1, 3)$과 $B(7, -1)$을 $3:1$로 내분하는 점의 좌표는?
탭하여 정답 확인
$P = \left(\frac{3 \cdot 7 + 1 \cdot 1}{3+1}, \frac{3 \cdot (-1) + 1 \cdot 3}{3+1}\right) = \left(\frac{22}{4}, \frac{0}{4}\right) = \left(\frac{11}{2}, 0\right)$
Q5. 삼각형의 꼭짓점이 $(0,0)$, $(6,0)$, $(3,9)$일 때, 무게중심의 좌표는?
탭하여 정답 확인
$G = \left(\frac{0+6+3}{3}, \frac{0+0+9}{3}\right) = (3, 3)$
Q6. 점 $(2, 3)$을 지나고 기울기가 $-\frac{3}{4}$인 직선의 방정식을 일반형으로 쓰라.
탭하여 정답 확인
점-기울기형: $y - 3 = -\frac{3}{4}(x-2)$
$4(y-3) = -3(x-2)$ → $4y-12 = -3x+6$ → $3x+4y-18=0$
Q7. 점 $(1, 2)$에서 직선 $3x-4y+5=0$까지의 거리를 구하라.
탭하여 정답 확인
$d = \frac{|3(1)-4(2)+5|}{\sqrt{9+16}} = \frac{|3-8+5|}{5} = \frac{0}{5} = 0$
거리가 0 → 점 $(1,2)$가 직선 위에 있다!
Q8. 두 직선 $y = 3x+1$과 $y = -\frac{1}{3}x+5$의 관계는? (평행/수직/기타)
탭하여 정답 확인
$m_1 \cdot m_2 = 3 \cdot \left(-\frac{1}{3}\right) = -1$ → 수직
Q9. $x^2+y^2+6x-2y-6=0$을 표준형으로 바꾸고, 중심과 반지름을 구하라.
탭하여 정답 확인
$(x^2+6x+9)+(y^2-2y+1) = 6+9+1$
$(x+3)^2+(y-1)^2 = 16$
중심 $(-3, 1)$, 반지름 $4$
Q10. 원 $x^2+y^2=25$과 직선 $y=x+7$의 교점의 개수를 거리 방법으로 판정하라.
탭하여 정답 확인
직선: $x-y+7=0$, 원의 중심 $(0,0)$, 반지름 $5$
$d = \frac{|0-0+7|}{\sqrt{1+1}} = \frac{7}{\sqrt{2}} = \frac{7\sqrt{2}}{2} \approx 4.95$
$d < r = 5$ → 두 점에서 만남 (간신히!)
Q11. 원 $x^2+y^2=10$ 위의 점 $(1, 3)$에서의 접선의 방정식은?
탭하여 정답 확인
$x_1 x + y_1 y = r^2$ → $1 \cdot x + 3 \cdot y = 10$ → $x+3y=10$
검증: 접점 $(1,3)$은 원 위($1+9=10$ ✓), 접선 위($1+9=10$ ✓)
Q12. $A(0,0)$과 $B(8,0)$에서 같은 거리에 있는 점의 자취 방정식을 구하라.
탭하여 정답 확인
$PA = PB$: $\sqrt{x^2+y^2} = \sqrt{(x-8)^2+y^2}$
양변 제곱: $x^2 = x^2-16x+64$
$16x = 64$ → $x = 4$
자취: 직선 $x = 4$ (AB의 수직이등분선)
Q13. 두 평행한 직선 $3x+4y-1=0$과 $3x+4y+9=0$ 사이의 거리는?
탭하여 정답 확인
$d = \frac{|-1-9|}{\sqrt{9+16}} = \frac{10}{5} = 2$
Q14. $A(0,0)$, $B(10,0)$에서 $PA:PB = 1:3$인 점의 자취를 구하라.
탭하여 정답 확인
$3PA = PB$ → $9PA^2 = PB^2$:
$9(x^2+y^2) = (x-10)^2+y^2$
$9x^2+9y^2 = x^2-20x+100+y^2$
$8x^2+8y^2+20x-100=0$
$x^2+y^2+\frac{5}{2}x-\frac{25}{2}=0$
$\left(x+\frac{5}{4}\right)^2+y^2 = \frac{25}{16}+\frac{25}{2} = \frac{225}{16}$

아폴로니우스의 원: 중심 $\left(-\frac{5}{4}, 0\right)$, 반지름 $\frac{15}{4}$
Q15. 두 직선 $y=2x$와 $y=\frac{1}{2}x$가 이루는 예각 $\theta$에 대해 $\tan\theta$를 구하라.
탭하여 정답 확인
$\tan\theta = \left|\frac{m_1-m_2}{1+m_1 m_2}\right| = \left|\frac{2-\frac{1}{2}}{1+2 \cdot \frac{1}{2}}\right| = \left|\frac{\frac{3}{2}}{2}\right| = \frac{3}{4}$
