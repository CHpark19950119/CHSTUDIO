📖 이론 — 띄어쓰기 총론 & 관형사 vs 접두사

1. 띄어쓰기 총론 (한글 맞춤법 제5장)

  • 제41항: 조사는 그 앞말에 붙여 쓴다. — 꽃이, 꽃마저, 꽃밖에, 꽃으로만
  • 제42항: 의존명사는 띄어 쓴다. — 아는 것이 힘이다. 먹을 만큼 먹어라.
  • 제43항: 단위를 나타내는 명사는 띄어 쓴다. — 한 개, 차 한 대, 연필 한 자루
  • 제44항: 수를 적을 때는 '만(萬)' 단위로 띄어 쓴다. — 십이억 삼천사백오십육만 칠천팔백구십팔
  • 제45항: 두 말을 이어 주거나 열거할 때 쓰이는 말은 띄어 쓴다. — 국장 겸 과장
  • 제46항: 단음절로 된 단어가 연이어 나타날 때는 붙여 쓸 수 있다. — 좀더 큰것
  • 제47항: 보조용언은 띄어 씀을 원칙으로 하되, 경우에 따라 붙여 씀도 허용한다.
  • 제48항: 성과 이름, 성과 호 등은 붙여 쓰고, 이에 덧붙는 호칭어·관직명은 띄어 쓴다.
  • 제49항: 성명 이외의 고유명사는 단어별로 띄어 씀을 원칙으로 하되, 단위별로 붙여 쓸 수 있다.
  • 제50항: 전문용어는 단어별로 띄어 씀을 원칙으로 하되, 붙여 쓸 수 있다.

→ 핵심 원리: "단어와 단어 사이는 띄어 쓴다"가 대원칙. 조사만 예외적으로 앞말에 붙인다.

2. 관형사의 정의

관형사(冠形詞): 체언(명사, 대명사, 수사) 앞에서 그 체언의 내용을 꾸며 주는 독립된 품사. 활용하지 않고, 조사가 붙지 않으며, 반드시 뒤에 체언이 온다. 독립된 단어이므로 뒤의 명사와 띄어 쓴다.

4. 접두사의 정의

접두사(接頭辭): 단어의 앞에 붙어 그 뜻을 제한하거나 새로운 뜻을 더해 주는 접사. 독립된 단어가 아니라 어근에 결합하는 형태소이므로 뒤의 어근에 붙여 쓴다.

6. 관형사 vs 접두사 판별 플로우차트

해당 표현을 국어사전에서 검색한다 │ ├─ 사전에 한 단어로 등재? ─── YES → 붙여 쓴다 (합성어/파생어) │ 예: 새해, 풋사과, 한여름, 첫사랑, 맨손 │ └─ 사전에 없다 ─── NO │ ├─ 앞 형태소가 홀로 쓰일 수 있는가? │ │ │ ├─ YES → 관형사 (띄어 쓴다) │ │ 예: 새 학기, 헌 옷, 첫 만남, 온 세상 │ │ │ └─ NO → 접두사 (붙여 쓴다) │ 예: 풋사과(풋- 홀로 못 씀), 군소리(군- 홀로 못 씀) │ └─ 의미 확인 ├─ '새로운' 뜻 → 관형사 '새' → 띄어 쓴다 (새 옷) ├─ '갓, 처음' 뜻 → 접두사 '새-' → 붙여 쓴다 (새해) └─ 확실치 않으면 → 사전 확인이 최종 기준

② 새해 복 많이 받으세요. — '새해'는 사전 등재 합성어, 붙여 쓴다.

⑥ 한겨울에도 뛰었다. — '한겨울'은 사전 등재 합성어, 붙여 쓴다.
⑦ 첫 출근은 떨린다. — '첫'은 관형사('처음의'), 띄어 쓴다.
⑧ 첫사랑을 잊지 못한다. — '첫사랑'은 사전 등재 합성어, 붙여 쓴다.
⑨ 풋사과는 시다. — '풋-'은 접두사, 붙여 쓴다.
⑩ 군소리 말고 일해라. — '군-'은 접두사('쓸데없는'), 붙여 쓴다.
⑪ 맨 처음부터 다시 하자. — '맨'은 관형사('가장'), 띄어 쓴다.
⑫ 맨손으로 잡았다. — '맨손'은 사전 등재 합성어, 붙여 쓴다.

핵심 규칙

  • 관형사: 독립된 단어로서 뒤의 명사를 꾸밈 → 새 학기, 온 세상, 첫 만남
  • 접두사: 뒤의 어근에 붙어 하나의 단어를 형성 → 새해, 온종일, 풋사랑
  • 판별법: 사전에 한 단어로 등재되어 있으면 붙여 씀. '새'가 '새로운'의 뜻이면 관형사(띄어쓰기), '갓~'의 뜻이면 접두사(붙여쓰기)
  • 한: 관형사 '한 사람'(어떤 한) vs 접두사 '한여름'(가운데)
  • 첫: 관형사 '첫 만남' vs 접두사 '첫사랑'(사전 등재 합성어)
A-01
다음 중 바르게 띄어 쓴 것은?
▶
정답: ①
'새'는 여기서 관형사로 '새로운'이라는 뜻이다. 관형사는 독립된 단어이므로 뒤의 명사 '학기'와 띄어 쓴다. 반면 '새해'에서의 '새-'는 접두사로 사전에 한 단어로 등재되어 있으므로 붙여 쓴다. 핵심: 국어사전에 한 단어로 올라 있는지 확인하는 것이 가장 확실한 방법이다.
A-02
다음 중 바르게 쓴 것은?
▶
정답: ①

1. 의존명사의 정의

의존명사(依存名詞): 자립성이 없어 홀로 쓰이지 못하고, 반드시 그 앞에 관형어(관형사형 어미 '-ㄴ/-는/-ㄹ/-던' 등)가 와야 쓸 수 있는 명사. 형식명사, 불완전명사라고도 한다. 명사이므로 앞말과 반드시 띄어 쓴다(한글 맞춤법 제42항).

  • 주의: 어미 '-ㄴ데', '-ㄹ걸', '-ㄴ바' 등은 붙여 씀 (의존명사 아님)

Section C · 보조용언 띄어쓰기
15문제 · 원칙은 띄어, 허용은 붙여
📖 이론 — 보조용언의 정의와 띄어쓰기 규칙

1. 보조용언의 정의

보조용언(補助用言): 홀로 쓰이지 못하고 본용언 뒤에서 그 뜻을 보충하는 용언. 보조동사와 보조형용사로 나뉜다. 본용언과 보조용언 사이에는 연결어미(-아/어, -고, -게, -지)가 온다.

2. 한글 맞춤법 제47항

  • 원칙: 보조용언은 띄어 쓴다.
  • 허용: -아/어 뒤에서는 붙여 쓰는 것도 허용한다.
  • 필수 띄기: -고, -게, -지 뒤에서는 반드시 띄어 쓴다 (붙여 쓰기 불허).
본용언 + 연결어미 + 보조용언 │ ├─ -아/어 연결 → 띄어쓰기 원칙, 붙여쓰기 허용 │ 예: 먹어 보다 (원칙) = 먹어보다 (허용) ← 둘 다 맞음 │ ├─ -고 연결 → 반드시 띄어 쓴다 │ 예: 먹고 싶다 (○) / 먹고싶다 (×) │ ├─ -게 연결 → 반드시 띄어 쓴다 │ 예: 되게 되다 → 알게 되다 (○) / 알게되다 (×) │ └─ -지 연결 → 반드시 띄어 쓴다 예: 가지 않다 (○) / 가지않다 (×)

4. 합성동사 vs 보조용언 구분

  • 합성동사: 두 동사가 결합해 새로운 단일 의미를 형성 → 사전에 한 단어로 등재 → 붙여 쓴다
  • 예: 들어가다(진입), 뛰어넘다(극복), 물러나다(후퇴)
  • 보조용언: 본용언의 의미를 보충 → 사전에 별도 → 띄어 쓴다(원칙)
  • 예: 들어 보다(들어서 시도), 뛰어 가다(뛰면서 가다)
  • 판별법: 사전에 한 단어인지 확인 + "본래 뜻이 살아 있는가?"를 기준으로 판단

핵심 규칙

  • 원칙: 본용언과 보조용언은 띄어 쓴다 — 먹어 보다, 해 주다
  • 허용: 붙여 쓰는 것도 인정 — 먹어보다, 해주다
  • 단, -아/어 연결에서만 허용. -고, -게, -지 뒤는 반드시 띄어 쓴다
  • 주요 보조용언: ~아/어 보다, ~아/어 주다, ~아/어 버리다, ~아/어 놓다, ~아/어 두다, ~아/어 가다/오다, ~고 싶다, ~고 있다, ~게 되다, ~지 않다, ~지 못하다
  • 합성동사 주의: '들어가다'(진입)는 합성동사(한 단어) ≠ '해 가다'(보조용언)

정답: ②
'되다'는 보조용언이다. -게 연결 뒤이므로 반드시 띄어 쓴다. 좋게 되다, 알게 되다, 하게 되다 등. -게 연결은 -고, -지와 마찬가지로 붙여쓰기 불가.

▶
정답: ③ (둘 다 맞음)
'가다'는 보조용언으로 '진행, 지속'을 나타낸다. -아/어 연결이므로 띄어쓰기(원칙)와 붙여쓰기(허용) 모두 가능하다. '있다'는 -고 뒤 보조용언이므로 반드시 띄어 쓴다: 가고 있다.

C-09

▶

▶
정답: ①

Section D · 합성어·파생어 판별
15문제 · 사전에 한 단어로 등재된 합성어는 붙여 쓴다
📖 이론 — 합성어·파생어의 정의와 판별

1. 단어의 구성

  • 단일어: 하나의 어근으로 된 단어 — 하늘, 바다, 사람
  • 복합어: 둘 이상의 형태소로 된 단어 — 합성어 + 파생어

2. 합성어의 정의

합성어(合成語): 어근 + 어근이 결합하여 하나의 새로운 단어가 된 것. 사전에 한 단어로 등재되어 있으며, 붙여 쓴다.

  • 두 어근이 합쳐져 원래 뜻과 다른 새 의미를 만드는 경우가 많다.
  • 예: '눈'(eye) + '물'(water) → '눈물'(tears) — 단순 합이 아닌 새 의미

3. 합성어 판별 기준

  1. 사전 등재 여부: 국어사전에 한 단어로 올라 있으면 합성어 → 붙여 쓴다. 이것이 가장 확실한 기준.
  2. 의미 변화: 두 단어의 합이 새로운 의미를 형성하면 합성어일 가능성이 높다.
  3. 사잇소리 현상: 합성어에서 사이시옷, 된소리 등이 나타나면 합성어의 증거.
  4. 결합 빈도: 늘 결합해서 쓰이고 그 사이에 다른 단어를 끼워 넣기 어려우면 합성어.

4. 합성어 유형 분류

5. 파생어의 정의

파생어(派生語): 접사(접두사/접미사) + 어근이 결합하여 새 단어가 된 것. 접사는 독립된 단어가 아니므로 붙여 쓴다.

8. 합성어 vs 구(句) — 붙여 쓰기 vs 띄어 쓰기

붙여 쓰는 합성어: 비바람, 돌다리, 눈물, 손수건, 첫사랑, 한여름, 들판, 새해
띄어 쓰는 구: 큰 집, 새 옷, 첫 만남, 좋은 사람, 높은 산, 맑은 물, 손 글씨

→ 최종 기준은 항상 국어사전 등재 여부. 자신이 없으면 사전을 찾는 것이 정답이다.

핵심 규칙

  • 합성어: 둘 이상의 어근이 결합하여 새로운 의미의 한 단어가 됨 → 붙여 쓴다
  • 구(句): 두 단어가 의미적으로 연결되지만 각각 독립 → 띄어 쓴다
  • 판별법: 국어사전에 한 단어로 올라 있으면 합성어(붙여쓰기), 없으면 구(띄어쓰기)
  • 예: 눈물(합성어, 눈+물) vs 눈 위(구), 손수건(합성어) vs 손 글씨(구)

Section E · 실전 문장 교정 — 띄어쓰기
10문제 · 문장에서 띄어쓰기 오류를 모두 찾아 교정하시오
E-01 그는 할수있는것이 없다고 생각하지만, 사실 할수있는일이 많다.
교정:

그는 할 수 있는 것이 없다고 생각하지만, 사실 할 수 있는 일이 많다.

'수'는 의존명사 → 띄어쓰기. '것'도 의존명사 → 띄어쓰기. '있는'은 관형형으로 '것/일' 앞에서 띄어 쓴다. 총 4곳 수정.
E-02 오래전에 만든것인데, 먹어봐도 괜찮을지 모르겠다.
교정:

오래전에 만든 것인데, 먹어 봐도 괜찮을지 모르겠다.

'것'은 의존명사 → '만든 것'. '보다'는 보조용언이고 -아/어 연결이므로 '먹어 봐도'(원칙)와 '먹어봐도'(허용) 모두 가능하나, '만든것'은 반드시 '만든 것'으로 띄어 써야 한다.
E-03 비가 온지 한참됐는데 아직도 도로가 젖어있다.
교정:

비가 온 지 한참 됐는데 아직도 도로가 젖어 있다.

'지'는 시간 경과를 나타내는 의존명사 → '온 지'. '있다'는 -아/어 연결의 보조용언 → '젖어 있다'(원칙) / '젖어있다'(허용).
E-04 시키는대로만 하면 되는데 왜 맨날 딴짓을 하고싶어하는지 모르겠다.
교정:

시키는 대로만 하면 되는데 왜 맨날 딴짓을 하고 싶어하는지 모르겠다.

'대로'는 관형어 뒤의 의존명사 → '시키는 대로'. '싶다'는 -고 뒤의 보조용언 → '하고 싶어하는지'(반드시 띄어쓰기).
E-05 새학기가 시작된지 얼마안됐는데 벌써 지칠데까지 지쳤다.
교정:

새 학기가 시작된 지 얼마 안 됐는데 벌써 지칠 데까지 지쳤다.

'새'는 관형사 → '새 학기'. '지'는 의존명사(시간 경과) → '시작된 지'. '안'은 부사 → '얼마 안 됐는데'. '데'는 의존명사(곳, 경우) → '지칠 데까지'.
E-06 그럴리가 없다고 생각했지만 들은바에 따르면 사실인것같다.
교정:

그럴 리가 없다고 생각했지만 들은 바에 따르면 사실인 것 같다.

'리'는 의존명사(까닭) → '그럴 리'. '바'는 의존명사(사실) → '들은 바'. '것'은 의존명사 → '사실인 것'. '같다'는 보조형용사 → '것 같다'.
E-07 맨발로 뛰어다니던 풋사랑의 기억이 온종일 머릿속에서 떠나지않는다.
교정:

맨발로 뛰어다니던 풋사랑의 기억이 온종일 머릿속에서 떠나지 않는다.

'맨발', '풋사랑', '온종일', '머릿속'은 모두 사전 등재 합성어로 붙여쓰기가 맞다. 유일한 오류는 '떠나지않는다' → '떠나지 않는다'. '않다'는 -지 뒤의 보조용언으로 반드시 띄어 써야 한다.
E-08 한참을 걸은뒤에야 비로소 내가갈곳을 정할수있었다.
교정:

한참을 걸은 뒤에야 비로소 내가 갈 곳을 정할 수 있었다.

'뒤'는 의존명사 → '걸은 뒤'. '곳'은 의존명사(장소) → '갈 곳'. '수'는 의존명사 → '정할 수 있었다'. '내가갈곳' → '내가 갈 곳'.
E-09 이왕 한거 끝까지 해볼만한 가치가 있다고 본다.
교정:

이왕 한 거 끝까지 해 볼 만한 가치가 있다고 본다.

'거'는 '것'의 구어형, 의존명사 → '한 거'. '볼'은 보조용언 '보다'의 관형형 → '해 볼'(원칙). '만하다'는 보조형용사 → '해 볼 만한'. 다만 -아/어 연결이므로 '해볼'도 허용.
E-10 웃을뿐 대답하지못한채 그자리에 서있었다.
교정:

웃을 뿐 대답하지 못한 채 그 자리에 서 있었다.

'뿐'은 의존명사 → '웃을 뿐'. '못하다'는 -지 뒤 보조용언 → '대답하지 못한'. '채'는 의존명사 → '못한 채'. '있다'는 -아/어 뒤 보조용언 → '서 있었다'(원칙). '그 자리'는 관형사+명사 → 띄어쓰기.

Section F · 사이시옷 규칙

📖 이론 — 사이시옷 규칙 전문 (한글 맞춤법 제30항)

1. 한글 맞춤법 제30항 전문

사이시옷은 다음과 같은 경우에 받치어 적는다.

  1. 순우리말로 된 합성어로서 앞말이 모음으로 끝난 경우
    • (가) 뒷말의 첫소리가 된소리로 나는 것
    • (나) 뒷말의 첫소리 'ㄴ, ㅁ' 앞에서 'ㄴ' 소리가 덧나는 것
    • (다) 뒷말의 첫소리 모음 앞에서 'ㄴㄴ' 소리가 덧나는 것
  2. 순우리말과 한자어로 된 합성어로서 앞말이 모음으로 끝난 경우 — 위 (가)(나)(다)와 같음
  3. 두 음절로 된 다음 한자어: 곳간(庫間), 셋방(貰房), 숫자(數字), 찻간(車間), 툇간(退間), 횟수(回數)

2. 6가지 조건 상세 설명

조건설명예시발음
조건 ① 순우리말 또는 순우리말+한자어 합성어일 것 나무+잎, 비+물, 귀+속(屬) —
조건 ② 앞말이 모음으로 끝날 것 나무(○), 밤(×, 자음 끝) —
조건 ③-가 뒷말 첫소리가 된소리로 발음 나뭇잎, 냇가, 잇몸 [나문닙], [내:까], [인몸]
조건 ③-나 뒷말 첫소리 'ㄴ, ㅁ' 앞에서 'ㄴ' 덧남 아랫니, 잇몸, 깻묵 [아랜니], [인몸], [깬묵]
조건 ③-다 뒷말 첫소리 모음 앞에서 'ㄴㄴ' 덧남 뒷일, 나뭇잎, 베갯잇 [뒨닐], [나문닙], [베겐닏]
조건 ④ 한자어+한자어는 원칙적으로 사이시옷 안 씀 초점(焦點)(○), 찻점(×) —

3. 사이시옷 적용 예시 모음

된소리 발음 (③-가)
· 나무+잎 → 나뭇잎 [나문닙]  / 내+가 → 냇가 [내:까]
· 비+물 → 빗물 [빈물]  / 기+발 → 깃발 [기빨]
· 해+빛 → 햇빛 [해삗]  / 아래+도리 → 아랫도리 [아래또리]

'ㄴ' 덧남 (③-나)
· 아래+니 → 아랫니 [아랜니]  / 이+몸 → 잇몸 [인몸]
· 깨+묵 → 깻묵 [깬묵]  / 나무+잎 → 나뭇잎 [나문닙]

'ㄴㄴ' 덧남 (③-다)
· 뒤+일 → 뒷일 [뒨닐]  / 위+일 → 윗일 [윈닐]
· 예사+일 → 예삿일 [예산닐]

4. 한자어+한자어 예외 6개

원칙적으로 한자어끼리 결합하면 사이시옷을 쓰지 않지만, 다음 6개만 예외로 사이시옷을 인정한다:

단어한자발음뜻
곳간庫間[곧깐]곡식 저장소
셋방貰房[세빵]세를 놓는 방
숫자數字[숟짜]수를 나타내는 글자
찻간車間[차깐]기차 칸
툇간退間[퇴깐]툇마루가 있는 칸
횟수回數[회쑤]되풀이한 수

※ '숫자'와 '횟수'가 시험에 가장 자주 출제된다.

5. 외래어에 사이시옷을 쓰지 않는 이유

한글 맞춤법 제30항의 적용 대상은 "순우리말" 또는 "순우리말+한자어" 합성어로 한정된다. 외래어는 이 범위에 포함되지 않으므로 사이시옷을 쓰지 않는다.

피자집 (○) — 피잣집 (×)
커피잔 (○) — 커핏잔 (×)
택시비 (○) — 택싯비 (×)
소시지국 (○) — 소시짓국 (×)

6. 사이시옷 판별 플로우차트

합성어인가? │ ├─ NO → 사이시옷 해당 없음 │ └─ YES → 앞말이 모음으로 끝나는가? │ ├─ NO → 사이시옷 안 씀 (예: 밤+길 → 밤길) │ └─ YES → 어종(語種) 확인 │ ├─ 순우리말+순우리말 → 조건 ③ 확인 → 사이시옷 적용 ├─ 순우리말+한자어 → 조건 ③ 확인 → 사이시옷 적용 ├─ 한자어+한자어 → 원칙 불가 (예외 6개만 적용) └─ 외래어 포함 → 사이시옷 안 씀

사이시옷 적용 조건 (모두 충족 시)

  • 조건 1: 순우리말 합성어이거나, 순우리말+한자어 합성어일 것
  • 조건 2: 앞말이 모음으로 끝날 것
  • 조건 3: 다음 중 하나에 해당:
    ① 뒷말 첫소리가 된소리로 발음 (나뭇잎 → [나문닙])
    ② 뒷말 첫소리 'ㄴ, ㅁ' 앞에서 'ㄴ' 소리가 덧남 (아랫니 → [아랜니])
    ③ 뒷말 첫소리 모음 앞에서 'ㄴㄴ' 소리가 덧남 (뒷일 → [뒨닐])
  • 한자어+한자어 예외 6개: 곳간(庫間), 셋방(貰房), 숫자(數字), 찻간(車間), 툇간(退間), 횟수(回數)
  • 외래어 합성어에는 사이시옷 안 씀: 피자집(○), 피잣집(×)

Section G · 된소리·경음화
10문제 · 발음은 된소리지만 표기는 예사소리
📖 이론 — 된소리(경음화) 규칙과 표준 발음법

1. 된소리(경음)란?

된소리: ㄲ, ㄸ, ㅃ, ㅆ, ㅉ의 다섯 가지. 성문(聲門)을 긴장시켜 발음하는 소리로, 예사소리(ㄱ, ㄷ, ㅂ, ㅅ, ㅈ)보다 세고 단단하게 난다.

경음화(硬音化): 예사소리가 특정 환경에서 된소리로 발음되는 현상. 발음은 된소리지만 표기는 원래 형태를 밝혀 적는다(한글 맞춤법 제1항: 표음주의 + 형태주의 절충).

2. 표준 발음법 관련 조항

제23항 — 받침 'ㄱ(ㄲ, ㅋ, ㄳ, ㄺ), ㄷ(ㅅ, ㅆ, ㅈ, ㅊ, ㅌ), ㅂ(ㅍ, ㄼ, ㄿ, ㅄ)' 뒤에 연결되는 'ㄱ, ㄷ, ㅂ, ㅅ, ㅈ'은 된소리로 발음한다.

· 국밥 → [국빱] (표기: 국밥 ○, 국빱 ×)
· 깎다 → [깍따]  / 넋과 → [넉꽈]
· 삯돈 → [삭똔]  / 볶다 → [복따]
· 닭장 → [닥짱]  / 칼국수 → [칼국쑤]

제24항 — 어간 받침 'ㄴ(ㄵ), ㅁ(ㄻ)' 뒤에 연결되는 어미의 첫소리 'ㄱ, ㄷ, ㅅ, ㅈ'은 된소리로 발음한다.

· 신고 → [신꼬]  / 껴안다 → [껴안따]
· 삼고 → [삼꼬]  / 젊지 → [점찌]

제25항 — 어간 받침 'ㄼ, ㄾ' 뒤에 연결되는 어미의 첫소리 'ㄱ, ㄷ, ㅅ, ㅈ'은 된소리로 발음한다.

· 넓게 → [널께]  / 핥다 → [할따]

제26항 — 한자어에서 'ㄹ' 받침 뒤에 연결되는 'ㄷ, ㅅ, ㅈ'은 된소리로 발음한다.

· 발달 → [발딸]  / 절도 → [절또]
· 말살 → [말쌀]  / 불세출 → [불쎄출]
· 물질 → [물찔]  / 발전 → [발쩐]

제27항 — 관형사형 '-(으)ㄹ' 뒤에 연결되는 'ㄱ, ㄷ, ㅂ, ㅅ, ㅈ'은 된소리로 발음한다.

· 할 것을 → [할 꺼슬]  / 갈 데가 → [갈 떼가]
· 할 바를 → [할 빠를]  / 할 수는 → [할 쑤는]
· 할 적에 → [할 쩌게]

3. 핵심 원리: 발음 ≠ 표기

  • 한글 맞춤법 제1항: "한글 맞춤법은 표준어를 소리대로 적되, 어법에 맞도록 함을 원칙으로 한다."
  • 된소리로 발음되더라도 원래 형태소를 밝혀 적는다.
  • 예: '국+밥' → [국빱]이지만 표기는 국밥. '눈+동자' → [눈똥자]이지만 표기는 눈동자.
  • 예외: 합성어에서 된소리로 굳어진 일부는 된소리로 표기 — 해쌀(×) → 햅쌀(○), 찹쌀(○)

📖 이론 — 혼동 단어 쌍별 상세 설명

1. 왠지 vs 웬

  • 왠지: '왜인지'의 준말. 부사. 이 형태만 표준어이다. "왠지 오늘 기분이 좋다."
  • 웬: 관형사. '어찌 된', '뜻밖의'라는 뜻. "웬일이야?", "웬 떡이냐."
  • '웬지'는 없는 말이다. '왠지'와 '웬'을 혼동하여 만든 비표준 형태.
· 왠지 모르게 불안하다. (○)
· 웬 비가 이렇게 오냐. (○ — '어찌 된')
· 웬지 모르게 불안하다. (× — '웬지'는 없는 말)

2. -로서 vs -로써

  • -로서: 자격·지위·신분을 나타냄. "~으로서" = "~의 자격/입장에서"
  • -로써: 수단·도구·재료를 나타냄. "~으로써" = "~을 가지고/이용하여"
· 학생으로서 본분을 다해라. (자격)
· 친구로서 충고한다. (자격)
· 나무로써 집을 지었다. (재료)
· 대화로써 문제를 해결했다. (수단)
· 판별법: "~의 자격으로" 대입 가능 → -로서, "~을 이용하여" 대입 가능 → -로써

3. -데 vs -대

  • -데: 자기가 직접 경험한 것을 회상하여 말할 때. 어미. "그 사람 키가 크데." (내가 봤더니)
  • -대: 남의 말을 전달(간접 인용)할 때. '-다고 해'의 준말. "그 사람 키가 크대." (누가 그러더라)
· 거기 음식이 맛있데. (내가 직접 먹어 봤더니)
· 거기 음식이 맛있대. (누군가가 맛있다고 하더라)
· 판별법: "내가 경험" → -데, "남이 말해줌" → -대

4. 됐 vs 됬

  • 됐: '되었'의 준말. '되-' + '-었-' → '됐'. 이것만 올바른 표기.
  • 됬: 비표준. '되-'에 '-엇-'이 결합하는 활용은 존재하지 않는다.
  • 같은 원리: '하였' → '했' (○), '하엿' → '핬' (×)
· 준비가 됐다. (○)  / 준비가 됬다. (×)
· 됐어, 이제 그만. (○)  / 됬어. (×)

5. 맞추다 vs 맞히다

  • 맞추다: ① 서로 비교하다 ("답을 맞추어 보다") ② 조립하다 ("퍼즐을 맞추다") ③ 맞게 하다 ("양복을 맞추다")
  • 맞히다: ① 정답을 맞히다 = 적중시키다 ("정답을 맞히다") ② 명중시키다 ("과녁을 맞히다")
  • '맞히다'는 '맞다'의 사동형(맞+히+다). 무언가를 맞게 만드는 것.
· 정답을 맞히다. (적중)  / 시간을 맞추다. (조절)
· 양복을 맞추다. (주문 제작)  / 과녁을 맞히다. (명중)
· 판별법: "적중/명중" → 맞히다, "비교/조립/조절" → 맞추다

6. 부치다 vs 붙이다

  • 부치다: ① 편지를 부치다 (발송) ② 전을 부치다 (요리) ③ 안건을 회의에 부치다 ④ 논을 부치다 (경작) ⑤ 힘에 부치다 (감당 못함)
  • 붙이다: ① 풀로 붙이다 (접착) ② 이름을 붙이다 (명명) ③ 불을 붙이다 (점화) ④ 조건을 붙이다 (부가) ⑤ 흥정을 붙이다
· 편지를 부치다. (발송)  / 우표를 붙이다. (접착)
· 전을 부치다. (요리)  / 별명을 붙이다. (명명)
· 판별법: '붙이다'는 '붙다'의 사동형(붙+이+다) — 물리적/추상적 접착. '부치다'는 별개의 동사 — 발송, 요리 등.

7. -던 vs -든

  • -던: 과거 회상의 어미. 지난 일을 떠올림. "내가 다니던 학교", "먹던 빵"
  • -든(지): 선택·상관없음의 어미. "가든 말든 네 맘이다", "뭐든 좋다"
· 내가 살던 집. (과거 회상 — 예전에 살았던)
· 어디든 좋다. (선택 — 어디라도)
· 판별법: "과거 경험 회상" → -던, "아무거나/상관없음" → -든

8. 안 vs 않

  • 안: 부사. 용언 앞에 놓여 부정을 나타냄. "안 가다", "안 먹다"
  • 않다: 보조용언. '-지 않다' 구성에서 쓰임. "가지 않다", "먹지 않다"
  • 둘 다 부정 표현이지만, 문장 내 위치와 문법적 역할이 다르다.
· 나는 안 간다. (부사 — 용언 앞에 독립적으로)
· 나는 가지 않는다. (보조용언 — '-지' 뒤에)
· 판별법: '~지' 뒤에 온다 → 않다, 용언 바로 앞에 독립적으로 온다 → 안
· 확인법: "않"을 빼고 "~지 __ 다" → 문장 성립 안 되면 '않' 필요. "안"을 빼면 긍정문 됨.

핵심 구분법 요약

  • 왠지 vs 웬지: '왠지'(= 왜인지)만 표준어. '웬'은 '어찌 된, 뜻밖의' (웬일이야)
  • -로서 vs -로써: 자격·지위·신분 → '-로서', 수단·도구·재료 → '-로써'
  • 데 vs 대: '데'(장소/경우, 의존명사), '대'(간접 인용 '-다고 해'의 준말)
  • 됬 vs 됐: '되었'의 준말 = '됐'. '됬'은 비표준
  • 않 vs 안: '않다'(~지 않다, 보조용언), '안'(부사, 안 가다)
  • 맞추다 vs 맞히다: '맞추다'(비교·조립), '맞히다'(정답을 맞히다, 명중)
  • 부치다 vs 붙이다: '부치다'(편지, 전 부침), '붙이다'(접착, 이름 붙이다)
H-01
"( ) 오늘따라 기분이 우울하다."
▶
정답: ①
왠지 = '왜인지'의 준말. '이유는 모르겠지만'의 뜻으로 부사로 쓰인다. 웬은 관형사로 '어찌 된, 뜻밖의'라는 뜻 → "웬일이야", "웬 떡이야". '웬지'라는 단어는 존재하지 않는다.
H-02
"학생( ) 본분을 다해야 한다."
▶
정답: ②
-로서: 자격, 지위, 신분을 나타낸다. "학생으로서", "친구로서", "부모로서". -로써: 수단, 도구, 재료를 나타낸다. "칼로써 잘랐다", "노력으로써 이겼다". 구분법: "~의 자격/입장에서" 바꿔 말할 수 있으면 '-로서'.
H-03
"노력으( ) 성공을 이룰 수 있다."
▶
정답: ①
'노력'은 성공을 이루는 수단이다. 수단·도구 → '-로써'. "노력으로써 성공을 이룰 수 있다."
H-04
"아직 준비가 안 ( )."
▶
정답: ②
'되었다' → 준말 됐다. '됬다'는 비표준 표기이다. 'ㅚ+ㅆ' = '됐'이 올바르다. 확인법: '되었'을 줄이면 '됐'(ㅚ+ㅆ). 'ㅞ+ㅆ' = '됬'은 잘못된 조합.
H-05
"시험 문제를 모두 ( )."
▶
정답: ①
맞히다: 정답을 알아내다, 과녁에 명중시키다. 시험 문제의 정답을 알아낸 것이므로 '맞히다'가 맞다. '맞혔다' = '맞히+었+다'. 맞추다: 서로 비교하다(답을 맞추다=비교), 조립하다(퍼즐을 맞추다), 조정하다(시간을 맞추다). 구분: '적중' → 맞히다, '비교·조립' → 맞추다.
H-06
"가구를 벽에 ( ) 놓았다."
▶
정답: ②
붙이다: 접착하다, 밀착시키다. 가구를 벽에 밀착시킨 것이므로 '붙이다' → '붙여'. 부치다: ① 편지를 부치다, ② 전을 부치다, ③ 부쳐(=보내다). 구분: 접착/밀착 → 붙이다, 발송/부침 → 부치다.
H-07
"어제 편지를 ( )."
▶
정답: ①
편지를 보내는 것은 '부치다'이다. '부치다'의 활용: 부치+었+다 = 부쳤다.
H-08
"그 사람이 온( ) 말을 들었다."
▶
정답: ②
'-대': '-다고 해'의 준말, 간접 인용. "온대" = "온다고 해". '-데': ① 의존명사(장소/경우) "갈 데가 없다", ② 어미(회상) "어제 갔는데 좋더라". 이 문장에서 '온다고 들었다'의 뜻이므로 간접 인용 '-대'를 쓴다.
H-09
"이 영화 재미있( )."
▶
정답: ①
'-던': 과거 회상을 나타내는 관형사형 어미. "내가 봤는데 재미있던데" = 과거 경험을 회상. '-든': 선택·양보를 나타내는 어미. "가든 말든", "무엇이든". 이 문장은 과거 경험을 회상하므로 '-던'이 맞다.
H-10
"가( ) 말( ) 네가 결정해."
▶
정답: ②
'-든': 선택·양보. "가든 말든" = "가거나 말거나". 두 가지 선택지를 나열하는 것이므로 '-든'이 맞다. "가던 말던"(×).
H-11
"오늘 밥을 ( ) 먹었다."
▶
정답: ①
'안': 부사. 용언 앞에 놓여 부정을 나타냄. "안 먹다", "안 가다". '않다': 보조용언. '-지 않다' 꼴. "먹지 않다", "가지 않다". 이 문장에서는 '먹다' 앞에 부사 '안'을 씀. 구분법: '안'은 부사(용언 앞), '않'은 보조용언('-지 않-' 꼴).
H-12
"이것은 책( )."
▶
정답: ①
'-이에요'/'-예요' 구분: 받침 있는 체언 뒤 → '-이에요', 받침 없는 체언 뒤 → '-예요'. '책'은 받침 'ㄱ'이 있으므로 '-이에요'가 맞다. "책이에요"(○), "책예요"(×). 반대로 "사과예요"(○, 받침 없음).
H-13
"이 문제를 ( ) 풀어야 한다."
▶
정답: ①
반드시: 부사 '틀림없이, 꼭'. 반듯이: '반듯하다'(바르고 곧다)의 부사형. "반듯이 누워라" = 자세가 반듯하게. "반드시 풀어야 한다" = 꼭 풀어야 한다.
H-14
"시간이 많이 ( )."
▶
정답: ①
'흐르다' → 불규칙 활용. '흐르+었+다' → '흘렀다'. 'ㄹ' 불규칙: '르' 앞의 모음이 탈락하고 'ㄹ'이 겹침. '흘렸다'(×) — 이 형태는 '흘리다'(눈물을 흘리다)의 과거형이므로 뜻이 다르다. 시간이 경과하는 것은 '흐르다' → '흘렀다'.
H-15
"어머니( ) 요리를 잘하신다."
▶
정답: ①
'-께서': 주격 조사의 높임말. 어른이나 윗사람에게 쓴다. '-게서'는 비표준이다. "어머니께서", "선생님께서", "할아버지께서" 등.
H-16
"오늘 학교에서 ( ) 일이 있었어?"
▶
정답: ②
'웬': 관형사 '어찌 된, 뜻밖의'. "웬일이야", "웬 떡이야", "웬 일이 있었어". '왠': '왜인지'의 준말이지만 오직 '왠지'에서만 쓰임. 관형사로는 '웬'만 사용. "웬 일" (○), "왠 일" (×).
H-17
"낮( ) 밤( ) 공부만 했다."
▶
정답: ①
'-이든': 선택·양보의 조사. "낮이든 밤이든" = 낮이거나 밤이거나(어느 것이든). '-이던': 과거 회상 + '이다'. "학생이던 시절" = 학생이었던 시절. 선택/양보 → '-든', 회상 → '-던'.
H-18
"할머니가 전을 ( )."
▶
정답: ②
'전을 부치다'가 맞다. '부치다'의 뜻 중 하나가 '번철에 기름을 두르고 전을 익히다'이다. '전을 붙이다'(×). 편지도 '부치다', 전도 '부치다'.
H-19
"그 일은 내가 할 수 있는 일이 ( )."
▶
정답: ①
'아니다'는 형용사이다. 형용사는 '-ㄴ다' 활용을 하지 않는다. '아니다'의 활용: 아니+다 = 아니다, 아니+ㄴ = 아닌, 아니+어서 = 아니어서. '아닌다'는 비표준.
H-20
"( ) 너까지 그럴 줄은 몰랐다."
▶
정답: ①
'설마'는 부사로 '아무리 그렇더라도, 설마하니'의 뜻이다. 표기는 설마가 맞다. 혼동되는 표현: '설령'(부사, 가정·양보 "설령 실패하더라도"), '설사'(부사, 양보 "설사 그렇다 하더라도"). 세 단어 모두 뜻이 다르므로 구분해야 한다.
Section I · '-이' / '-히' 부사
10문제 · 부사 파생 접미사 구분법
📖 이론 — '-이'/'-히' 부사 파생 규칙 전문

1. 부사 파생의 원리

형용사 어간에 접미사 '-이' 또는 '-히'를 붙여 부사를 만든다. 문제는 발음상 [이]와 [히]가 구분되지 않는 경우가 많아 표기를 혼동한다는 점이다. 한글 맞춤법 제51항에서 이를 규정한다.

2. 받침별 규칙 총정리

끝 받침접미사규칙예
ㄱ-히ㄱ 받침 뒤는 거의 '-히'극히, 솔직히, 정확히, 엄격히, 침착히, 특히
ㅅ-이ㅅ 받침 뒤는 '-이'깨끗이, 반듯이, 나긋이, 버젓이, 산뜻이, 지긋이
ㅂ-이ㅂ 불규칙 활용 → '-이'가까이, 괴로이, 새로이, 외로이, 즐거이
ㅎ-히ㅎ 받침 → '-히' (ㅎ 탈락형)익히, 특히 (예외적 패턴)
받침 없음-이받침 없는 어근 + '-이'높이, 깊이, 같이, 많이, 넓이

3. '-하다' 파생 여부에 따른 구분

  • '-하다'가 붙는 어근: 대체로 -히를 쓴다.
    예: 급히(급하다), 분명히(분명하다), 정중히(정중하다), 간편히(간편하다), 소홀히(소홀하다), 당당히(당당하다), 조용히(조용하다)
  • '-하다'가 안 붙는 어근: 대체로 -이를 쓴다.
    예: 깨끗이(깨끗하다? → 가능하지만 ㅅ 받침 우선 규칙), 가까이(가깝다 → ㅂ 불규칙)
  • 주의: '-하다'가 붙더라도 ㅅ 받침이면 '-이'가 우선. 깨끗이(○) / 깨끗히(×)

4. 핵심 예외 & 주의 사항

올바른 표기틀린 표기규칙 근거

일일이일일히 ×받침 없음 → '-이'

5. 판별 플로우차트

형용사/어근의 끝 받침 확인 │ ├─ ㅅ 받침 → 무조건 -이 │ 예: 깨끗이, 반듯이, 나긋이 │ ├─ ㄱ 받침 → 거의 항상 -히 │ 예: 솔직히, 정확히, 극히, 엄격히 │ ├─ ㅂ 불규칙 (ㅂ→우) → -이 │ 예: 가까이(가깝-), 괴로이(괴롭-) │ ├─ 받침 없음 → 대체로 -이 │ 예: 같이, 높이, 많이, 일일이 │ └─ -하다 가능한 한자어 어근 → 대체로 -히 예: 급히, 분명히, 정중히, 조용히 단, ㅅ 받침은 '-이' 우선 (깨끗이)

6. 암기 구결

"ㄱ히 ㅅ이" — ㄱ 받침은 '-히', ㅅ 받침은 '-이'. 이것만 확실히 외우면 80% 해결.
나머지: ㅂ 불규칙은 '-이', '-하다' 어근은 '-히', 확실치 않으면 사전.

'-이' vs '-히' 구분 규칙

  • '-히'를 쓰는 경우:
    ① 'ㄱ' 받침 → '-히': 솔직히, 정확히, 극히, 침착히
    ② '-하다'가 붙는 한자어 어근 중 일부: 분명히, 정중히, 간편히, 소홀히, 급히
  • '-이'를 쓰는 경우:
    ① 'ㅅ' 받침 → '-이': 깨끗이, 반듯이, 나긋이, 산뜻이
    ② 'ㅂ' 불규칙 용언 → '-이': 가까이, 괴로이
    ③ 기타: 같이, 높이, 깊이
  • 실전 팁: 'ㄱ' 받침 → 거의 '-히', 'ㅅ' 받침 → 거의 '-이'. 나머지는 사전 확인이 가장 확실
I-01
"일을 ( ) 처리했다."
▶
정답: ①
'깨끗하다' → 어근 '깨끗'의 받침 'ㅅ'. 'ㅅ' 받침 → '-이'가 원칙. 깨끗이.
I-02
"그 점을 ( ) 알고 있다."
▶
정답: ②
'분명하다' → '-하다'가 자연스럽게 붙는 어근. 'ㅇ' 받침이지만 '-하다' 결합 → '-히'. 분명히. 비슷한 예: 당연히, 정중히, 엄격히.
I-03
"( ) 말하면 그건 거짓말이다."
▶
정답: ①
'솔직하다' → 어근 '솔직'의 받침 'ㄱ'. 'ㄱ' 받침 → '-히'. 솔직히.
I-04
"그 소식을 ( ) 들었다."
▶
정답: ②
'똑똑하다'(분명하다) → 어근 '똑똑'의 받침 'ㄱ'. 'ㄱ' 받침 → '-히'. 똑똑히.
I-05
"자세를 ( ) 하고 앉아라."
▶
정답: ①
'반듯하다' → 어근 '반듯'의 받침 'ㅅ'. 'ㅅ' 받침 → '-이'. 반듯이. ('반드시'와 혼동 주의 — '반드시'는 별개의 부사.)
I-06
"( ) 서두르지 마라."
▶
정답: ②
'급하다' → 어근 '급'의 받침 'ㅂ'. 보통 'ㅂ' 받침은 '-이'를 쓰지만, '급하다'는 '-하다'가 자연스러운 한자어 어근이므로 '-히'를 쓴다. 급히. 이것이 '-이'/'-히' 구분이 까다로운 이유이다.
I-07
"그녀는 ( ) 미소 지었다."
▶
정답: ①
'나긋하다' → 어근 '나긋'의 받침 'ㅅ'. 'ㅅ' 받침 → '-이'. 나긋이.
I-08
"그는 어떤 상황에서도 ( ) 대처했다."
▶
정답: ②
'침착하다' → 어근 '침착'의 받침 'ㄱ'. 'ㄱ' 받침 → '-히'. 침착히.
I-09
"그녀와 ( ) 지내고 있다."
▶
정답: ①
'가깝다' → 어근의 받침 'ㅂ'(가까+ㅂ). 'ㅂ' 불규칙 활용을 하는 형용사. 부사형은 가까이. '가까히'(×). 'ㅂ' 불규칙 용언은 대체로 '-이' 부사를 만든다.
I-10
"그는 ( ) 끝까지 해냈다."
▶
정답: ②
'묵묵하다' → 어근 '묵묵'의 받침 'ㄱ'. 'ㄱ' 받침 → '-히'. 묵묵히. 'ㄱ' 받침 어근의 부사형은 거의 대부분 '-히'이다.
Section J · 실전 문장 교정 — 맞춤법
10문제 · 문장에서 맞춤법 오류를 모두 찾아 교정하시오
J-01 왠지 오늘은 모든것이 잘 될것 같은 기분이 든데.
교정:

왠지 오늘은 모든 것이 잘 될 것 같은 기분이 드는데.

'모든것' → '모든 것' (의존명사 띄어쓰기). '될것' → '될 것' (의존명사 띄어쓰기). '든데' → '드는데' ('들다'의 활용 + '-는데' 어미). '왠지'는 맞는 표기.
J-02 솔직이 말해서 이번 시험은 됬든 안됬든 별로 신경쓰지 않는다.
교정:

솔직히 말해서 이번 시험은 됐든 안 됐든 별로 신경 쓰지 않는다.

'솔직이' → '솔직히' (ㄱ 받침 → -히). '됬든' → '됐든' ('되었'의 준말 '됐'). '안됬든' → '안 됐든' (부사 '안' 띄어쓰기 + '됐'). '신경쓰지' → '신경 쓰지' (명사+동사 띄어쓰기).
J-03 학생으로써 본분을 다하지 못한채 졸업하게 됬다.
교정:

학생으로서 본분을 다하지 못한 채 졸업하게 됐다.

'으로써' → '으로서' (자격·신분 → -로서). '못한채' → '못한 채' ('채'는 의존명사 → 띄어쓰기). '됬다' → '됐다' ('되었다'의 준말).
J-04 웬지 오늘따라 촛점이 안맞는 느낌이다. 깨끗히 정리하고 다시 해봐야겠다.
교정:

왠지 오늘따라 초점이 안 맞는 느낌이다. 깨끗이 정리하고 다시 해 봐야겠다.

'웬지' → '왠지' ('왜인지'의 준말). '촛점' → '초점' (한자+한자, 예외 아님). '안맞는' → '안 맞는' (부사 '안' 띄어쓰기). '깨끗히' → '깨끗이' (ㅅ 받침 → -이).
J-05 그는 반듯히 누워서 묵묵이 천장만 바라봤다.
교정:

그는 반듯이 누워서 묵묵히 천장만 바라봤다.

'반듯히' → '반듯이' (ㅅ 받침 → -이). '묵묵이' → '묵묵히' (ㄱ 받침 → -히). 정반대의 실수를 유도하는 함정 문제.
J-06 피잣집에서 만둣국을 먹을수는 없으니까 만두집에 가자.
교정:

피자집에서 만둣국을 먹을 수는 없으니까 만두집에 가자.

'피잣집' → '피자집' (외래어+순우리말 → 사이시옷 없음). '먹을수는' → '먹을 수는' ('수'는 의존명사 → 띄어쓰기). '만둣국'은 한자어+순우리말이므로 사이시옷 적용 → 맞는 표기.
J-07 정확이 맞추다와 맞히다의 차이를 모르겠는데, 시험문제를 맞추려면 어떻게 해야되?
교정:

정확히 '맞추다'와 '맞히다'의 차이를 모르겠는데, 시험 문제를 맞히려면 어떻게 해야 돼?

'정확이' → '정확히' (ㄱ 받침 → -히). '시험문제' → '시험 문제' (합성어 아님, 띄어쓰기). '맞추려면' → '맞히려면' (정답을 알아내는 것 → 맞히다). '해야되' → '해야 돼' ('되다'의 활용 '돼' = '되어').
J-08 그 아이는 자기 입장에서만 생각하는데가 문제야. 남의 입장으로써 생각해 볼줄 모른다니까.
교정:

그 아이는 자기 입장에서만 생각하는 데가 문제야. 남의 입장으로서 생각해 볼 줄 모른다니까.

'생각하는데가' → '생각하는 데가' ('데'는 의존명사 → 띄어쓰기). '으로써' → '으로서' (입장·자격 → -로서). '볼줄' → '볼 줄' ('줄'은 의존명사 → 띄어쓰기).
J-09 그가 온대서 기다렸는데, 왠일인지 안오더라.
교정:

그가 온다고 해서 기다렸는데, 웬일인지 안 오더라.

'온대서' → '온다고 해서' (간접 인용을 풀어 쓰면 더 정확. '온대서'도 구어체에서 쓰이지만). '왠일' → '웬일' ('왠'은 '왠지'에서만, 관형사는 '웬'). '안오더라' → '안 오더라' (부사 '안' 띄어쓰기).
J-10 반듯이 해야되는 일이 있는데, 급이 서둘르면 오히려 일을 그릇치게 될것이다.
교정:

반드시 해야 되는 일이 있는데, 급히 서두르면 오히려 일을 그르치게 될 것이다.

'반듯이' → '반드시' ('꼭'의 뜻이므로 '반드시'. '반듯이'는 자세가 바른 모양). '해야되는' → '해야 되는' (보조용언 띄어쓰기). '급이' → '급히' (한자어 어근 → -히). '서둘르면' → '서두르면' ('서두르다'가 표준형). '그릇치게' → '그르치게' ('그르치다'가 표준형). '될것이다' → '될 것이다' (의존명사 띄어쓰기).
