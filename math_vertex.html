2차함수 꼭짓점

"왜 -b/2a인지" 직관으로 이해하기

꼭짓점이 뭔데?

공을 위로 던져봐.

올라간다... 올라간다... 점점 느려진다...

멈췄다.

그리고 떨어진다.

그 "멈추는 순간"이 꼭짓점이야.

꼭짓점에서는:

- 더 이상 올라가지 않아

- 아직 내려가지도 않아

- 변화가 0인 순간

수학적으로 "변화가 0" = 미분값이 0. 꼭짓점을 찾는 것 = 미분의 맛보기야. 지금은 미분 몰라도 돼.
· · ·

왜 중요한 건데?

네가 사장이야. 빵을 만들어 팔아.


적게 만들면? → 매출이 적어.

많이 만들면? → 재료비가 폭등해서 적자.


어딘가에 "딱 이만큼 만들면 이윤이 최대"인 지점이 있어.

그게 꼭짓점이야.

현실에서 "최대" 또는 "최소"를 찾는 문제는 전부 꼭짓점 찾기야:

- 이윤이 최대가 되는 생산량

- 비용이 최소가 되는 지점

- 로켓이 가장 높이 올라가는 순간

- 울타리로 가장 넓은 면적을 만드는 방법

꼭짓점 = "가장 좋은 지점"을 찾는 도구

· · ·

포물선의 비밀: 대칭

포물선을 반으로 접어봐.

왼쪽과 오른쪽이 완벽하게 겹쳐.

접히는 선 = 대칭축.

꼭짓점은 그 대칭축 위에 있어.

포물선이 x축과 만나는 두 점(근)이 x₁, x₂라면:

x₁ ────── 꼭짓점 ────── x₂

꼭짓점 = 정가운데 = (x₁ + x₂) / 2

대칭이니까 당연히 꼭짓점은 두 근의 정중앙이야. 직관적이지?

· · ·

그래서 왜 -b/2a ?

y = ax² + bx + c 의 두 근은 근의 공식으로 구해:

x₁ = (-b + √(b²-4ac)) / 2a
x₂ = (-b - √(b²-4ac)) / 2a

두 근의 정중앙을 구하면:

(x₁ + x₂) / 2

더해보자:

x₁ + x₂
= (-b + √...) / 2a + (-b - √...) / 2a

+√... 와 -√... 가 상쇄돼서:

= -2b / 2a = -b/a

정중앙 = 이걸 2로 나누면:

(-b/a) / 2 = -b / 2a

-b/2a = "두 근의 정중앙"

포물선이 대칭이니까, 정중앙이 꼭대기(또는 바닥).

그게 전부야.

외울 필요 없어. "대칭이니까 가운데" → "두 근 더해서 2로 나누면 -b/2a" 이 흐름만 기억하면 언제든 유도 가능해.
· · ·

완전제곱식 — "별명 붙이기"

"서울특별시 서대문구 대현동 34-15 5층"

이건 정확한 주소야. 근데 직관적이지 않아.


"이대앞 스카"

이건 별명이야. 같은 곳인데 바로 알아들어.

2차함수도 마찬가지야:

y = 2x² - 8x + 10 ← 주소 (꼭짓점 안 보임)

↓ 변환 ↓

y = 2(x - 2)² + 2 ← 별명 (꼭짓점 바로 보임!)

별명(완전제곱식) y = a(x - p)² + q 에서:

p = 꼭짓점의 x좌표 → 여기선 2

q = 꼭짓점의 y좌표 → 여기선 2

a = 뾰족한 정도 → 여기선 2 (양수니까 아래로 볼록)

어떻게 변환하는 건데?

y = 2x² - 8x + 10 을 변환하는 과정:

① 최고차항 계수를 밖으로 빼:
y = 2(x² - 4x) + 10

② 괄호 안을 완전제곱식으로 만들어:
x² - 4x + 4 - 4 = (x-2)² - 4

③ 다시 넣으면:
y = 2{(x-2)² - 4} + 10
y = 2(x-2)² - 8 + 10
y = 2(x-2)² + 2
"4x의 절반인 2를 제곱해서 4를 더하고 빼고" — 이 부분이 트릭이야. 왜 절반이냐면 (x-a)² = x² - 2ax + a² 에서 가운데 항이 2a이니까 절반하면 a가 나오는 거야.
· · ·

a의 의미 — 뾰족함과 방향

a가 크면? → 포물선이 좁고 뾰족. 변화가 급격.

a가 작으면? → 포물선이 넓고 완만. 변화가 느림.


a > 0 (양수) → 아래로 볼록 (U자) → 꼭짓점이 최솟값

a < 0 (음수) → 위로 볼록 (∩자) → 꼭짓점이 최댓값

경제학에서 이윤 함수는 보통 a < 0 (위로 볼록). 생산량 늘리면 이윤이 올라가다가 어느 순간 꺾여. 그 꼭짓점이 "이윤 극대화 생산량"이야.
· · ·

실전 — 이윤 최대화

빵집 이윤 함수:

이윤(x) = -2x² + 100x - 300

x = 빵 생산량


몇 개 만들어야 이윤이 최대일까?

a = -2, b = 100, c = -300

꼭짓점 x = -b/2a
= -100 / (2 × -2)
= -100 / -4
= 25

25개 만들 때 이윤 최대.

그때 이윤은?

이윤(25) = -2(25)² + 100(25) - 300
= -1250 + 2500 - 300
= 950

최대 이윤: 950.

꼭짓점 (25, 950) = "빵 25개일 때 이윤 950으로 최대"


이 문제를 미분으로 풀면?

이윤'(x) = -4x + 100 = 0

x = 25 ← 같은 답!


꼭짓점 찾기 = 미분해서 0 놓기. 같은 거야.

· · ·

정리

1. 꼭짓점 = 변화가 0인 순간 = 최대/최소

2. -b/2a = 두 근의 정중앙 (대칭이니까)

3. 완전제곱식 = 꼭짓점이 바로 보이는 별명

4. a > 0 → 최솟값, a < 0 → 최댓값

5. 경제학에서 이윤 최대화 = 꼭짓점 찾기 = 미분의 맛보기
