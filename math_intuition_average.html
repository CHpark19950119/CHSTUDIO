평균의 직관

"왜 수식으로 사람 수가 정해지는가" — 내면화를 위한 글

1. 평균은 시소다

천홍아, 평균을 숫자로 생각하지 마. 시소로 생각해.


시소 위에 여러 사람이 앉아있어. 시소가 균형 잡히는 지점 = 평균.


무거운 사람이 한 명 내리면? 시소가 반대쪽으로 기울어. 평균이 바뀌는 거야.

이 문제에서:

- 전체가 앉아있을 때 균형점 = 9.5

- 가장 무거운 사람(최고점) 내리면 → 균형점이 9.4로 내려가

- 가장 가벼운 사람(최저점) 내리면 → 균형점이 9.55로 올라가

한 사람이 내렸는데 균형이 많이 바뀌면 → 사람이 적다는 뜻.

한 사람이 내렸는데 균형이 거의 안 바뀌면 → 사람이 많다는 뜻.

100명이 탄 시소에서 1명 내리면 거의 안 움직여. 3명이 탄 시소에서 1명 내리면 확 기울어. "평균의 변화량"이 "사람 수"를 알려주는 거야.

2. 직접 움직여봐

심사위원 수를 바꿔보면서 최고점/최저점이 어떻게 변하는지 봐.

최고점 = 0.1 × 4 + 9.4 = 9.8
최저점 = -0.05 × 4 + 9.55 = 9.35
슬라이더를 7 이상으로 올려봐. 최고점이 10을 넘어. 10점이 최대인 현실에서 불가능한 거야. 이게 "현실이 수식에 브레이크를 거는 순간"이야.
· · ·

3. 물통 비유

심사위원 점수를 물통이라고 생각해. 각 물통에 물(점수)이 들어있어.


전체 물의 양 = 9.5 × n (평균 × 개수)


가장 큰 물통을 빼면, 남은 물통들의 평균이 9.4가 돼.

그 말은 — 빠진 물통에는 평균보다 많은 물이 있었다는 거야.


얼마나 많이? 그건 물통이 몇 개냐에 달려있어.

물통 2개일 때: 큰 물통 빼면 1개만 남아. 그 1개가 9.4. 큰 물통 = 9.5×2 - 9.4 = 9.6.

물통 6개일 때: 큰 물통 빼면 5개 남아. 5개 평균 9.4. 큰 물통 = 9.5×6 - 9.4×5 = 57 - 47 = 10.

물통 7개일 때: 큰 물통 = 9.5×7 - 9.4×6 = 66.5 - 56.4 = 10.1. 물통은 10까지만 담을 수 있어. 불가능.

물통 용량(10점 상한)이 사람 수의 한계를 만드는 거야.

4. 왜 이게 "수리적으로" 가능한가

천홍아, 네가 느끼는 불편함은 이거야:


"평균 3개를 알려줬을 뿐인데, 왜 사람 수까지 나오지? 정보가 부족한 것 아닌가?"

직관적으로 정보가 부족해 보이지. 근데 아니야.

평균 1개를 아는 것 = 전체 합을 아는 것.

최고점 제외 평균을 아는 것 = 최고점의 값을 n으로 표현 가능.

최저점 제외 평균을 아는 것 = 최저점의 값을 n으로 표현 가능.

즉, 평균 3개 = 방정식 3개. 미지수가 n 하나. 여유가 있어.

"평균"이라는 단어가 뭉뚱그려서 정보가 적어 보이는 거야. 근데 평균 = 합계/개수이고, 합계는 구체적인 숫자야. 변환하면 정보가 드러나.

이건 일상에서도 마찬가지야:

"우리 팀 평균 월급이 300만원인데, 사장님 빼면 250만원이래."


이 한 문장으로 뭘 알 수 있어?

→ 사장 월급이 팀 규모에 따라 정해져. 팀이 클수록 사장 월급이 높아야 평균을 그만큼 올리거든.

→ 근데 월급에도 상한이 있어 (회사 매출 한계). 그래서 팀 규모도 한계가 생겨.

5. 핵심 — "한 명의 영향력"

이 모든 것의 직관적 핵심:

평균에서 한 명의 영향력 = 그 사람의 값과 평균의 차이 / 전체 인원수


인원이 적으면 → 한 명의 영향력이 크다 → 평균이 많이 변한다

인원이 많으면 → 한 명의 영향력이 작다 → 평균이 조금 변한다


"평균이 얼마나 변했는지"를 보면 "몇 명인지"를 역으로 알 수 있다.

이 문제에서:

- 최고점 빠지니 평균이 0.1 내려갔어 → 최고점의 영향력 = 0.1 × n

- 최고점의 영향력이 실제 점수로 실현 가능하려면 → 10점 이하여야 해

- 그래서 n ≤ 6

6. 내면화 테스트

수식 없이 답해봐 (머릿속으로):


"5명이 시험을 봤어. 평균 80점. 1명을 빼니 평균이 75점이 됐어. 빠진 사람은 몇 점?"

직관: 빠진 사람이 평균을 5점이나 올리고 있었어. 5명 중 1명이니까 영향력이 크네.
계산: 전체 합 = 80×5 = 400. 나머지 합 = 75×4 = 300. 빠진 사람 = 100점.

하나 더:


"평균이 80점인데 1명 빼니 79.5점이 됐어. 100점이 최대라면 최대 몇 명일 수 있어?"

직관: 평균이 0.5밖에 안 변했어 → 사람이 꽤 많았나 봐.
빠진 사람 = 0.5n + 80. 이게 100 이하여야 하니까: 0.5n ≤ 20, n ≤ 40.
최대 40명.

· · ·

내면화 = "수식을 쓰기 전에 방향이 보이는 것"


1. 평균이 많이 변했으면 → 사람 적다

2. 평균이 조금 변했으면 → 사람 많다

3. 점수에 상한이 있으면 → 사람 수에 한계가 생긴다


이 세 줄이 수식 없이도 답의 방향을 알려줘.
