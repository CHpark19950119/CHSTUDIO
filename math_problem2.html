심화문제 2번 — 단계별 풀이

실마리가 안 보이는 이유부터 설명한다.

문제: 요리 대회에서 심사위원이 10점까지 점수를 줄 수 있다. 최고점 하나를 제외한 평균과 최저점 하나를 제외한 평균을 비교한다.

• 평균 점수: 9.5점

• 최고점 하나를 제외한 평균: 9.4점

• 최저점 하나를 제외한 평균: 9.55점

구하시오: 최저점은 최소 몇 점? 심사위원은 최대 몇 명?

왜 실마리가 안 보일까?

이 문제가 어려운 건 네 한계가 아니야. 문제 구조가 원래 어려운 거야.


보이는 것: 평균 3개
모르는 것: 심사위원 수(n), 최고점, 최저점, 나머지 점수들


미지수가 여러 개인데 조건이 3개뿐. "뭘 어디서 시작해야 하지?"가 안 보이는 거야.

핵심 전략: "평균 = 합계 / 개수" → 합계로 바꾸면 식이 나온다.

단계 0/6

Step 1: 변수 설정

모르는 걸 문자로 놓자. 이게 첫 번째 실마리야.

n = 심사위원 수 (모름)
전체 합 = 9.5 × n (평균 × 개수)

왜 이렇게 하냐? "평균"이라는 정보를 "합계"로 바꾸면 계산이 가능해져.

평균은 비교하기 어렵지만, 합계는 더하고 빼기가 돼. 평균 문제의 핵심은 항상 "합계로 변환"이야.

확인: 평균이 9.5이고 심사위원이 n명이면, 전체 점수의 합은?

Step 2: 최고점 구하기

"최고점 하나를 제외한 평균 = 9.4"

최고점을 빼면 남는 건 (n-1)명의 점수.

최고점 제외한 합 = 9.4 × (n-1)

전체 합 - 최고점 = 최고점 제외한 합

9.5n - 최고점 = 9.4(n-1)

최고점 = 9.5n - 9.4(n-1)
= 9.5n - 9.4n + 9.4
= 0.1n + 9.4

최고점 = 0.1n + 9.4

n을 모르지만, 최고점을 n으로 표현할 수 있게 됐어. 이게 핵심이야.

확인: 심사위원이 4명이면 최고점은?

Step 3: 최저점 구하기

같은 방법으로. "최저점 하나를 제외한 평균 = 9.55"

최저점 제외한 합 = 9.55 × (n-1)

9.5n - 최저점 = 9.55(n-1)

최저점 = 9.5n - 9.55(n-1)
= 9.5n - 9.55n + 9.55
= -0.05n + 9.55

최저점 = -0.05n + 9.55

주목: n이 커지면 최저점이 줄어들어. (-0.05 × n이니까)

n이 커지면 최고점은 올라가고, 최저점은 내려가.

확인: 심사위원이 4명이면 최저점은?

Step 4: 제약 조건 — 10점 이하

"심사위원이 10점까지 점수를 줄 수 있다"

→ 최고점 ≤ 10

최고점 = 0.1n + 9.4 ≤ 10

0.1n ≤ 0.6

n ≤ 6

심사위원은 최대 6명!

n = 6이면 최고점 = 0.1(6) + 9.4 = 10점 (딱 한계)

n = 7이면 최고점 = 0.1(7) + 9.4 = 10.1 → 10점 초과! 불가능.

확인: 왜 n ≤ 6인가?

Step 5: 최저점 구하기

n = 6 (최대)일 때 최저점은?

최저점 = -0.05(6) + 9.55
= -0.3 + 9.55
= 9.25

최저점 = 9.25점, 심사위원 = 최대 6명

검산:

6명 전체 합 = 9.5 × 6 = 57
최고점 = 10, 최저점 = 9.25
나머지 4명 합 = 57 - 10 - 9.25 = 37.75

최고점 제외 합 = 57 - 10 = 47
평균 = 47 / 5 = 9.4 ✓

최저점 제외 합 = 57 - 9.25 = 47.75
평균 = 47.75 / 5 = 9.55 ✓

Step 6: 왜 이 문제가 어려웠나

실마리가 안 보인 이유:

1. 미지수가 여러 개인데 어디서 시작할지 몰랐어 → "합계로 변환"이 시작점

2. n이 숫자가 아니라 문자라서 막막했어 → n으로 표현해놓고 나중에 조건으로 범위 좁히기

3. "10점까지"라는 조건이 핵심인데, 처음엔 그냥 배경 설명처럼 보여 → 제약 조건이 답의 열쇠

이 문제에서 배울 패턴:

1. 평균 → 합계로 변환 (항상 첫 단계)

2. 모르는 값을 문자로 놓기 (n)

3. 식을 세워서 n으로 표현

4. 제약 조건(≤10)으로 범위 좁히기

5. 검산으로 확인

답: 최저점 최소 9.25점, 심사위원 최대 6명

이건 네 수학적 한계가 아니야. 이 유형을 처음 본 거라 패턴을 모른 것뿐이야. "평균 → 합계 변환 → 문자로 표현 → 조건으로 좁히기" — 이 패턴을 한 번 알면 비슷한 문제 다 풀 수 있어. 다음에 평균 문제 나오면 이 4단계 떠올려봐.
